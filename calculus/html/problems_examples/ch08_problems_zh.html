8.1 微分方程基礎 (Basics of Differential Equations)

🔑 本節重點：微分方程的階、通解與特解、初始值問題、驗證解

📝 例題 8.1
驗證微分方程的解
驗證 $y = e^{-3x} + 2x + 3$ 是微分方程 $y' + 3y = 6x + 11$ 的解。
解：先求導數：$y' = -3e^{-3x} + 2$。代入左側：$y' + 3y = (-3e^{-3x} + 2) + 3(e^{-3x} + 2x + 3)$
$= -3e^{-3x} + 2 + 3e^{-3x} + 6x + 9 = 6x + 11$。等於右側，驗證完畢。
✅ 答案：$y = e^{-3x} + 2x + 3$ 滿足方程式 ✓
📝 例題 8.2
辨識微分方程的階
求下列微分方程的階： (a) $y' + y = 3x^2$ (b) $y'' + 2y' + y = 0$ (c) $y^{(4)} + y''' = \cos x$
解：(a) 最高導數為 $y'$，階為 1。
(b) 最高導數為 $y''$，階為 2。
(c) 最高導數為 $y^{(4)}$，階為 4。
✅ 答案：(a) 1 階 (b) 2 階 (c) 4 階
📝 例題 8.3
通解與特解
求微分方程 $y' = 2x$ 的通解，並求滿足 $y(0) = 3$ 的特解。
解：通解：積分 $\int y'\,dx = \int 2x\,dx$，得 $y = x^2 + C$。
特解：代入 $y(0)=3$，得 $0^2 + C = 3$，故 $C = 3$。特解為 $y = x^2 + 3$。
✅ 答案：通解：$y = x^2 + C$；特解：$y = x^2 + 3$
📝 例題 8.4
初始值問題
解初始值問題：$y' = \frac{y}{x}$，$y(1) = 2$。
解：分離變數：$\frac{dy}{y} = \frac{dx}{x}$。積分：$\ln|y| = \ln|x| + C$。取指數：$|y| = e^C|x|$，即 $y = Kx$（$K = \pm e^C$）。
代入 $y(1)=2$：$2 = K \cdot 1$，得 $K=2$。故特解為 $y = 2x$。
✅ 答案：$y = 2x$

8.2 方向場與數值方法 (Direction Fields and Numerical Methods)

🔑 本節重點：方向場繪製、歐拉法近似解

📝 例題 8.5
繪製方向場
描繪 $y' = x - y$ 的方向場（部分格子點）。
解：在網格點 $(x,y)$ 處，斜率 $m = x - y$。例如：$(0,0): m=0$；$(1,0): m=1$；$(0,1): m=-1$；$(1,1): m=0$；$(2,0): m=2$。
在每個網格點畫一小段斜率為 $m$ 的線段，即構成方向場。積分曲線沿著這些小線段的方向走。
✅ 答案：方向場如上述描述
📝 例題 8.6
歐拉法：y'=y，y(0)=1
使用歐拉法，步長 $h=0.1$，近似 $y' = y$、$y(0)=1$ 在 $x=0.5$ 的解。
解：$y_0 = 1$，$f(x,y)=y$。
$y_1 = y_0 + h f(x_0,y_0) = 1 + 0.1 \cdot 1 = 1.1$
$y_2 = 1.1 + 0.1 \cdot 1.1 = 1.21$
$y_3 = 1.21 + 0.1 \cdot 1.21 = 1.331$
$y_4 = 1.331 + 0.1 \cdot 1.331 = 1.4641$
$y_5 = 1.4641 + 0.1 \cdot 1.4641 = 1.61051$
精確解 $y=e^{0.5} \approx 1.6487$，歐拉近似 $\approx 1.6105$。
✅ 答案：$y(0.5) \approx 1.6105$

8.3 可分離微分方程 (Separable Equations)

🔑 本節重點：分離變數法、指數成長與衰減、牛頓冷卻定律

📝 例題 8.7
基本分離變數：y'=x²/y
解 $\frac{dy}{dx} = \frac{x^2}{y}$。
解：分離變數：$y\,dy = x^2\,dx$。積分：$\int y\,dy = \int x^2\,dx$，得 $\frac{y^2}{2} = \frac{x^3}{3} + C$。
乘以 2：$y^2 = \frac{2x^3}{3} + 2C$。令 $K = 2C$：$y = \pm\sqrt{\frac{2x^3}{3} + K}$。
✅ 答案：$y = \pm\sqrt{\frac{2x^3}{3} + K}$
📝 例題 8.8
指數成長：dp/dt=kp
解 $\frac{dp}{dt} = kp$，初始條件 $p(0) = p_0$。
解：分離：$\frac{dp}{p} = k\,dt$。積分：$\ln|p| = kt + C$。$|p| = e^{kt+C} = e^C e^{kt}$。
$p = A e^{kt}$（$A = \pm e^C$）。代入 $p(0)=p_0$：$A = p_0$。故 $p = p_0 e^{kt}$。
✅ 答案：$p(t) = p_0 e^{kt}$
📝 例題 8.9
指數衰減：放射性衰變
放射性物質的半衰期為 5 年。若初始質量為 100g，求 10 年後的剩餘質量。
解：$m(t) = m_0 e^{-kt}$。半衰期：$50 = 100 e^{-k \cdot 5}$，即 $e^{-5k} = 0.5$，$-5k = \ln 0.5 = -\ln 2$，$k = \frac{\ln 2}{5}$。
$m(10) = 100 e^{-\frac{\ln 2}{5} \cdot 10} = 100 e^{-2\ln 2} = 100 \cdot \frac{1}{4} = 25$g。
✅ 答案：$25$ g
📝 例題 8.10
牛頓冷卻定律
物體從 100°C 放入 20°C 環境中。1 分鐘後溫度降至 80°C。求 3 分鐘後的溫度。
解：$\frac{dT}{dt} = -k(T - 20)$。分離：$\frac{dT}{T-20} = -k\,dt$。積分：$\ln|T-20| = -kt + C$，$T-20 = Ae^{-kt}$。
$T(0)=100$：$100-20 = 80 = A$，故 $T-20 = 80e^{-kt}$。
$T(1)=80$：$80-20 = 60 = 80e^{-k}$，$e^{-k} = 0.75$，$k = -\ln 0.75$。
$T(3) = 20 + 80(e^{-k})^3 = 20 + 80(0.75)^3 = 20 + 80 \cdot 0.421875 = 53.75$°C。
✅ 答案：$53.75$°C

8.4 羅吉斯方程 (The Logistic Equation)

🔑 本節重點：羅吉斯成長模型、承載容量、解的推導與應用

📝 例題 8.11
羅吉斯方程的通解
解 $\frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right)$。
解：使用部分分式：$\frac{1}{P(1-P/K)} = \frac{1}{P} + \frac{1/K}{1-P/K}$。
分離變數後積分：$\int \left(\frac{1}{P} + \frac{1/K}{1-P/K}\right)dP = \int r\,dt$
$\ln|P| - \ln|1-P/K| = rt + C$。$\ln\left|\frac{P}{1-P/K}\right| = rt + C$。
$\frac{P}{1-P/K} = Ae^{rt}$。解出 $P$：$P = \frac{K}{1 + Be^{-rt}}$，其中 $B = \frac{K-P_0}{P_0}$。
✅ 答案：$P(t) = \frac{K}{1 + \left(\frac{K-P_0}{P_0}\right)e^{-rt}}$
📝 例題 8.12
羅吉斯成長應用
人口以羅吉斯模型成長，$r=0.1$、$K=1000$、$P_0=100$。求 $t=20$ 時的人口。
解：$P(t) = \frac{1000}{1 + 9e^{-0.1t}}$（因為 $\frac{1000-100}{100}=9$）。
$P(20) = \frac{1000}{1 + 9e^{-2}} = \frac{1000}{1 + 9 \cdot 0.1353} = \frac{1000}{2.218} \approx 451$。
✅ 答案：$P(20) \approx 451$
📝 例題 8.13
羅吉斯方程：求承載容量
羅吉斯模型 $\frac{dP}{dt} = 0.5P\left(1 - \frac{P}{500}\right)$。求承載容量與成長率。
解：比較標準形式 $\frac{dP}{dt} = rP(1-P/K)$：$r = 0.5$、$K = 500$。承載容量為 500，最大相對成長率為 0.5。
✅ 答案：$K = 500$，$r = 0.5$

8.5 一階線性微分方程 (First-order Linear Equations)

🔑 本節重點：積分因子法、標準形式 y'+P(x)y=Q(x)

📝 例題 8.14
積分因子法基本型
解 $y' + 2y = e^x$。
解：$P(x)=2$，積分因子 $\mu(x) = e^{\int 2\,dx} = e^{2x}$。
乘 $\mu$：$e^{2x}y' + 2e^{2x}y = e^{3x}$，即 $\frac{d}{dx}(e^{2x}y) = e^{3x}$。
積分：$e^{2x}y = \int e^{3x}\,dx = \frac{e^{3x}}{3} + C$。$y = \frac{e^x}{3} + Ce^{-2x}$。
✅ 答案：$y = \frac{e^x}{3} + Ce^{-2x}$
📝 例題 8.15
積分因子法：y'+xy=x
解 $y' + xy = x$。
解：$P(x)=x$，$\mu(x)=e^{\int x\,dx}=e^{x^2/2}$。
$\frac{d}{dx}(e^{x^2/2}y) = x e^{x^2/2}$。
積分：$e^{x^2/2}y = \int x e^{x^2/2}\,dx$。令 $u = x^2/2$、$du = x\,dx$：$= e^{x^2/2} + C$。
$y = 1 + Ce^{-x^2/2}$。
✅ 答案：$y = 1 + Ce^{-x^2/2}$
📝 例題 8.16
積分因子法：y'+(1/x)y=sin x
解 $y' + \frac{1}{x}y = \sin x$（$x>0$）。
解：$P(x)=1/x$，$\mu(x)=e^{\ln|x|}=x$。
$\frac{d}{dx}(xy) = x\sin x$。
積分：使用分部積分 $\int x\sin x\,dx = -x\cos x + \sin x + C$。
$xy = -x\cos x + \sin x + C$，$y = -\cos x + \frac{\sin x}{x} + \frac{C}{x}$。
✅ 答案：$y = -\cos x + \frac{\sin x}{x} + \frac{C}{x}$
📝 例題 8.17
RL 電路問題
RL 電路：$L\frac{di}{dt} + Ri = E$。若 $L=1$H、$R=10\Omega$、$E=12$V、$i(0)=0$，求 $i(t)$。
解：$\frac{di}{dt} + 10i = 12$。$P(t)=10$，$\mu=e^{10t}$。
$\frac{d}{dt}(e^{10t}i) = 12e^{10t}$。
積分：$e^{10t}i = \frac{12}{10}e^{10t} + C = 1.2e^{10t} + C$。
$i(0)=0$：$0 = 1.2 + C$，$C=-1.2$。$i = 1.2(1-e^{-10t})$。
✅ 答案：$i(t) = 1.2(1 - e^{-10t})$ A
📝 例題 8.18
混合問題（鹽水槽）
水槽含 100L 純水。鹽水以 5L/min 注入（濃度 2g/L），同時以 5L/min 排出。求時刻 $t$ 的鹽量。
解：設鹽量 $Q(t)$。$\frac{dQ}{dt} = \text{注入率} - \text{排出率} = 5 \cdot 2 - 5 \cdot \frac{Q}{100} = 10 - \frac{Q}{20}$。
$\frac{dQ}{dt} + \frac{1}{20}Q = 10$。$\mu = e^{t/20}$。
$\frac{d}{dt}(e^{t/20}Q) = 10e^{t/20}$。$e^{t/20}Q = 200e^{t/20} + C$。
$Q(0)=0$：$0 = 200 + C$，$C=-200$。$Q = 200(1-e^{-t/20})$。
✅ 答案：$Q(t) = 200(1 - e^{-t/20})$ g
📝 例題 8.19
積分因子：y'+3x²y=6x²
解 $y' + 3x^2 y = 6x^2$。
解：$P(x)=3x^2$，$\mu=e^{x^3}$。
$\frac{d}{dx}(e^{x^3}y) = 6x^2 e^{x^3}$。
令 $u=x^3$、$du=3x^2dx$：$\int 6x^2 e^{x^3}\,dx = 2\int e^u\,du = 2e^{x^3} + C$。
$e^{x^3}y = 2e^{x^3} + C$，$y = 2 + Ce^{-x^3}$。
✅ 答案：$y = 2 + Ce^{-x^3}$
