2.1 微積分預覽（A Preview of Calculus）

例題 2.1 — 求割線斜率（Finding Slopes of Secant Lines）
題目：考慮函數 $f(x) = x^2$。求通過點 $(1, 1)$ 和以下各點的割線斜率：(a) $(2, 4)$；(b) $(1.5, 2.25)$；(c) $(1.1, 1.21)$。當第二點趨近 $(1, 1)$ 時，割線斜率趨近何值？
解：割線斜率公式：$m_{\text{sec}} = \dfrac{f(x) - f(1)}{x - 1} = \dfrac{x^2 - 1}{x - 1}$
(a) $x = 2$：$m = \frac{4 - 1}{2 - 1} = 3$
(b) $x = 1.5$：$m = \frac{2.25 - 1}{1.5 - 1} = \frac{1.25}{0.5} = 2.5$
(c) $x = 1.1$：$m = \frac{1.21 - 1}{1.1 - 1} = \frac{0.21}{0.1} = 2.1$
當 $x \to 1$ 時，$m = \frac{(x-1)(x+1)}{x-1} = x+1 \to 2$。
答：割線斜率趨近 $2$，這就是切線斜率。
例題 2.2 — 求平均速度（Finding Average Velocity）
題目：一物體的位置函數為 $s(t) = t^2$（$t$：秒，$s$：英尺）。求以下時間區間的平均速度：(a) $[1, 2]$；(b) $[1, 1.5]$；(c) $[1, 1.1]$。瞬時速度在 $t=1$ 時為何？
解：平均速度 $v_{\text{avg}} = \dfrac{s(b) - s(a)}{b - a}$
(a) $[1,2]$：$v_{\text{avg}} = \frac{4 - 1}{2 - 1} = 3$ ft/s
(b) $[1, 1.5]$：$v_{\text{avg}} = \frac{2.25 - 1}{1.5 - 1} = 2.5$ ft/s
(c) $[1, 1.1]$：$v_{\text{avg}} = \frac{1.21 - 1}{1.1 - 1} = 2.1$ ft/s
瞬時速度 $v(1) = \lim_{t \to 1}\frac{t^2-1}{t-1} = \lim_{t \to 1}(t+1) = 2$ ft/s。
答：$t=1$ 時瞬時速度為 $2$ ft/s。
例題 2.3 — 以矩形估計面積（Estimation Using Rectangles）
題目：使用 $4$ 個等寬矩形，以左端點法（left endpoints）估計 $y = x^2$ 在 $[0, 2]$ 下方的面積。
解：
區間 $[0,2]$ 分成 $4$ 等份，每份寬度 $\Delta x = 0.5$。
左端點：$x_0 = 0,\ x_1 = 0.5,\ x_2 = 1.0,\ x_3 = 1.5$
矩形面積和：
$A \approx \Delta x[f(0) + f(0.5) + f(1) + f(1.5)]$
$= 0.5[0 + 0.25 + 1 + 2.25] = 0.5 \times 3.5 = 1.75$
答：估計面積約 $1.75$。（實際面積為 $\int_0^2 x^2 dx = \frac{8}{3} \approx 2.667$）

2.2 函數的極限（The Limit of a Function）

例題 2.4 — 使用函數值表格求極限 1（Evaluating a Limit Using a Table of Functional Values 1）
題目：利用數值表格估計 $\displaystyle\lim_{x \to 2}\dfrac{x^2 - 4}{x - 2}$。
解：製作表格，$x$ 從左右兩側趨近 $2$：
　$x$：1.9, 1.99, 1.999, 2.001, 2.01, 2.1
　$f(x)$：3.9, 3.99, 3.999, 4.001, 4.01, 4.1
數值趨近 $4$。驗證：$\frac{x^2-4}{x-2} = \frac{(x-2)(x+2)}{x-2} = x+2$（$x \neq 2$），所以 $\lim_{x \to 2}(x+2) = 4$。
答：$\displaystyle\lim_{x \to 2}\frac{x^2-4}{x-2} = 4$。
例題 2.5 — 使用函數值表格求極限 2（Evaluating a Limit Using a Table of Functional Values 2）
題目：估計 $\displaystyle\lim_{x \to 0}\dfrac{\sin x}{x}$。
解：製作表格（弧度制）：
　$x$：±0.1, ±0.01, ±0.001
　$\frac{\sin x}{x}$：0.99833, 0.99998, 1.00000
函數值從兩側趨近 $1$。
答：$\displaystyle\lim_{x \to 0}\frac{\sin x}{x} = 1$。這是一個重要的三角極限。
例題 2.6 — 使用圖形求極限（Evaluating a Limit Using a Graph）
題目：利用 $f(x) = \dfrac{|x-2|}{x-2}$ 的圖形求 $\displaystyle\lim_{x \to 2}f(x)$。
解：
當 $x > 2$：$|x-2| = x-2$，$f(x) = 1$
當 $x < 2$：$|x-2| = -(x-2)$，$f(x) = -1$
左極限 $\lim_{x \to 2^-}f(x) = -1$，右極限 $\lim_{x \to 2^+}f(x) = 1$。
由於左右極限不相等，$\displaystyle\lim_{x \to 2}f(x)$ 不存在。
例題 2.7 — 求不存在的極限（Evaluating a Limit That Fails to Exist）
題目：求 $\displaystyle\lim_{x \to 0}\sin\!\big(\frac{1}{x}\big)$。
解：
當 $x \to 0$ 時，$\frac{1}{x} \to \pm\infty$，$\sin(\frac{1}{x})$ 在 $-1$ 和 $1$ 之間無限振盪。
無論 $x$ 多靠近 $0$，函數值都不會趨近於任何單一數值。
答：極限不存在（does not exist）。
例題 2.8 — 求單側極限（Evaluating One-Sided Limits）
題目：求 (a) $\displaystyle\lim_{x \to 0^+}\frac{1}{x}$；(b) $\displaystyle\lim_{x \to 0^-}\frac{1}{x}$。
解：
(a) 當 $x \to 0^+$（從右側趨近 $0$），$x > 0$ 且很小，$\frac{1}{x} \to +\infty$。
(b) 當 $x \to 0^-$（從左側趨近 $0$），$x < 0$ 且很小，$\frac{1}{x} \to -\infty$。
答：(a) $+\infty$；(b) $-\infty$。兩側極限不相等，故 $\lim_{x \to 0}\frac{1}{x}$ 不存在。
例題 2.9 — 辨識無窮極限（Recognizing an Infinite Limit）
題目：求 $\displaystyle\lim_{x \to 1}\dfrac{1}{(x-1)^2}$。
解：
無論 $x \to 1^+$ 或 $x \to 1^-$，$(x-1)^2 \to 0^+$（趨近 $0$ 的正數）。
因此 $\frac{1}{(x-1)^2} \to +\infty$。
答：$\displaystyle\lim_{x \to 1}\frac{1}{(x-1)^2} = +\infty$（或稱極限不存在，函數趨向正無窮）。
例題 2.10 — 求垂直漸近線（Finding a Vertical Asymptote）
題目：求 $f(x) = \dfrac{x+1}{x^2-1}$ 的所有垂直漸近線。
解：
分母 $x^2-1 = (x-1)(x+1)$。分母為零處：$x = 1$ 和 $x = -1$。
在 $x = 1$：$\lim_{x \to 1}\frac{x+1}{(x-1)(x+1)} = \lim_{x \to 1}\frac{1}{x-1}$。此極限在 $x \to 1^+$ 時為 $+\infty$，$x \to 1^-$ 時為 $-\infty$。因此 $x = 1$ 是垂直漸近線。
在 $x = -1$：$\lim_{x \to -1}\frac{x+1}{(x-1)(x+1)} = \lim_{x \to -1}\frac{1}{x-1} = \frac{1}{-2}$。此為可移除不連續點，非垂直漸近線。
答：僅 $x = 1$ 為垂直漸近線。

2.3 極限法則（The Limit Laws）

例題 2.11 — 求基本極限（Evaluating a Basic Limit）
題目：求 $\displaystyle\lim_{x \to 3}(2x^2 - 3x + 1)$。
解：因多項式函數處處連續，可直接代入：
$\lim_{x \to 3}(2x^2 - 3x + 1) = 2(3^2) - 3(3) + 1 = 18 - 9 + 1 = 10$
答：$10$。
例題 2.12 — 使用極限法則求極限（Evaluating a Limit Using Limit Laws）
題目：已知 $\lim_{x \to 2}f(x) = 4$ 和 $\lim_{x \to 2}g(x) = -3$，求 $\displaystyle\lim_{x \to 2}[3f(x) - 2g(x)]$。
解：使用極限的線性性質：
$\lim_{x \to 2}[3f(x) - 2g(x)] = 3\lim_{x \to 2}f(x) - 2\lim_{x \to 2}g(x) = 3(4) - 2(-3) = 12 + 6 = 18$
答：$18$。
例題 2.13 — 重複使用極限法則（Using Limit Laws Repeatedly）
題目：求 $\displaystyle\lim_{x \to -2}\sqrt{x^3 + 2x + 4}$。
解：
先使用冪法則求內部極限：$\lim_{x \to -2}(x^3 + 2x + 4) = (-2)^3 + 2(-2) + 4 = -8 - 4 + 4 = -8$
但根號內為負數，函數在 $x = -2$ 附近無定義（實數範圍內）。
因此極限不存在（在實數範圍內）。
答：極限不存在（定義域問題）。
例題 2.14 — 求有理函數的極限（Evaluating a Limit of a Rational Function）
題目：求 $\displaystyle\lim_{x \to 5}\dfrac{x^2 - 25}{x - 5}$。
解：直接代入得 $\frac{0}{0}$ 不定型。因式分解：
$\dfrac{x^2 - 25}{x - 5} = \dfrac{(x-5)(x+5)}{x-5} = x+5$（對 $x \neq 5$）
$\lim_{x \to 5}(x+5) = 10$
答：$10$。
例題 2.15 — 因式分解後求極限（Evaluating a Limit by Factoring and Canceling）
題目：求 $\displaystyle\lim_{x \to -1}\dfrac{x^2 + 3x + 2}{x + 1}$。
解：直接代入得 $\frac{0}{0}$ 不定型：
$x^2 + 3x + 2 = (x+1)(x+2)$
$\dfrac{(x+1)(x+2)}{x+1} = x+2$（對 $x \neq -1$）
$\lim_{x \to -1}(x+2) = 1$
答：$1$。
例題 2.16 — 乘以共軛後求極限（Evaluating a Limit by Multiplying by a Conjugate）
題目：求 $\displaystyle\lim_{x \to 0}\dfrac{\sqrt{x+1} - 1}{x}$。
解：直接代入得 $\frac{0}{0}$ 不定型。乘以共軛：
$\dfrac{\sqrt{x+1}-1}{x} \cdot \dfrac{\sqrt{x+1}+1}{\sqrt{x+1}+1} = \dfrac{(x+1)-1}{x(\sqrt{x+1}+1)} = \dfrac{x}{x(\sqrt{x+1}+1)} = \dfrac{1}{\sqrt{x+1}+1}$
$\lim_{x \to 0}\dfrac{1}{\sqrt{x+1}+1} = \dfrac{1}{1+1} = \dfrac{1}{2}$
答：$\frac{1}{2}$。
例題 2.17 — 化簡繁分數後求極限（Evaluating a Limit by Simplifying a Complex Fraction）
題目：求 $\displaystyle\lim_{x \to 0}\dfrac{\frac{1}{x+1} - 1}{x}$。
解：直接代入得 $\frac{0}{0}$。合併分子：
$\dfrac{\frac{1}{x+1} - 1}{x} = \dfrac{\frac{1 - (x+1)}{x+1}}{x} = \dfrac{\frac{-x}{x+1}}{x} = \dfrac{-x}{x(x+1)} = \dfrac{-1}{x+1}$
$\lim_{x \to 0}\dfrac{-1}{x+1} = -1$
答：$-1$。
例題 2.18 — 極限法則不適用的情況（Evaluating a Limit When the Limit Laws Do Not Apply）
題目：求 $\displaystyle\lim_{x \to 0}\dfrac{|x|}{x}$。
解：
當 $x > 0$：$\frac{|x|}{x} = 1$，所以 $\lim_{x \to 0^+}\frac{|x|}{x} = 1$
當 $x < 0$：$\frac{|x|}{x} = -1$，所以 $\lim_{x \to 0^-}\frac{|x|}{x} = -1$
左右極限不相等，故極限不存在。
答：極限不存在（DNE）。
例題 2.19 — 使用極限法則求單側極限（Evaluating a One-Sided Limit Using the Limit Laws）
題目：求 $\displaystyle\lim_{x \to 3^+}\sqrt{x-3}$。
解：
當 $x \to 3^+$ 時，$x-3 \to 0^+$，所以 $\sqrt{x-3} \to 0$。
答：$0$。
例題 2.20 — 使用極限法則求雙側極限（Evaluating a Two-Sided Limit Using the Limit Laws）
題目：求 $\displaystyle\lim_{x \to 2}\dfrac{\sqrt{x} - \sqrt{2}}{x - 2}$。
解：直接代入得 $\frac{0}{0}$。乘以共軛：
$\dfrac{\sqrt{x}-\sqrt{2}}{x-2} \cdot \dfrac{\sqrt{x}+\sqrt{2}}{\sqrt{x}+\sqrt{2}} = \dfrac{x-2}{(x-2)(\sqrt{x}+\sqrt{2})} = \dfrac{1}{\sqrt{x}+\sqrt{2}}$
$\lim_{x \to 2}\dfrac{1}{\sqrt{x}+\sqrt{2}} = \dfrac{1}{2\sqrt{2}} = \dfrac{\sqrt{2}}{4}$
答：$\frac{\sqrt{2}}{4}$。
例題 2.21 — 求 $K/(x-a)$ 型極限（Evaluating a Limit of the Form $K/0$）
題目：求 $\displaystyle\lim_{x \to 2}\dfrac{x+3}{(x-2)^2}$。
解：
當 $x \to 2$ 時，分子 $\to 5$，分母 $\to 0^+$。因此 $\frac{5}{0^+} \to +\infty$。
答：$+\infty$。
例題 2.22 — 夾擠定理的應用（Applying the Squeeze Theorem）
題目：利用夾擠定理求 $\displaystyle\lim_{x \to 0}x^2\sin\!\big(\frac{1}{x}\big)$。
解：
已知 $-1 \le \sin(\frac{1}{x}) \le 1$，乘以 $x^2 \ge 0$：
$-x^2 \le x^2\sin(\frac{1}{x}) \le x^2$
$\lim_{x \to 0}(-x^2) = 0$ 且 $\lim_{x \to 0}x^2 = 0$。
由夾擠定理，$\lim_{x \to 0}x^2\sin(\frac{1}{x}) = 0$。
答：$0$。
例題 2.23 — 重要三角極限（Evaluating an Important Trigonometric Limit）
題目：求 $\displaystyle\lim_{\theta \to 0}\dfrac{1 - \cos\theta}{\theta}$。
解：
乘以共軛：$\dfrac{1-\cos\theta}{\theta} \cdot \dfrac{1+\cos\theta}{1+\cos\theta} = \dfrac{1-\cos^2\theta}{\theta(1+\cos\theta)} = \dfrac{\sin^2\theta}{\theta(1+\cos\theta)}$
$= \dfrac{\sin\theta}{\theta} \cdot \dfrac{\sin\theta}{1+\cos\theta}$
$\lim_{\theta \to 0}\dfrac{\sin\theta}{\theta} = 1$ 且 $\lim_{\theta \to 0}\dfrac{\sin\theta}{1+\cos\theta} = \dfrac{0}{2} = 0$
乘積為 $1 \times 0 = 0$。
答：$0$。
例題 2.24 — 三角函數的極限（Evaluating a Limit of a Trigonometric Function）
題目：求 $\displaystyle\lim_{x \to 0}\dfrac{\sin 5x}{x}$。
解：
$\dfrac{\sin 5x}{x} = 5 \cdot \dfrac{\sin 5x}{5x}$
令 $u = 5x$，當 $x \to 0$ 時 $u \to 0$。
$\lim_{x \to 0}5 \cdot \dfrac{\sin 5x}{5x} = 5 \cdot \lim_{u \to 0}\dfrac{\sin u}{u} = 5 \times 1 = 5$
答：$5$。
例題 2.25 — 三角函數極限進階（Another Trigonometric Limit）
題目：求 $\displaystyle\lim_{x \to 0}\dfrac{\tan x}{x}$。
解：
$\dfrac{\tan x}{x} = \dfrac{\sin x}{x} \cdot \dfrac{1}{\cos x}$
$\lim_{x \to 0}\dfrac{\tan x}{x} = \lim_{x \to 0}\dfrac{\sin x}{x} \cdot \lim_{x \to 0}\dfrac{1}{\cos x} = 1 \times 1 = 1$
答：$1$。

2.4 連續性（Continuity）

例題 2.26 — 判斷點上的連續性：條件 1（Determining Continuity at a Point, Condition 1）
題目：判斷 $f(x) = \dfrac{1}{x}$ 在 $x = 0$ 是否連續。
解：連續性三條件：
① $f(0)$ 必須有定義。但 $f(0) = \frac{1}{0}$ 未定義。
因此 $f$ 在 $x = 0$ 不連續（連第一個條件都不滿足）。
例題 2.27 — 判斷點上的連續性：條件 2（Determining Continuity at a Point, Condition 2）
題目：判斷 $f(x) = \begin{cases} \dfrac{x^2-1}{x-1} & x \neq 1 \\ 3 & x = 1 \end{cases}$ 在 $x = 1$ 是否連續。
解：
① $f(1) = 3$ ✓
② $\lim_{x \to 1}f(x) = \lim_{x \to 1}\frac{x^2-1}{x-1} = \lim_{x \to 1}(x+1) = 2$
③ $f(1) = 3 \neq 2 = \lim_{x \to 1}f(x)$ ✗
因此 $f$ 在 $x = 1$ 不連續（條件 3 不滿足：極限存在但不等于函數值）。
例題 2.28 — 判斷點上的連續性：條件 3（Determining Continuity at a Point, Condition 3）
題目：判斷 $f(x) = \begin{cases} \dfrac{x^2-1}{x-1} & x \neq 1 \\ 2 & x = 1 \end{cases}$ 在 $x = 1$ 是否連續。
解：
① $f(1) = 2$ ✓
② $\lim_{x \to 1}f(x) = 2$（同上題）✓
③ $\lim_{x \to 1}f(x) = 2 = f(1)$ ✓
因此 $f$ 在 $x = 1$ 連續。
例題 2.29 — 有理函數的連續性（Continuity of a Rational Function）
題目：判斷 $f(x) = \dfrac{x+1}{x^2-4}$ 在何處連續，何處不連續。
解：
分母 $x^2-4 = (x-2)(x+2) = 0$ 時，$x = \pm 2$。
有理函數在其定義域上處處連續。
答：$f$ 在 $(-\infty, -2) \cup (-2, 2) \cup (2, \infty)$ 上連續；在 $x = \pm 2$ 處不連續（無窮不連續點，infinite discontinuities）。
例題 2.30 — 不連續點分類（Classifying a Discontinuity）
題目：分類 $f(x) = \dfrac{x^2-1}{x-1}$ 在 $x = 1$ 的不連續點類型。
解：
$\lim_{x \to 1}f(x) = \lim_{x \to 1}\frac{(x-1)(x+1)}{x-1} = \lim_{x \to 1}(x+1) = 2$
極限存在（為 $2$），但 $f(1)$ 未定義或定義不等於 $2$。
這是可移除不連續點（removable discontinuity）。重新定義 $f(1)=2$ 即可修復。
例題 2.31 — 跳躍不連續點（Classifying a Discontinuity — Jump）
題目：分類 $f(x) = \begin{cases} x+1 & x < 1 \\ x^2 & x \ge 1 \end{cases}$ 在 $x = 1$ 的不連續性。
解：
$\lim_{x \to 1^-}f(x) = 1+1 = 2$
$\lim_{x \to 1^+}f(x) = 1^2 = 1$
$\lim_{x \to 1^-}f(x) = 2 \neq 1 = \lim_{x \to 1^+}f(x)$
左右極限存在但不相等 → 跳躍不連續點（jump discontinuity）。
例題 2.32 — 無窮不連續點（Classifying a Discontinuity — Infinite）
題目：分類 $f(x) = \dfrac{1}{(x-2)^2}$ 在 $x = 2$ 的不連續性。
解：
$\lim_{x \to 2}f(x) = +\infty$
極限為無窮大 → 無窮不連續點（infinite discontinuity）。
圖形在 $x = 2$ 處有垂直漸近線。
例題 2.33 — 區間上的連續性（Continuity on an Interval）
題目：判斷 $f(x) = \sqrt{4 - x^2}$ 在其定義域上是否連續。
解：
定義域：$4 - x^2 \ge 0 \implies x \in [-2, 2]$。
平方根函數在其定義域上連續，且 $4 - x^2$ 是多項式（處處連續），合成後在 $[-2, 2]$ 上連續。
答：$f$ 在 $[-2, 2]$ 上連續。
例題 2.34 — 區間連續性判定（Continuity over an Interval）
題目：求 $f(x) = \ln(x-1)$ 在其定義域上的連續區間。
解：
定義域：$x-1 > 0 \implies x > 1$。
對數函數在其定義域上處處連續。
答：$f$ 在 $(1, \infty)$ 上連續。
例題 2.35 — 合成函數的極限（Limit of a Composite Cosine Function）
題目：求 $\displaystyle\lim_{x \to \pi/2}\cos\!\big(\sin x\big)$。
解：
$\sin(\pi/2) = 1$，$\cos$ 在 $x=1$ 處連續。
由於合成函數連續（$\cos$ 和 $\sin$ 皆連續）：
$\lim_{x \to \pi/2}\cos(\sin x) = \cos(\sin(\pi/2)) = \cos 1 \approx 0.5403$
答：$\cos 1 \approx 0.5403$。

2.5 極限的精確定義（The Precise Definition of a Limit）

例題 2.36 — 中間值定理應用（Application of the Intermediate Value Theorem）
題目：證明方程式 $x^3 - x - 1 = 0$ 在 $[1, 2]$ 上有一個根。
解：
令 $f(x) = x^3 - x - 1$，$f$ 為多項式，在 $[1,2]$ 上連續。
$f(1) = 1 - 1 - 1 = -1 < 0$
$f(2) = 8 - 2 - 1 = 5 > 0$
由中間值定理（IVT），存在 $c \in (1, 2)$ 使得 $f(c) = 0$。
答：方程式在 $(1, 2)$ 內至少有一個根。
例題 2.37 — 中間值定理何時適用？（When Can You Apply the Intermediate Value Theorem?）
題目：函數 $f(x) = \dfrac{1}{x-2}$ 在 $[0, 4]$ 上，$f(0) = -\frac{1}{2}$，$f(4) = \frac{1}{2}$。能否用 IVT 證明存在 $c$ 使 $f(c) = 0$？
解：
IVT 要求函數在閉區間上連續。但 $f(x) = \frac{1}{x-2}$ 在 $x = 2$ 處有無窮不連續點，而 $2 \in [0, 4]$。
因此 $f$ 在 $[0, 4]$ 上不連續，不能使用 IVT。
（事實上 $\frac{1}{x-2} = 0$ 確實無解。）
例題 2.38 — 中間值定理不適用的情況（When Can You Not Apply the Intermediate Value Theorem?）
題目：$f(x) = \tan x$ 在 $[0, \pi]$ 上，$f(0) = 0$，$f(\pi) = 0$。能否用 IVT 證明存在 $c$ 使 $f(c) = 1$？
解：
$\tan x$ 在 $x = \pi/2 \in [0, \pi]$ 處不連續（垂直漸近線）。
因此 $f$ 在 $[0, \pi]$ 上不連續，不能使用 IVT。
（不過 $\tan x = 1$ 確實在 $x = \pi/4$ 有解，這只是 IVT 非必要條件。）
例題 2.39 — $\varepsilon$-$\delta$：證明特定函數的極限（Proving a Statement about the Limit of a Specific Function）
題目：使用 $\varepsilon$-$\delta$ 定義證明 $\displaystyle\lim_{x \to 3}(2x - 1) = 5$。
解：
對任意 $\varepsilon > 0$，需找到 $\delta > 0$ 使得 $0 < |x-3| < \delta \implies |(2x-1) - 5| < \varepsilon$。
$|(2x-1)-5| = |2x-6| = 2|x-3|$
要使 $2|x-3| < \varepsilon$，只需 $|x-3| < \varepsilon/2$。
取 $\delta = \varepsilon/2$。則當 $0 < |x-3| < \delta$ 時：
$|(2x-1)-5| = 2|x-3| < 2(\varepsilon/2) = \varepsilon$。
答：得證。
例題 2.40 — $\varepsilon$-$\delta$：證明一般極限（Proving a Statement about a Limit）
題目：使用 $\varepsilon$-$\delta$ 定義證明 $\displaystyle\lim_{x \to a}x = a$。
解：
對任意 $\varepsilon > 0$，需找到 $\delta > 0$ 使得 $0 < |x-a| < \delta \implies |x - a| < \varepsilon$。
顯然取 $\delta = \varepsilon$ 即可。
答：得證（取 $\delta = \varepsilon$）。
例題 2.41 — $\varepsilon$-$\delta$：幾何方法（Proving a Statement about the Limit of a Specific Function — Geometric Approach）
題目：使用 $\varepsilon$-$\delta$ 定義證明 $\displaystyle\lim_{x \to 1}x^2 = 1$。
解：
對任意 $\varepsilon > 0$，需 $|x^2 - 1| < \varepsilon$ 當 $0 < |x-1| < \delta$。
$|x^2-1| = |x-1||x+1|$
若限制 $\delta \le 1$，則 $|x-1| < 1 \implies 0 < x < 2 \implies 1 < x+1 < 3 \implies |x+1| < 3$。
因此 $|x^2-1| = |x-1||x+1| < 3|x-1|$。
要使 $3|x-1| < \varepsilon$，取 $\delta = \min(1, \varepsilon/3)$。
當 $0 < |x-1| < \delta$ 時，$|x^2-1| < \varepsilon$。
答：得證，取 $\delta = \min(1, \varepsilon/3)$。
例題 2.42 — $\varepsilon$-$\delta$：代數方法（Proving a Statement about the Limit of a Specific Function — Algebraic Approach）
題目：使用 $\varepsilon$-$\delta$ 定義證明 $\displaystyle\lim_{x \to 2}(x^2 - 3x) = -2$。
解：
對任意 $\varepsilon > 0$：
$|(x^2-3x) - (-2)| = |x^2 - 3x + 2| = |(x-1)(x-2)|$
限制 $\delta \le 1$，則 $|x-2| < 1 \implies 1 < x < 3 \implies 0 < x-1 < 2 \implies |x-1| < 2$。
因此 $|(x^2-3x)+2| = |x-1||x-2| < 2|x-2|$。
取 $\delta = \min(1, \varepsilon/2)$。則當 $0 < |x-2| < \delta$ 時：
$|(x^2-3x)+2| < 2\delta \le \varepsilon$。
答：得證，取 $\delta = \min(1, \varepsilon/2)$。
例題 2.43 — 證明極限不存在（Showing That a Limit Does Not Exist）
題目：使用 $\varepsilon$-$\delta$ 定義證明 $\displaystyle\lim_{x \to 0}\frac{|x|}{x}$ 不存在。
解：
假設極限存在為 $L$，取 $\varepsilon = \frac{1}{2}$。
對任意 $\delta > 0$：
　取 $x_1 = \delta/2 > 0$：$f(x_1) = 1$，$|f(x_1) - L| = |1 - L|$
　取 $x_2 = -\delta/2 < 0$：$f(x_2) = -1$，$|f(x_2) - L| = |-1 - L|$
若 $L \ge 0$，則 $|f(x_2)-L| = |-1-L| = 1+L \ge 1 > \varepsilon$。
若 $L < 0$，則 $|f(x_1)-L| = |1-L| > 1 > \varepsilon$。
無論 $L$ 為何值，都存在 $x$ 在 $(-\delta, \delta)$ 中使 $|f(x)-L| > \varepsilon$。
答：極限不存在。
例題 2.44 — 證明右極限（Proving a Statement about a Limit From the Right）
題目：使用 $\varepsilon$-$\delta$ 定義證明 $\displaystyle\lim_{x \to 0^+}\sqrt{x} = 0$。
解：
對任意 $\varepsilon > 0$，需找到 $\delta > 0$ 使得 $0 < x < \delta \implies |\sqrt{x} - 0| < \varepsilon$。
$|\sqrt{x}| = \sqrt{x} < \varepsilon \iff x < \varepsilon^2$。
取 $\delta = \varepsilon^2$。則當 $0 < x < \delta$ 時：
$\sqrt{x} < \sqrt{\varepsilon^2} = \varepsilon$。
答：得證，取 $\delta = \varepsilon^2$。
