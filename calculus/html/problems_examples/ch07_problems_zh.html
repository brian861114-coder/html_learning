7.1 分部積分法 (Integration by Parts)

🔑 本節重點：LIATE 法則、基本分部積分、重複分部積分、循環型分部積分、定積分分部積分、降冪公式

📝 例題 7.1
基本分部積分：∫ x eˣ dx
求 $\int x e^x\,dx$。
解：根據 LIATE 法則，$x$ 為代數函數 (A)，$e^x$ 為指數函數 (E)，A 優先於 E，故選 $u = x$、$dv = e^x dx$。則 $du = dx$、$v = e^x$。代入分部積分公式 $\int u\,dv = uv - \int v\,du$：
$$\int x e^x\,dx = x e^x - \int e^x\,dx = x e^x - e^x + C$$
✅ 答案：$e^x(x-1) + C$
💡 驗算：$\frac{d}{dx}[e^x(x-1)] = e^x(x-1) + e^x = xe^x$ ✓
📝 例題 7.2
對數函數的積分：∫ ln x dx
求 $\int \ln x\,dx$。
解：雖然只有一個函數，但可視為 $1 \cdot \ln x$。根據 LIATE，$\ln x$ 為對數函數 (L)，選 $u = \ln x$、$dv = dx$。則 $du = \frac{1}{x}dx$、$v = x$。
$$\int \ln x\,dx = x\ln x - \int x \cdot \frac{1}{x}\,dx = x\ln x - \int 1\,dx = x\ln x - x + C$$
✅ 答案：$x\ln x - x + C$
💡 這是重要的基本積分公式，可直接背誦。
📝 例題 7.3
重複分部積分：∫ x² eˣ dx
求 $\int x^2 e^x\,dx$。
解：第一次分部積分：令 $u = x^2$、$dv = e^x dx$，則 $du = 2x\,dx$、$v = e^x$。
$$\int x^2 e^x\,dx = x^2 e^x - \int 2x e^x\,dx = x^2 e^x - 2\int x e^x\,dx$$
第二次分部積分：對 $\int x e^x\,dx$ 使用分部積分（參考例題 7.1），得 $e^x(x-1)$。
$$\int x^2 e^x\,dx = x^2 e^x - 2e^x(x-1) + C = e^x(x^2 - 2x + 2) + C$$
✅ 答案：$e^x(x^2 - 2x + 2) + C$
💡 每分部積分一次，$x$ 的冪次降一次。
📝 例題 7.4
例外情況：∫ arctan x dx
求 $\int \arctan x\,dx$。
解：令 $u = \arctan x$、$dv = dx$。則 $du = \frac{1}{1+x^2}dx$、$v = x$。
$$\int \arctan x\,dx = x\arctan x - \int \frac{x}{1+x^2}\,dx$$
對第二項積分，令 $w = 1+x^2$，則 $dw = 2x\,dx$，即 $x\,dx = \frac{dw}{2}$：
$$\int \frac{x}{1+x^2}\,dx = \frac{1}{2}\int \frac{dw}{w} = \frac{1}{2}\ln|w| + C = \frac{1}{2}\ln(1+x^2) + C$$
故 $\int \arctan x\,dx = x\arctan x - \frac{1}{2}\ln(1+x^2) + C$。
✅ 答案：$x\arctan x - \frac{1}{2}\ln(1+x^2) + C$
📝 例題 7.5
循環型分部積分：∫ eˣ sin x dx
求 $\int e^x \sin x\,dx$。
解：設 $I = \int e^x \sin x\,dx$。
第一次分部積分：$u = \sin x$、$dv = e^x dx$，則 $du = \cos x\,dx$、$v = e^x$。
$$I = e^x\sin x - \int e^x \cos x\,dx$$
第二次分部積分：對 $\int e^x \cos x\,dx$，令 $u = \cos x$、$dv = e^x dx$，$du = -\sin x\,dx$、$v = e^x$。
$$\int e^x \cos x\,dx = e^x\cos x + \int e^x \sin x\,dx = e^x\cos x + I$$
代入：$I = e^x\sin x - (e^x\cos x + I) = e^x\sin x - e^x\cos x - I$
移項得 $2I = e^x(\sin x - \cos x)$，故 $I = \frac{e^x}{2}(\sin x - \cos x) + C$。
✅ 答案：$\frac{e^x}{2}(\sin x - \cos x) + C$
💡 循環型：原積分重新出現於等式右側，移項求解。
📝 例題 7.6
定積分的分部積分：∫₀¹ x e⁻ˣ dx
求區域 $y = xe^{-x}$ 與 $x$ 軸之間，在 $[0,1]$ 上的面積。
解：面積 $= \int_0^1 xe^{-x}\,dx$。令 $u = x$、$dv = e^{-x}dx$，則 $du = dx$、$v = -e^{-x}$。
$$\int_0^1 xe^{-x}\,dx = \Big[-xe^{-x}\Big]_0^1 - \int_0^1 (-e^{-x})\,dx = \Big[-xe^{-x} - e^{-x}\Big]_0^1$$
$$= \left[-\frac{x+1}{e^x}\right]_0^1 = -\frac{2}{e} - (-1) = 1 - \frac{2}{e} \approx 0.264$$
✅ 答案：$1 - \frac{2}{e} \approx 0.264$
📝 例題 7.7
降冪公式：∫ sinⁿ x dx 的推導
推導降冪公式 $\int \sin^n x\,dx = -\frac{1}{n}\sin^{n-1}x\cos x + \frac{n-1}{n}\int \sin^{n-2}x\,dx$。
解：將 $\sin^n x$ 寫為 $\sin^{n-1}x \cdot \sin x$。令 $u = \sin^{n-1}x$、$dv = \sin x\,dx$，則 $du = (n-1)\sin^{n-2}x\cos x\,dx$、$v = -\cos x$。
$$\int \sin^n x\,dx = -\sin^{n-1}x\cos x + (n-1)\int \sin^{n-2}x\cos^2 x\,dx$$
利用 $\cos^2 x = 1 - \sin^2 x$：
$$= -\sin^{n-1}x\cos x + (n-1)\int \sin^{n-2}x(1-\sin^2 x)\,dx$$
$$= -\sin^{n-1}x\cos x + (n-1)\int \sin^{n-2}x\,dx - (n-1)\int \sin^n x\,dx$$
移項：$n\int \sin^n x\,dx = -\sin^{n-1}x\cos x + (n-1)\int \sin^{n-2}x\,dx$，兩邊除以 $n$ 即得公式。
✅ 答案：$-\frac{1}{n}\sin^{n-1}x\cos x + \frac{n-1}{n}\int \sin^{n-2}x\,dx$

7.2 三角函數積分 (Trigonometric Integrals)

🔑 本節重點：sinᵐ cosⁿ 三種策略（奇次冪／偶次冪）、tanᵐ secⁿ 兩種策略、半角公式與降冪公式、積化和差公式、∫ sec³ x 循環型

📝 例題 7.8
∫ sin³ x dx
求 $\int \sin^3 x\,dx$。
解：$m=3$ 為奇數。保留一個 $\sin x$：$\sin^3 x = \sin^2 x \cdot \sin x = (1-\cos^2 x)\sin x$。令 $u = \cos x$、$du = -\sin x\,dx$：
$$\int \sin^3 x\,dx = \int (1-u^2)(-du) = \int (u^2-1)du = \frac{u^3}{3} - u + C = \frac{\cos^3 x}{3} - \cos x + C$$
✅ 答案：$\frac{\cos^3 x}{3} - \cos x + C$
📝 例題 7.9
∫ cos⁵ x dx
求 $\int \cos^5 x\,dx$。
解：$n=5$ 為奇數。保留一個 $\cos x$：$\cos^5 x = \cos^4 x \cdot \cos x = (1-\sin^2 x)^2\cos x$。令 $u = \sin x$、$du = \cos x\,dx$：
$$\int \cos^5 x\,dx = \int (1-u^2)^2\,du = \int (1-2u^2+u^4)du = u - \frac{2u^3}{3} + \frac{u^5}{5} + C$$
$$= \sin x - \frac{2\sin^3 x}{3} + \frac{\sin^5 x}{5} + C$$
✅ 答案：$\sin x - \frac{2\sin^3 x}{3} + \frac{\sin^5 x}{5} + C$
📝 例題 7.10
∫ sin² x dx
求 $\int \sin^2 x\,dx$。
解：$m=2$ 為偶數，使用半角公式 $\sin^2 x = \frac{1-\cos 2x}{2}$：
$$\int \sin^2 x\,dx = \int \frac{1-\cos 2x}{2}\,dx = \frac{1}{2}x - \frac{1}{4}\sin 2x + C$$
✅ 答案：$\frac{x}{2} - \frac{\sin 2x}{4} + C$
📝 例題 7.11
∫ cos⁴ x dx
求 $\int \cos^4 x\,dx$。
解：使用半角公式降冪：$\cos^4 x = (\cos^2 x)^2 = \left(\frac{1+\cos 2x}{2}\right)^2 = \frac{1+2\cos 2x+\cos^2 2x}{4}$。
對 $\cos^2 2x$ 再次降冪：$\cos^2 2x = \frac{1+\cos 4x}{2}$。
$$\int \cos^4 x\,dx = \frac{1}{4}\int \left(1+2\cos 2x+\frac{1+\cos 4x}{2}\right)dx$$
$$= \frac{1}{4}\int \left(\frac{3}{2}+2\cos 2x+\frac{\cos 4x}{2}\right)dx = \frac{3x}{8} + \frac{\sin 2x}{4} + \frac{\sin 4x}{32} + C$$
✅ 答案：$\frac{3x}{8} + \frac{\sin 2x}{4} + \frac{\sin 4x}{32} + C$
📝 例題 7.12
∫ sin² x cos² x dx
求 $\int \sin^2 x \cos^2 x\,dx$。
解：$$\sin^2 x \cos^2 x = \frac{1-\cos 2x}{2} \cdot \frac{1+\cos 2x}{2} = \frac{1-\cos^2 2x}{4} = \frac{\sin^2 2x}{4}$$ 再降冪：$\frac{\sin^2 2x}{4} = \frac{1-\cos 4x}{8}$。
$$\int \sin^2 x \cos^2 x\,dx = \frac{1}{8}\int (1-\cos 4x)dx = \frac{x}{8} - \frac{\sin 4x}{32} + C$$
✅ 答案：$\frac{x}{8} - \frac{\sin 4x}{32} + C$
📝 例題 7.13
∫ sin(3x) cos(2x) dx
求 $\int \sin(3x)\cos(2x)\,dx$。
解：使用積化和差公式 $\sin A\cos B = \frac{1}{2}[\sin(A-B) + \sin(A+B)]$：
$\sin(3x)\cos(2x) = \frac{1}{2}[\sin x + \sin(5x)]$。
$$\int \sin(3x)\cos(2x)\,dx = \frac{1}{2}\int (\sin x + \sin 5x)dx = -\frac{1}{2}\cos x - \frac{1}{10}\cos 5x + C$$
✅ 答案：$-\frac{1}{2}\cos x - \frac{1}{10}\cos 5x + C$
📝 例題 7.14
∫ tan³ x sec⁴ x dx
求 $\int \tan^3 x \sec^4 x\,dx$。
解：$n=4$ 為偶數。保留 $\sec^2 x$：$\tan^3 x \sec^4 x = \tan^3 x \sec^2 x \cdot \sec^2 x = \tan^3 x (\tan^2 x+1)\sec^2 x$。令 $u = \tan x$、$du = \sec^2 x\,dx$：
$$\int u^3(u^2+1)du = \int (u^5+u^3)du = \frac{u^6}{6} + \frac{u^4}{4} + C = \frac{\tan^6 x}{6} + \frac{\tan^4 x}{4} + C$$
✅ 答案：$\frac{\tan^6 x}{6} + \frac{\tan^4 x}{4} + C$
📝 例題 7.15
∫ tan x sec³ x dx
求 $\int \tan x \sec^3 x\,dx$。
解：保留 $\sec x\tan x$：$\tan x\sec^3 x = \sec^2 x \cdot \sec x\tan x$。令 $u = \sec x$、$du = \sec x\tan x\,dx$：
$$\int \tan x \sec^3 x\,dx = \int u^2\,du = \frac{u^3}{3} + C = \frac{\sec^3 x}{3} + C$$
✅ 答案：$\frac{\sec^3 x}{3} + C$
📝 例題 7.16
∫ tan⁴ x dx
求 $\int \tan^4 x\,dx$。
解：$\tan^4 x = \tan^2 x \cdot \tan^2 x = \tan^2 x(\sec^2 x-1) = \tan^2 x\sec^2 x - \tan^2 x$。
對第一項：令 $u = \tan x$、$du = \sec^2 x\,dx$，得 $\int \tan^2 x\sec^2 x\,dx = \frac{\tan^3 x}{3}$。
對第二項：$\int \tan^2 x\,dx = \int (\sec^2 x-1)dx = \tan x - x$。
$$\int \tan^4 x\,dx = \frac{\tan^3 x}{3} - \tan x + x + C$$
✅ 答案：$\frac{\tan^3 x}{3} - \tan x + x + C$
📝 例題 7.17
∫ sec x dx 的推導
推導 $\int \sec x\,dx = \ln|\sec x + \tan x| + C$。
解：使用巧妙技巧：分子分母同乘 $(\sec x + \tan x)$：
$$\int \sec x\,dx = \int \sec x \cdot \frac{\sec x + \tan x}{\sec x + \tan x}\,dx = \int \frac{\sec^2 x + \sec x\tan x}{\sec x + \tan x}\,dx$$
令 $u = \sec x + \tan x$，則 $du = (\sec x\tan x + \sec^2 x)dx$，恰好是分子。因此：
$$\int \frac{du}{u} = \ln|u| + C = \ln|\sec x + \tan x| + C$$
✅ 答案：$\ln|\sec x + \tan x| + C$
📝 例題 7.18
∫ sec³ x dx
求 $\int \sec^3 x\,dx$（循環型分部積分）。
解：令 $u = \sec x$、$dv = \sec^2 x\,dx$，則 $du = \sec x\tan x\,dx$、$v = \tan x$。
$$\int \sec^3 x\,dx = \sec x\tan x - \int \sec x\tan^2 x\,dx$$
利用 $\tan^2 x = \sec^2 x - 1$：$$= \sec x\tan x - \int \sec x(\sec^2 x - 1)dx$$
$$= \sec x\tan x - \int \sec^3 x\,dx + \int \sec x\,dx$$
移項：$2\int \sec^3 x\,dx = \sec x\tan x + \ln|\sec x + \tan x| + C$
$$\int \sec^3 x\,dx = \frac{1}{2}\sec x\tan x + \frac{1}{2}\ln|\sec x + \tan x| + C$$
✅ 答案：$\frac{1}{2}\sec x\tan x + \frac{1}{2}\ln|\sec x + \tan x| + C$
💡 此為經典循環型積分，結果常用於後續三角代換中。

7.3 三角代換 (Trigonometric Substitution)

🔑 本節重點：√(a²−x²)→x=a sin θ、√(a²+x²)→x=a tan θ、√(x²−a²)→x=a sec θ、參考三角形還原變數

📝 例題 7.19
∫ √(9−x²) dx
求 $\int \sqrt{9-x^2}\,dx$。
解：根號型式 $\sqrt{a^2-x^2}$，其中 $a=3$。令 $x = 3\sin\theta$，則 $dx = 3\cos\theta\,d\theta$，且 $\sqrt{9-x^2} = \sqrt{9-9\sin^2\theta} = 3\cos\theta$（$\theta \in [-\pi/2,\pi/2]$）。
$$\int \sqrt{9-x^2}\,dx = \int 3\cos\theta \cdot 3\cos\theta\,d\theta = 9\int \cos^2\theta\,d\theta$$
使用半角公式：$= 9\int \frac{1+\cos 2\theta}{2}d\theta = \frac{9}{2}\left(\theta + \frac{\sin 2\theta}{2}\right) + C$
由參考三角形：$\sin\theta = x/3$，$\theta = \arcsin(x/3)$，$\cos\theta = \sqrt{9-x^2}/3$，$\sin 2\theta = 2\sin\theta\cos\theta = \frac{2x\sqrt{9-x^2}}{9}$。
$$= \frac{9}{2}\arcsin\frac{x}{3} + \frac{x\sqrt{9-x^2}}{2} + C$$
✅ 答案：$\frac{9}{2}\arcsin\frac{x}{3} + \frac{x\sqrt{9-x^2}}{2} + C$
📝 例題 7.20
∫ dx/(x²√(4−x²))
求 $\int \frac{dx}{x^2\sqrt{4-x^2}}$。
解：令 $x = 2\sin\theta$，則 $dx = 2\cos\theta\,d\theta$，$\sqrt{4-x^2} = 2\cos\theta$。
$$\int \frac{dx}{x^2\sqrt{4-x^2}} = \int \frac{2\cos\theta\,d\theta}{4\sin^2\theta \cdot 2\cos\theta} = \int \frac{d\theta}{4\sin^2\theta} = \frac{1}{4}\int \csc^2\theta\,d\theta$$
$$= -\frac{1}{4}\cot\theta + C$$
由參考三角形：$\cot\theta = \frac{\sqrt{4-x^2}}{x}$。
$$= -\frac{\sqrt{4-x^2}}{4x} + C$$
✅ 答案：$-\frac{\sqrt{4-x^2}}{4x} + C$
📝 例題 7.21
∫ dx/(x²+4)
求 $\int \frac{dx}{x^2+4}$。
解：使用 $x = 2\tan\theta$（$a=2$），則 $dx = 2\sec^2\theta\,d\theta$，$x^2+4 = 4\tan^2\theta+4 = 4\sec^2\theta$。
$$\int \frac{dx}{x^2+4} = \int \frac{2\sec^2\theta\,d\theta}{4\sec^2\theta} = \int \frac{1}{2}d\theta = \frac{\theta}{2} + C = \frac{1}{2}\arctan\frac{x}{2} + C$$
✅ 答案：$\frac{1}{2}\arctan\frac{x}{2} + C$
📝 例題 7.22
∫ √(x²+4)/x dx
求 $\int \frac{\sqrt{x^2+4}}{x}\,dx$。
解：令 $x = 2\tan\theta$，則 $dx = 2\sec^2\theta\,d\theta$，$\sqrt{x^2+4} = 2\sec\theta$。
$$\int \frac{2\sec\theta}{2\tan\theta} \cdot 2\sec^2\theta\,d\theta = 2\int \frac{\sec^3\theta}{\tan\theta}\,d\theta = 2\int \csc\theta\sec^2\theta\,d\theta$$
可寫為 $2\int \csc\theta(1+\tan^2\theta)d\theta$。經分部積分或使用三角恆等式化簡後：
$$= 2\ln|\csc\theta-\cot\theta| + 2\sec\theta + C$$
還原：$\sec\theta = \frac{\sqrt{x^2+4}}{2}$，$\csc\theta = \frac{\sqrt{x^2+4}}{x}$，$\cot\theta = \frac{2}{x}$。
$$= 2\ln\left|\frac{\sqrt{x^2+4}-2}{x}\right| + \sqrt{x^2+4} + C$$
✅ 答案：$2\ln\left|\frac{\sqrt{x^2+4}-2}{x}\right| + \sqrt{x^2+4} + C$
📝 例題 7.23
∫ dx/(x²√(x²−1))
求 $\int \frac{dx}{x^2\sqrt{x^2-1}}$（$x>1$）。
解：使用 $x = \sec\theta$，則 $dx = \sec\theta\tan\theta\,d\theta$，$\sqrt{x^2-1} = \tan\theta$（$x>1$ 時 $\tan\theta>0$）。
$$\int \frac{dx}{x^2\sqrt{x^2-1}} = \int \frac{\sec\theta\tan\theta\,d\theta}{\sec^2\theta \cdot \tan\theta} = \int \cos\theta\,d\theta = \sin\theta + C$$
由參考三角形：$\sec\theta = x$，$\sin\theta = \frac{\sqrt{x^2-1}}{x}$。
$$= \frac{\sqrt{x^2-1}}{x} + C$$
✅ 答案：$\frac{\sqrt{x^2-1}}{x} + C$
📝 例題 7.24
∫ √(x²−a²)/x⁴ dx
求 $\int \frac{\sqrt{x^2-a^2}}{x^4}\,dx$（$a>0$、$x>a$）。
解：令 $x = a\sec\theta$，$dx = a\sec\theta\tan\theta\,d\theta$，$\sqrt{x^2-a^2} = a\tan\theta$。
$$\int \frac{a\tan\theta}{a^4\sec^4\theta} \cdot a\sec\theta\tan\theta\,d\theta = \frac{1}{a^2}\int \frac{\tan^2\theta}{\sec^3\theta}\,d\theta = \frac{1}{a^2}\int \sin^2\theta\cos\theta\,d\theta$$
令 $w = \sin\theta$、$dw = \cos\theta\,d\theta$：$$= \frac{1}{a^2}\int w^2\,dw = \frac{w^3}{3a^2} + C = \frac{\sin^3\theta}{3a^2} + C$$
由參考三角形：$\sin\theta = \frac{\sqrt{x^2-a^2}}{x}$。
$$= \frac{(x^2-a^2)^{3/2}}{3a^2 x^3} + C$$
✅ 答案：$\frac{(x^2-a^2)^{3/2}}{3a^2 x^3} + C$

7.4 部分分式 (Partial Fractions)

🔑 本節重點：有理函數分解、線性因式、重複線性因式、二次不可約因式、重複二次因式、多項式長除法

📝 例題 7.25
∫ (x+1)/(x²−1) dx
求 $\int \frac{x+1}{x^2-1}\,dx$。
解：分解分母：$x^2-1 = (x-1)(x+1)$。設 $\frac{x+1}{(x-1)(x+1)} = \frac{A}{x-1} + \frac{B}{x+1}$。
化簡左側：$\frac{x+1}{(x-1)(x+1)} = \frac{1}{x-1}$（因為 $x+1$ 可約分）。
若不分約分而直接分解：$x+1 = A(x+1) + B(x-1)$。
令 $x=1$：$2 = 2A$，得 $A=1$。令 $x=-1$：$0 = -2B$，得 $B=0$。
因此 $\frac{x+1}{x^2-1} = \frac{1}{x-1}$。
$$\int \frac{dx}{x-1} = \ln|x-1| + C$$
✅ 答案：$\ln|x-1| + C$
📝 例題 7.26
∫ (x²+2)/(x³−x) dx
求 $\int \frac{x^2+2}{x^3-x}\,dx$。
解：分解分母：$x^3-x = x(x-1)(x+1)$。設 $\frac{x^2+2}{x(x-1)(x+1)} = \frac{A}{x} + \frac{B}{x-1} + \frac{C}{x+1}$。
$x^2+2 = A(x-1)(x+1) + Bx(x+1) + Cx(x-1)$。
令 $x=0$：$2 = -A$，$A=-2$。令 $x=1$：$3 = 2B$，$B=3/2$。令 $x=-1$：$3 = 2C$，$C=3/2$。
$$\int \left(-\frac{2}{x} + \frac{3/2}{x-1} + \frac{3/2}{x+1}\right)dx = -2\ln|x| + \frac{3}{2}\ln|x-1| + \frac{3}{2}\ln|x+1| + C$$
✅ 答案：$-2\ln|x| + \frac{3}{2}\ln|x-1| + \frac{3}{2}\ln|x+1| + C$
📝 例題 7.27
∫ dx/(x²−1)
求 $\int \frac{dx}{x^2-1}$。
解：$x^2-1 = (x-1)(x+1)$。設 $\frac{1}{x^2-1} = \frac{A}{x-1} + \frac{B}{x+1}$。
$1 = A(x+1) + B(x-1)$。令 $x=1$：$1=2A$，$A=1/2$。令 $x=-1$：$1=-2B$，$B=-1/2$。
$$\int \left(\frac{1/2}{x-1} - \frac{1/2}{x+1}\right)dx = \frac{1}{2}\ln|x-1| - \frac{1}{2}\ln|x+1| + C = \frac{1}{2}\ln\left|\frac{x-1}{x+1}\right| + C$$
✅ 答案：$\frac{1}{2}\ln\left|\frac{x-1}{x+1}\right| + C$
📝 例題 7.28
∫ (x³)/(x²+1) dx（長除法）
求 $\int \frac{x^3}{x^2+1}\,dx$。
解：分子次數 ≥ 分母次數，先做長除法：$x^3 \div (x^2+1) = x - \frac{x}{x^2+1}$。
$$\int \frac{x^3}{x^2+1}\,dx = \int x\,dx - \int \frac{x}{x^2+1}\,dx$$
第一項：$\frac{x^2}{2}$。第二項：令 $u = x^2+1$、$du = 2x\,dx$，$\int \frac{x\,dx}{x^2+1} = \frac{1}{2}\ln|x^2+1|$。
$$= \frac{x^2}{2} - \frac{1}{2}\ln(x^2+1) + C$$
✅ 答案：$\frac{x^2}{2} - \frac{1}{2}\ln(x^2+1) + C$
📝 例題 7.29
∫ dx/(x³+x)（二次不可約因式）
求 $\int \frac{dx}{x^3+x}$。
解：$x^3+x = x(x^2+1)$。設 $\frac{1}{x(x^2+1)} = \frac{A}{x} + \frac{Bx+C}{x^2+1}$。
$1 = A(x^2+1) + (Bx+C)x = (A+B)x^2 + Cx + A$。
比較係數：$x^2$ 項：$A+B=0$；$x$ 項：$C=0$；常數項：$A=1$。得 $A=1$、$B=-1$、$C=0$。
$$\int \left(\frac{1}{x} - \frac{x}{x^2+1}\right)dx = \ln|x| - \frac{1}{2}\ln(x^2+1) + C$$
✅ 答案：$\ln|x| - \frac{1}{2}\ln(x^2+1) + C$
📝 例題 7.30
∫ (2x³−x²)/(x⁴+x²) dx
求 $\int \frac{2x^3-x^2}{x^4+x^2}\,dx$。
解：分母 $x^4+x^2 = x^2(x^2+1)$。約分 $x^2$：$\frac{2x^3-x^2}{x^2(x^2+1)} = \frac{x(2x-1)}{x^2+1}?$ 應直接分解。
設 $\frac{2x^3-x^2}{x^2(x^2+1)} = \frac{A}{x} + \frac{B}{x^2} + \frac{Cx+D}{x^2+1}$。
$2x^3-x^2 = Ax(x^2+1) + B(x^2+1) + (Cx+D)x^2$。
展開比較係數：$x^3$：$A+C=2$；$x^2$：$B+D=-1$；$x$：$A=0$；常數：$B=0$。得 $A=0$、$B=0$、$C=2$、$D=-1$。
$$\int \frac{2x-1}{x^2+1}\,dx = \int \frac{2x}{x^2+1}\,dx - \int \frac{dx}{x^2+1} = \ln(x^2+1) - \arctan x + C$$
✅ 答案：$\ln(x^2+1) - \arctan x + C$
📝 例題 7.31
∫ (x²+2x−1)/(x³−1) dx
求 $\int \frac{x^2+2x-1}{x^3-1}\,dx$。
解：$x^3-1 = (x-1)(x^2+x+1)$（$x^2+x+1$ 為二次不可約因式，判別式 $1-4=-3<0$）。
設 $\frac{x^2+2x-1}{(x-1)(x^2+x+1)} = \frac{A}{x-1} + \frac{Bx+C}{x^2+x+1}$。
$x^2+2x-1 = A(x^2+x+1) + (Bx+C)(x-1)$。
令 $x=1$：$1+2-1=2 = A(3)$，$A=2/3$。
比較係數（或代 $x=0$、$x=-1$）：得 $B=1/3$、$C=5/3$。
$$\int \left(\frac{2/3}{x-1} + \frac{\frac{1}{3}x+\frac{5}{3}}{x^2+x+1}\right)dx$$
後者配方法：$x^2+x+1 = (x+\frac{1}{2})^2 + \frac{3}{4}$，經代換後可積出涉及 $\ln$ 與 $\arctan$ 的組合。
$$= \frac{2}{3}\ln|x-1| + \frac{1}{6}\ln(x^2+x+1) + \frac{2\sqrt{3}}{3}\arctan\frac{2x+1}{\sqrt{3}} + C$$
✅ 答案：$\frac{2}{3}\ln|x-1| + \frac{1}{6}\ln(x^2+x+1) + \frac{2\sqrt{3}}{3}\arctan\frac{2x+1}{\sqrt{3}} + C$

7.5 其他積分策略 (Other Strategies for Integration)

🔑 本節重點：積分表的使用、電腦代數系統 (CAS)、綜合策略選用

📝 例題 7.32
使用積分表：∫ dx/(x²√(4x²+9))
使用積分表求 $\int \frac{dx}{x^2\sqrt{4x^2+9}}$。
解：整理為標準型：$\int \frac{dx}{x^2\sqrt{4x^2+9}} = \int \frac{dx}{x^2\cdot 2\sqrt{x^2+\frac{9}{4}}} = \frac{1}{2}\int \frac{dx}{x^2\sqrt{x^2+(3/2)^2}}$。
查積分公式 $\int \frac{dx}{x^2\sqrt{x^2+a^2}} = -\frac{\sqrt{x^2+a^2}}{a^2x} + C$，取 $a = \frac{3}{2}$：
$$= \frac{1}{2}\left(-\frac{\sqrt{x^2+\frac{9}{4}}}{\frac{9}{4}\cdot x}\right) + C = -\frac{2}{9}\cdot\frac{\sqrt{x^2+\frac{9}{4}}}{x} + C = -\frac{\sqrt{4x^2+9}}{9x} + C$$
✅ 答案：$-\frac{\sqrt{4x^2+9}}{9x} + C$
📝 例題 7.33
∫ sin⁵ x cos² x dx（策略選擇）
求 $\int \sin^5 x \cos^2 x\,dx$。
解：$m=5$（奇），使用策略 1：保留 $\sin x$，其餘轉為 $\cos x$。
$\sin^5 x \cos^2 x = \sin^4 x \cos^2 x \cdot \sin x = (1-\cos^2 x)^2 \cos^2 x \cdot \sin x$。
令 $u = \cos x$、$du = -\sin x\,dx$：
$$\int (1-u^2)^2 u^2 (-du) = -\int (u^2 - 2u^4 + u^6)du = -\frac{u^3}{3} + \frac{2u^5}{5} - \frac{u^7}{7} + C$$
$$= -\frac{\cos^3 x}{3} + \frac{2\cos^5 x}{5} - \frac{\cos^7 x}{7} + C$$
✅ 答案：$-\frac{\cos^3 x}{3} + \frac{2\cos^5 x}{5} - \frac{\cos^7 x}{7} + C$
📝 例題 7.34
∫ x/(x²+1)² dx（代換法）
求 $\int \frac{x}{(x^2+1)^2}\,dx$。
解：令 $u = x^2+1$，則 $du = 2x\,dx$，即 $x\,dx = \frac{du}{2}$。
$$\int \frac{x}{(x^2+1)^2}\,dx = \int \frac{du/2}{u^2} = \frac{1}{2}\int u^{-2}du = \frac{1}{2}\cdot\frac{u^{-1}}{-1} + C = -\frac{1}{2u} + C = -\frac{1}{2(x^2+1)} + C$$
✅ 答案：$-\frac{1}{2(x^2+1)} + C$
💡 有時簡單代換即可解決，無需複雜技巧。
📝 例題 7.35
∫ sin² x cos³ x dx（綜合）
求 $\int \sin^2 x \cos^3 x\,dx$。
解：$n=3$（奇），保留 $\cos x$：$\sin^2 x \cos^3 x = \sin^2 x \cos^2 x \cdot \cos x = \sin^2 x(1-\sin^2 x)\cos x$。
令 $u = \sin x$、$du = \cos x\,dx$：
$$\int u^2(1-u^2)du = \int (u^2-u^4)du = \frac{u^3}{3} - \frac{u^5}{5} + C = \frac{\sin^3 x}{3} - \frac{\sin^5 x}{5} + C$$
✅ 答案：$\frac{\sin^3 x}{3} - \frac{\sin^5 x}{5} + C$
📝 例題 7.36
∫ x ln(x) dx（分部積分策略）
求 $\int x \ln x\,dx$。
解：根據 LIATE，$\ln x$ (L) 優先於 $x$ (A)。令 $u = \ln x$、$dv = x\,dx$，則 $du = \frac{dx}{x}$、$v = \frac{x^2}{2}$。
$$\int x\ln x\,dx = \frac{x^2}{2}\ln x - \int \frac{x^2}{2}\cdot\frac{1}{x}\,dx = \frac{x^2}{2}\ln x - \frac{1}{2}\int x\,dx$$
$$= \frac{x^2}{2}\ln x - \frac{x^2}{4} + C$$
✅ 答案：$\frac{x^2}{2}\ln x - \frac{x^2}{4} + C$

7.6 數值積分 (Numerical Integration)

🔑 本節重點：中點法則、梯形法則、辛普森法則、誤差估計

📝 例題 7.37
中點法則：∫₀¹ √(1+x²) dx，n=4
使用中點法則，取 $n=4$，近似計算 $\int_0^1 \sqrt{1+x^2}\,dx$。
解：$\Delta x = \frac{1-0}{4} = 0.25$。子區間中點：$0.125, 0.375, 0.625, 0.875$。
$$M_4 = 0.25\Big[\sqrt{1+(0.125)^2} + \sqrt{1+(0.375)^2} + \sqrt{1+(0.625)^2} + \sqrt{1+(0.875)^2}\Big]$$
$$\approx 0.25[1.0078 + 1.0680 + 1.1792 + 1.3288] = 0.25 \times 4.5838 = 1.1459$$
✅ 答案：$M_4 \approx 1.1459$
📝 例題 7.38
梯形法則：∫₀¹ √(1+x²) dx，n=4
使用梯形法則，取 $n=4$，近似計算 $\int_0^1 \sqrt{1+x^2}\,dx$。
解：$\Delta x = 0.25$。端點值：$f(0)=1$、$f(0.25)\approx 1.0308$、$f(0.5)\approx 1.1180$、$f(0.75)\approx 1.2500$、$f(1)\approx 1.4142$。
$$T_4 = \frac{0.25}{2}\Big[1 + 2(1.0308+1.1180+1.2500) + 1.4142\Big]$$
$$= 0.125[1 + 6.7976 + 1.4142] = 0.125 \times 9.2118 = 1.1515$$
✅ 答案：$T_4 \approx 1.1515$
📝 例題 7.39
辛普森法則：∫₀¹ √(1+x²) dx，n=4
使用辛普森法則，取 $n=4$，近似計算 $\int_0^1 \sqrt{1+x^2}\,dx$。
解：$\Delta x = 0.25$。
$$S_4 = \frac{0.25}{3}\Big[1 + 4(1.0308+1.2500) + 2(1.1180) + 1.4142\Big]$$
$$= \frac{0.25}{3}[1 + 9.1232 + 2.2360 + 1.4142] = \frac{0.25}{3} \times 13.7734 = 1.1478$$
（精確值 $\approx 1.1478$，辛普森法則在此例中幾乎給出精確答案）
✅ 答案：$S_4 \approx 1.1478$
📝 例題 7.40
梯形法則誤差：∫₀¹ xeˣ dx
求使用梯形法則近似 $\int_0^1 xe^x\,dx$ 時，$n=10$ 的誤差界限。
解：梯形法則誤差界限：$|E_T| \leq \frac{K(b-a)^3}{12n^2}$，其中 $K$ 為 $|f''(x)|$ 在 $[a,b]$ 上的最大值。
$f(x)=xe^x$，$f'(x)=e^x(x+1)$，$f''(x)=e^x(x+2)$。在 $[0,1]$ 上，$f''(x)$ 遞增，最大值在 $x=1$：$f''(1)=3e \approx 8.155$。
$$|E_T| \leq \frac{8.155 \cdot (1-0)^3}{12 \cdot 10^2} = \frac{8.155}{1200} \approx 0.00680$$
✅ 答案：$|E_T| \leq 0.00680$
📝 例題 7.41
辛普森誤差：∫₁² dx/x
求使用辛普森法則近似 $\int_1^2 \frac{dx}{x}$ 時，$n=6$ 的誤差界限。
解：辛普森誤差界限：$|E_S| \leq \frac{K(b-a)^5}{180n^4}$。
$f(x)=1/x$，$f^{(4)}(x)=24/x^5$。在 $[1,2]$ 上，$|f^{(4)}(x)|$ 的最大值在 $x=1$：$|f^{(4)}(1)|=24$。
$$|E_S| \leq \frac{24 \cdot (2-1)^5}{180 \cdot 6^4} = \frac{24}{180 \cdot 1296} = \frac{24}{233280} \approx 0.000103$$
（精確值 $\ln 2 \approx 0.693147$）
✅ 答案：$|E_S| \leq 0.000103$
📝 例題 7.42
決定 n 使梯形法則滿足精度
求 $n$ 使得梯形法則近似 $\int_0^2 e^{-x^2}\,dx$ 的誤差 ≤ 0.0001。
解：$|E_T| \leq \frac{K(b-a)^3}{12n^2} \leq 0.0001$。需找 $K = \max|f''(x)|$。$f(x)=e^{-x^2}$、$f''(x)=2e^{-x^2}(2x^2-1)$。$|f''(x)| \leq 2$（可由分析得）。
$$\frac{2 \cdot 8}{12n^2} \leq 0.0001 \Rightarrow \frac{16}{12n^2} \leq 0.0001 \Rightarrow n^2 \geq \frac{16}{0.0012} \approx 13333 \Rightarrow n \geq 116$$
取 $n = 116$。
✅ 答案：$n \geq 116$
📝 例題 7.43
∫₀¹ eˣ² dx 的中點法則，n=6
使用中點法則，取 $n=6$，近似計算 $\int_0^1 e^{x^2}\,dx$。
解：$\Delta x = \frac{1}{6}$。中點：$\frac{1}{12}, \frac{3}{12}, \frac{5}{12}, \frac{7}{12}, \frac{9}{12}, \frac{11}{12}$。
$$M_6 = \frac{1}{6}\Big[e^{(1/12)^2} + e^{(3/12)^2} + e^{(5/12)^2} + e^{(7/12)^2} + e^{(9/12)^2} + e^{(11/12)^2}\Big]$$
$$\approx \frac{1}{6}[1.0070 + 1.0645 + 1.1890 + 1.4055 + 1.7551 + 2.3164] \approx 1.4563$$
✅ 答案：$M_6 \approx 1.4563$
📝 例題 7.44
辛普森法則：∫₀² sin(x²) dx，n=4
使用辛普森法則，取 $n=4$，近似計算 $\int_0^2 \sin(x^2)\,dx$。
解：$\Delta x = 0.5$。$f(0)=0$, $f(0.5)\approx 0.2474$, $f(1)\approx 0.8415$, $f(1.5)\approx 0.7781$, $f(2)\approx -0.7568$。
$$S_4 = \frac{0.5}{3}\Big[0 + 4(0.2474+0.7781) + 2(0.8415) + (-0.7568)\Big]$$
$$= \frac{0.5}{3}[0 + 4.1020 + 1.6830 - 0.7568] = \frac{0.5}{3} \times 5.0282 \approx 0.8380$$
✅ 答案：$S_4 \approx 0.8380$

7.7 瑕積分 (Improper Integrals)

🔑 本節重點：無窮區間瑕積分、不連續被積函數瑕積分、比較定理、收斂／發散判定

📝 例題 7.45
∫₁^∞ dx/x²（收斂）
求 $\int_1^\infty \frac{dx}{x^2}$。
解：定義：$\int_1^\infty \frac{dx}{x^2} = \lim_{t\to\infty} \int_1^t x^{-2}\,dx = \lim_{t\to\infty} \left[-\frac{1}{x}\right]_1^t = \lim_{t\to\infty} \left(-\frac{1}{t} + 1\right) = 1$。
✅ 答案：$1$（收斂）
📝 例題 7.46
∫₁^∞ dx/x（發散）
求 $\int_1^\infty \frac{dx}{x}$。
解：$\int_1^\infty \frac{dx}{x} = \lim_{t\to\infty} \int_1^t \frac{dx}{x} = \lim_{t\to\infty} \ln|t| = \infty$。
✅ 答案：發散（$\to\infty$）
📝 例題 7.47
∫₀^∞ e⁻ˣ dx
求 $\int_0^\infty e^{-x}\,dx$。
解：$\lim_{t\to\infty} \int_0^t e^{-x}\,dx = \lim_{t\to\infty} \Big[-e^{-x}\Big]_0^t = \lim_{t\to\infty} (-e^{-t} + 1) = 1$。
✅ 答案：$1$（收斂）
📝 例題 7.48
∫₋∞^∞ dx/(1+x²)
求 $\int_{-\infty}^\infty \frac{dx}{1+x^2}$。
解：利用對稱性：$\int_{-\infty}^\infty \frac{dx}{1+x^2} = 2\int_0^\infty \frac{dx}{1+x^2}$。
$= 2\lim_{t\to\infty} \Big[\arctan x\Big]_0^t = 2\lim_{t\to\infty} \arctan t = 2 \cdot \frac{\pi}{2} = \pi$。
✅ 答案：$\pi$（收斂）
📝 例題 7.49
∫₀^∞ xe⁻ˣ dx（Gamma 函數）
求 $\int_0^\infty x e^{-x}\,dx$。
解：使用分部積分：$u=x$、$dv=e^{-x}dx$，$du=dx$、$v=-e^{-x}$。
$\int_0^\infty xe^{-x}\,dx = \lim_{t\to\infty} \Big[-xe^{-x}\Big]_0^t + \int_0^\infty e^{-x}\,dx$
$= \lim_{t\to\infty} (-te^{-t}) + 0 + 1 = 0 + 1 = 1$。（$\lim_{t\to\infty} te^{-t}=0$）
✅ 答案：$1$（收斂）
💡 此為 $\Gamma(2)$，Gamma 函數 $\Gamma(n) = \int_0^\infty x^{n-1}e^{-x}dx$，且 $\Gamma(n)=(n-1)!$。
📝 例題 7.50
∫₀^¹ dx/√x（不連續點）
求 $\int_0^1 \frac{dx}{\sqrt{x}}$。
解：$x=0$ 處被積函數無界。$\int_0^1 x^{-1/2}\,dx = \lim_{t\to 0^+} \int_t^1 x^{-1/2}\,dx = \lim_{t\to 0^+} \Big[2\sqrt{x}\Big]_t^1 = \lim_{t\to 0^+} (2 - 2\sqrt{t}) = 2$。
✅ 答案：$2$（收斂）
📝 例題 7.51
∫₀^¹ dx/x（不連續點，發散）
求 $\int_0^1 \frac{dx}{x}$。
解：$\lim_{t\to 0^+} \int_t^1 \frac{dx}{x} = \lim_{t\to 0^+} \Big[\ln x\Big]_t^1 = \lim_{t\to 0^+} (-\ln t) = \infty$。
✅ 答案：發散
📝 例題 7.52
∫₀^³ dx/(x−1)²（內部不連續點）
求 $\int_0^3 \frac{dx}{(x-1)^2}$。
解：$x=1$ 為內部不連續點。拆分為 $\int_0^1 + \int_1^3$。
$\int_0^1 \frac{dx}{(x-1)^2} = \lim_{t\to 1^-} \Big[\frac{-1}{x-1}\Big]_0^t = \lim_{t\to 1^-} \left(\frac{-1}{t-1} - 1\right)$，當 $t\to 1^-$ 時 $t-1\to 0^-$，故 $\frac{-1}{t-1}\to\infty$，此部分發散。整體瑕積分發散。
✅ 答案：發散
📝 例題 7.53
∫₁^∞ dx/(x³+1)（比較定理）
判斷 $\int_1^\infty \frac{dx}{x^3+1}$ 是否收斂。
解：對於 $x \geq 1$，$x^3+1 > x^3$，故 $\frac{1}{x^3+1} < \frac{1}{x^3}$。
$\int_1^\infty \frac{dx}{x^3}$ 為 $p$-積分，$p=3>1$，收斂。
由於 $0 \leq \frac{1}{x^3+1} \leq \frac{1}{x^3}$ 且 $\int_1^\infty \frac{dx}{x^3}$ 收斂，由比較定理，$\int_1^\infty \frac{dx}{x^3+1}$ 收斂。
✅ 答案：收斂（比較定理判定）
📝 例題 7.54
∫₂^∞ dx/(x−1)（p-積分）
判斷 $\int_2^\infty \frac{dx}{x-1}$ 是否收斂。
解：令 $u = x-1$、$du=dx$：$\int_2^\infty \frac{dx}{x-1} = \int_1^\infty \frac{du}{u}$。此為 $p$-積分，$p=1$，發散。
✅ 答案：發散
📝 例題 7.55
∫₁^∞ (1+e⁻ˣ)/x dx（比較定理）
判斷 $\int_1^\infty \frac{1+e^{-x}}{x}\,dx$ 是否收斂。
解：對 $x \geq 1$，$e^{-x} \geq 0$，故 $\frac{1+e^{-x}}{x} \geq \frac{1}{x}$。$\int_1^\infty \frac{dx}{x}$ 發散（$p=1$）。
由比較定理（被積函數大於一個發散的瑕積分），$\int_1^\infty \frac{1+e^{-x}}{x}\,dx$ 發散。
✅ 答案：發散（比較定理判定）
📝 例題 7.56
∫₀^∞ e⁻ˣ sin x dx（瑕積分+循環型）
求 $\int_0^\infty e^{-x}\sin x\,dx$。
解：先求不定積分 $\int e^{-x}\sin x\,dx$。參照例題 7.5 的方法（分部積分兩次後循環）：
$$\int e^{-x}\sin x\,dx = -\frac{e^{-x}}{2}(\sin x + \cos x) + C$$
代入瑕積分：$\int_0^\infty e^{-x}\sin x\,dx = \lim_{t\to\infty} \left[-\frac{e^{-t}}{2}(\sin t + \cos t)\right] - \left[-\frac{1}{2}(0+1)\right]$
$= 0 + \frac{1}{2} = \frac{1}{2}$。（極限為 0 因為 $e^{-t}\to 0$ 且 $\sin t+\cos t$ 有界。）
✅ 答案：$\frac{1}{2}$（收斂）
