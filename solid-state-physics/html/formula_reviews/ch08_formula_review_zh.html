8.1.1–8.1.3 接面電流的邊界條件 (8.1–8.20)

🔍 從準 Fermi 能階到少數載子邊界條件（20 條公式）
1pn 接面的電流成分 (8.1–8.4)
總電流 = 電洞電流 + 電子電流。在順向偏壓下，電洞從 p 側注入 n 側、電子從 n 側注入 p 側。
2⭐ 少數載子邊界條件 (8.16–8.17)
在空乏區邊緣，$pn = n_i^2 e^{eV_a/kT}$（Shockley 邊界條件）：$$\boxed{p_n(x_n) = p_{n0}\,e^{eV_a/kT}} \quad\text{（n 側邊界電洞濃度）} \tag{8.16}$$$$\boxed{n_p(-x_p) = n_{p0}\,e^{eV_a/kT}} \quad\text{（p 側邊界電子濃度）} \tag{8.17}$$$V_a>0$（順偏）→邊界少數載子濃度指數增加。$V_a<0$（逆偏）→邊界濃度趨近於零。
3過量少數載子濃度 (8.18–8.20)
$$\delta p_n(x_n) = p_{n0}(e^{eV_a/kT}-1) \tag{8.18}$$$\delta n_p(-x_p) = n_{p0}(e^{eV_a/kT}-1)$ (8.19)。

8.1.4 理想二極體電流-電壓關係 ⭐ (8.21–8.38)

⭐ 理想二極體方程（Shockley 方程）
$$\boxed{I = I_S\left(e^{eV_a/kT} - 1\right)} \tag{8.27}$$
$$I_S = eA\left(\frac{D_p p_{n0}}{L_p} + \frac{D_n n_{p0}}{L_n}\right) = eA n_i^2\left(\frac{D_p}{L_p N_d} + \frac{D_n}{L_n N_a}\right) \tag{8.28}$$
$I_S$ = 反向飽和電流。$V_a>0$（順偏）→指數增長；$V_a<0$（逆偏）→$I\to -I_S$。
🔍 推導——從擴散方程到 I-V 關係（18 條公式）
1少數載子擴散方程 (8.21–8.24)
n 側中性區中電洞的穩態擴散方程：$d^2\delta p_n/dx^2 = \delta p_n/L_p^2$。
解：$\delta p_n(x) = \delta p_n(x_n)e^{-(x-x_n)/L_p}$ (8.23)。
2擴散電流密度 (8.25–8.26)
電洞擴散電流在 $x=x_n$：$$J_p(x_n) = -eD_p\frac{d\delta p_n}{dx}\bigg|_{x_n} = \frac{eD_p p_{n0}}{L_p}(e^{eV_a/kT}-1) \tag{8.25}$$
3⭐ 總電流 (8.27–8.28)
$J = J_p(x_n) + J_n(-x_p)$ → $I = I_S(e^{eV_a/kT}-1)$。
$I_S \propto n_i^2$ → 寬能隙材料（如 GaAs）的 $I_S$ 遠小於 Si（因 $n_i$ 小）。
📐 例題 8.3 — Si pn 二極體 $I_S$
$N_a=10^{17}$，$N_d=10^{16}$ cm⁻³，$A=10^{-4}$ cm²，$D_p=10$，$D_n=25$ cm²/s，$L_p=L_n=10$ µm。
$p_{n0}=n_i^2/N_d=2.25\times10^4$，$n_{p0}=n_i^2/N_a=2.25\times10^3$ cm⁻³。
$I_S=1.6\times10^{-19}\cdot10^{-4}(10\cdot2.25\times10^4+25\cdot2.25\times10^3)/10^{-3}=4.5\times10^{-14}$ A。

8.1.5 物理效應 (8.39–8.52)

溫度效應
$$I_S \propto T^3 e^{-E_g/kT} \tag{8.42}$$
$I_S$ 隨溫度急劇增加——主要是 $n_i^2\propto e^{-E_g/kT}$ 的貢獻。
串聯電阻效應
$$I = I_S\left[\exp\!\left(\frac{e(V_a - IR_s)}{kT}\right) - 1\right] \tag{8.46}$$
高電流時中性區電阻 $R_s$ 使有效接面電壓小於外加電壓→I-V 曲線偏離理想指數。

8.2 產生-復合電流 (8.53–8.66)

復合電流（順偏）
$$I_{\text{rec}} \propto e^{eV_a/2kT} \tag{8.58}$$
低順向偏壓時，空乏區中的 SRH 復合電流主導→斜率為 $e/2kT$（非理想因子 $n\approx2$）。
產生電流（逆偏）
$$I_{\text{gen}} = \frac{e n_i W}{\tau_0} \tag{8.61}$$
反向偏壓下空乏區中的產生電流 $>I_S$（特別是寬能隙材料如 Si）。$I_{\text{gen}}\propto n_i$，而 $I_S\propto n_i^2$。

8.3 小訊號模型 ⭐ (8.67–8.75)

⭐ 擴散電導（Diffusion Conductance）
$$\boxed{g_d = \frac{dI}{dV} = \frac{e}{kT}(I + I_S) \approx \frac{eI}{kT}} \tag{8.68}$$
⭐ 擴散電容（Diffusion Capacitance）
$$\boxed{C_d = \frac{e}{kT}I\,\tau_{p0}} \quad\text{（n⁺p 接面）} \tag{8.75}$$
$C_d \propto I$——順向電流越大，儲存的過量少數載子越多→等效電容越大。
接面電容 $C_j$（第七章）主導反向偏壓；擴散電容 $C_d$ 主導順向偏壓。

8.4–8.5 電荷儲存與穿隧二極體 (8.76–8.85)

二極體反向恢復
$$t_s \approx \tau_{p0}\,\ln\!\left(1 + \frac{I_F}{I_R}\right) \tag{8.79}$$
$t_s$ = 儲存時間。從順向切換到反向時，儲存電荷需時間消散。
穿隧二極體（Tunnel Diode）
峰值電流密度：$J_p \propto \exp(-\text{const}/\sqrt{N})$ (8.82)
重摻雜（∼10¹⁹ cm⁻³）→空乏區極窄→電子直接穿隧。I-V 特性有負微分電阻區。
