5.1.1 漂移電流密度 (5.1–5.9)

🔍 完整推導——從單一載子受力到總漂移電流（9 條公式）
1一般定義 (5.1a, 5.1b)
$$J_{\text{drf}} = \rho\,v_d \tag{5.1a}$$$\rho$ = 電荷密度 (C/cm³)，$v_d$ = 平均漂移速度 (cm/s)。單位驗證：$J$ = (C/cm³)(cm/s) = A/cm² (5.1b)。
2電洞漂移電流 (5.2–5.5)
電洞電荷密度 $\rho = ep$：$$J_p = (ep)v_{dp} \tag{5.2}$$電洞運動方程 $F = m^*_{cp}a = eE$：$$F = m^*_{cp}\,a = eE \tag{5.3}$$碰撞後平均漂移速度 $\propto E$：$$v_{dp} = \mu_p E \tag{5.4}$$代入得電洞漂移電流：$$J_p = e\mu_p\,p\,E \tag{5.5}$$
3電子漂移電流 (5.6–5.8)
電子電荷密度 $\rho = -en$：$$J_n = (-en)v_{dn} \tag{5.6}$$電子帶負電，漂移方向與電場相反：$$v_{dn} = -\mu_n E \tag{5.7}$$代入得（負負得正，電流與電場同向）：$$J_n = (-en)(-\mu_n E) = e\mu_n\,n\,E \tag{5.8}$$
4⭐ 總漂移電流密度 (5.9)
$$\boxed{J_{\text{drf}} = e(\mu_n n + \mu_p p)E} \tag{5.9}$$
📐 例題 5.1 — GaAs 漂移電流
n 型 GaAs，$N_d=10^{16}$ cm⁻³，$E=10$ V/cm，$\mu_n=8500$ cm²/V·s。
$n_0\approx N_d=10^{16}$ cm⁻³，$p_0=n_i^2/n_0\approx3.2\times10^{-4}$ cm⁻³（可忽略）。
$J_{\text{drf}}\approx e\mu_n N_d E = (1.6\times10^{-19})(8500)(10^{16})(10)=136$ A/cm²。
極小的電場（10 V/cm）就能在半導體中產生可觀的電流密度。

5.1.2 遷移率效應 (5.10–5.16)

🔍 完整推導——散射、平均自由時間、Matthiessen 規則（7 條公式）
1受力與加速 (5.10–5.11)
$$F = m^*_{cp}\frac{dv}{dt} = eE \tag{5.10}$$設 $E$ 和 $m^*_{cp}$ 為常數，積分（初始 $v=0$）：$$v = \frac{eEt}{m^*_{cp}} \tag{5.11}$$
2平均漂移速度 (5.12a, 5.12b)
碰撞之間的平均自由時間 $\tau_{cp}$。峰值速度在 $t=\tau_{cp}$：$$v_{d,\text{peak}} = \frac{eE\tau_{cp}}{m^*_{cp}} \tag{5.12a}$$統計平均（均勻分布在 $0$ 到 $\tau_{cp}$）：$$\langle v_d \rangle = \frac{1}{2}\frac{eE\tau_{cp}}{m^*_{cp}} \tag{5.12b}$$較精確的統計分析給出 $\langle v_d \rangle = eE\tau_{cp}/m^*_{cp}$（無 1/2 因子）。
3遷移率與散射 (5.13–5.15)
由 $\mu = v_d/E$ 和 $v_d = eE\tau/m^*$：$\mu = e\tau/m^*$。
散射率相加（Matthiessen 規則）：$$\frac{1}{\mu} = \frac{1}{\mu_L} + \frac{1}{\mu_I} \tag{5.13}$$晶格散射（聲子）：$$\mu_L \propto T^{-3/2} \tag{5.14}$$雜質散射（游離雜質）：$$\mu_I \propto \frac{T^{3/2}}{N_I} \tag{5.15}$$其中 $N_I = N_d^+ + N_a^-$ 為總游離雜質濃度。
4經驗公式 (5.16)
結合晶格和雜質散射的經驗擬合公式 (5.16) 用於實際計算。

5.1.3 導電率 (5.17–5.26)

⭐ 導電率 (5.17–5.18)
$$\boxed{\sigma = e(\mu_n n + \mu_p p)} \tag{5.17}$$
$$\rho = \frac{1}{\sigma} = \frac{1}{e(\mu_n n + \mu_p p)} \tag{5.18}$$
🔍 推導與延伸 (5.19–5.26)
1n 型/p 型簡化 (5.19–5.20)
n 型 ($n\gg p$)：$\sigma \approx e\mu_n n \approx e\mu_n N_d$ (5.19)
p 型 ($p\gg n$)：$\sigma \approx e\mu_p p \approx e\mu_p N_a$ (5.20)
2遷移率對導電率的影響 (5.21–5.26)
$$\sigma = e\mu_n n_0 + e\mu_p p_0 \tag{5.21a}$$$$\sigma = \frac{1}{\rho} \tag{5.21b}$$電阻 $R$ 與電阻率的關係：$$R = \frac{\rho L}{A} = \frac{L}{\sigma A} \tag{5.22a, 5.22b}$$對矩形棒：$R = L/(e\mu_n n A)$ (5.23)。
本徵導電率 (5.24)：$\sigma_i = e n_i(\mu_n + \mu_p)$。
導電率隨溫度變化 (5.25–5.26)：低溫→雜質散射主導；高溫→$n_i$ 指數增長主導。
📐 例題 5.2–5.4
5.2：Si，$\mu_n=1350$，$N_d=10^{16}$→$\sigma=e\mu_n N_d=2.16$ (Ω·cm)⁻¹。
5.3：p 型 Si 棒，$L=0.1$ cm，$A=10^{-4}$ cm²，$\rho=2$ Ω·cm→$R=\rho L/A=2$ kΩ。

5.1.4 速度飽和 (5.27–5.34)

🔍 推導——從線性到飽和（8 條公式）
1低電場線性區 (5.27a, 5.27b)
$$v_d = \mu E \quad (E < 10^3\ \text{V/cm}) \tag{5.27a, 5.27b}$$
2高電場飽和 (5.28–5.31)
載子從電場獲得的能量速率 = 散射損失能量速率→平衡→飽和速度。$$v_d \to v_{\text{sat}} \quad (E > 10^4\ \text{V/cm}) \tag{5.28}$$Si：$v_{\text{sat}}\approx10^7$ cm/s (5.29)。GaAs：$v_{\text{sat}}\approx10^7$ cm/s (5.30)。
經驗擬合公式 (5.31)：$v_d = \dfrac{\mu E}{[1+(\mu E/v_{\text{sat}})^\beta]^{1/\beta}}$。
3GaAs 的特殊行為 (5.32–5.34)
GaAs 在 $E\approx3\times10^3$ V/cm 處出現負微分遷移率→Gunn 效應 (5.32–5.34)：電子從高遷移率的 $\Gamma$ 谷散射到低遷移率的 $L$ 谷→漂移速度下降。

5.2 載子擴散 (5.35–5.45)

🔍 完整推導——擴散電流 + 總電流（11 條公式）
1擴散電流密度 (5.35–5.38)
電子擴散（從高濃度往低濃度）：$$J_n = e D_n \frac{dn}{dx} \tag{5.35}$$電洞擴散（注意負號！）：$$J_p = -e D_p \frac{dp}{dx} \tag{5.36}$$$D_n, D_p$ = 擴散係數 (cm²/s)。(5.37–5.38) 為一維形式的完整表述。
2總電流密度 (5.39–5.41)
漂移 + 擴散：$$J = e n \mu_n E + e D_n \frac{dn}{dx} + e p \mu_p E - e D_p \frac{dp}{dx} \tag{5.39}$$電子總電流 (5.40)：$J_n = e\mu_n n E + e D_n (dn/dx)$。
電洞總電流 (5.41)：$J_p = e\mu_p p E - e D_p (dp/dx)$。
總電流 (5.42)：$J = J_n + J_p$。
3擴散係數與遷移率 (5.43–5.45)
擴散係數與平均自由路徑 $\ell$ 的關係 (5.43)：$D = \ell^2/(2\tau)$ 或 $D = v_{th}\ell$。
與遷移率的初步連結 (5.44–5.45)：預示 Einstein 關係。

5.3 Einstein 關係 (5.46–5.51)

⭐ Einstein 關係 (5.46a, 5.46b)
$$\boxed{\frac{D_n}{\mu_n} = \frac{kT}{e}} \tag{5.46a}$$
$$\boxed{\frac{D_p}{\mu_p} = \frac{kT}{e}} \tag{5.46b}$$
室溫 $kT/e = 0.0259$ V。漂移和擴散由同一熱運動驅動，非獨立參數。
🔍 推導——從熱平衡條件出發 (5.47–5.51)
1梯度摻雜產生內建電場 (5.47–5.48)
不均勻摻雜→載子擴散→電荷分離→內建電場 (5.47)。
熱平衡下總電流為零 (5.48)：$J_p = e\mu_p p E - e D_p (dp/dx) = 0$。
2推導 Einstein 關係 (5.49a–5.49b)
由 $J_p=0$：$\mu_p p E = D_p (dp/dx)$。$p = n_i e^{-(E_F-E_{Fi})/kT}$，而 $E = (1/e)dE_{Fi}/dx$。
代入整理：$$\frac{D_p}{\mu_p} = \frac{kT}{e} \tag{5.49a}$$同理對電子 (5.49b)。
3內建電場表達式 (5.50–5.51)
由不均勻摻雜產生的內建電場 (5.50)：$E = \dfrac{kT}{e}\dfrac{1}{p}\dfrac{dp}{dx}$。
電位分布 (5.51)：$\phi(x) = -\dfrac{kT}{e}\ln\dfrac{p(x)}{p_0}$。

5.4 霍爾效應 (5.52–5.60)

🔍 完整推導——Lorentz 力→霍爾電壓→材料參數（9 條公式）
1Lorentz 力與平衡 (5.52–5.53)
載子受 Lorentz 力：$\mathbf{F} = q(\mathbf{v} \times \mathbf{B})$。偏轉→累積電荷→霍爾電場。
平衡時 $F_E = F_B$：$eE_H = ev_d B$ (5.52)。
2霍爾電壓 (5.54–5.55)
$$V_H = E_H \cdot W = v_d B W \tag{5.54}$$$v_d = I/(epWd)$（由 $J = I/(Wd) = epv_d$）。代入：$$\boxed{V_H = \frac{IB}{epd}} \quad\text{（p 型）} \tag{5.55}$$
3霍爾係數 (5.56–5.58)
$$R_H = \frac{V_H d}{IB} = \frac{1}{ep} \quad\text{（p 型）} \tag{5.56}$$n 型：$R_H = -1/(en)$ (5.57)。$R_H$ 的正負號直接標示載子類型。
載子濃度由 $R_H$ 得出 (5.58)：$p = 1/(eR_H)$ 或 $n = -1/(eR_H)$。
4霍爾遷移率 (5.59–5.60)
$$\mu_H = |R_H|\,\sigma \tag{5.59}$$霍爾遷移率通常略大於漂移遷移率（因散射機制對霍爾效應的影響不同）(5.60)。
