先看總骨架

如果你把這本書當成積木堆疊，真正支撐所有裝置章節的主幹其實很少，主要只有五條。

  1. 能帶統計 → 本徵濃度 ni → mass action law → 平衡載子濃度。
  2. 平衡載子濃度 → 接面擴散與空乏區 → 內建電位 Vbi。
  3. 少數載子注入 + 連續方程 + 邊界條件 → 理想二極體方程式。
  4. MOS capacitor 電壓分配 + 強反轉條件 + 通道電荷積分 → MOSFET I-V。
  5. 雙接面注入/收集 + 薄基區傳輸 → BJT 放大；bandgap + 吸收/復合 → 光電元件。

推導 A：內建電位 Vbi

問題不是「公式長怎樣」，而是「為什麼接面一形成就必須長出這個勢壘」。

Step 1 ｜ 從平衡統計寫出 n 與 p
n = Nc exp[-(Ec - EF) / kT]
p = Nv exp[-(EF - Ev) / kT]
np = NcNv exp[-(Ec - Ev) / kT] = NcNv exp(-Eg / kT)
ni2 = NcNv exp(-Eg / kT)

所以平衡條件下自然得到 np = ni2。這就是 mass action law 的來源。

Step 2 ｜ 在非簡併、完全游離摻雜下寫出兩側濃度
n 區：nn0 ≈ Nd, pn0 = ni2 / Nd
p 區：pp0 ≈ Na, np0 = ni2 / Na

這一步把摻雜濃度和真正的載子濃度連上。後面所有接面推導都要靠它。

Step 3 ｜ 接觸後先擴散，再形成電場

電子從 n 側往 p 側擴散，電洞從 p 側往 n 側擴散。擴散留下不可移動的離子化施主與受主，因此空乏區出現。空乏區內的固定電荷產生電場，這個電場會阻止進一步擴散。

Step 4 ｜ 平衡要求整個結構只有一條 Fermi level
qVbi = Ei,p - Ei,n
n = ni exp[(EF - Ei) / kT], p = ni exp[(Ei - EF) / kT]

只要 EF 還不平，結構內就還有淨驅動力，所以還不是平衡。能帶彎曲存在的目的，就是把 EF 拉平。

Step 5 ｜ 代回去，Vbi 自然出現
EF - Ei,n = kT ln(Nd / ni)
Ei,p - EF = kT ln(Na / ni)
Ei,p - Ei,n = kT ln[(NaNd) / ni2]
Vbi = VT ln[(NaNd) / ni2], VT = kT/q
橋接：第 3 章給你統計基礎，第 4 章把它變成接面平衡。

推導 B：理想二極體方程式

核心不是硬背 I = IS(e^{V/VT} - 1)，而是看懂它其實只是少數載子注入與擴散的結果。

Step 1 ｜ 起點是漂移-擴散與連續方程
Jn = qnμnE + qDn(dn/dx)
Jp = qpμpE - qDp(dp/dx)
Dp d2Δpn/dx2 - Δpn/τp = 0
Dn d2Δnp/dx2 - Δnp/τn = 0

前提：低注入、穩態、一維、空乏區復合忽略、準中性區內電場很小。

Step 2 ｜ 正向偏壓改變的是邊界條件
pn(0) = pn0 exp(V / VT)
np(0) = np0 exp(V / VT)
Δpn(0) = pn0[exp(V / VT) - 1]
Δnp(0) = np0[exp(V / VT) - 1]

真正被改變的是空乏區邊緣的少數載子濃度。這就是為什麼二極體呈指數，而不是線性。

Step 3 ｜ 解擴散方程得到空間衰減
Lp = √(Dpτp), Ln = √(Dnτn)
Δpn(x) = Δpn(0) exp(-x / Lp)
Δnp(x) = Δnp(0) exp(-x / Ln)
Step 4 ｜ 電流由邊界梯度決定
Jp(0) = qDpΔpn(0) / Lp
Jn(0) = qDnΔnp(0) / Ln
J = q[(Dp/Lp)pn0 + (Dn/Ln)np0][exp(V / VT) - 1]
Step 5 ｜ 乘上面積得到 Shockley 方程
I = IS[exp(V / VT) - 1]
IS = qA[(Dp/Lp)pn0 + (Dn/Ln)np0]
pn0 = ni2/Nd, np0 = ni2/Na
橋接：第 5 章給 drift-diffusion，第 6 章給連續方程，第 7-8 章把它關成一顆真正可量測的 diode。

推導 C：MOS capacitor → MOSFET

MOSFET 不是突然冒出來的元件，它是 MOS capacitor 再加 source/drain 之後的自然延伸。

Step 1 ｜ 先寫 MOS capacitor 的電壓分配
VG = VFB + ψs + Qs/Cox
Cox = εox/tox
VFB = φms - Qox/Cox
Step 2 ｜ 門檻電壓來自強反轉條件
φF = VT ln(Na / ni)
ψs,th = 2φF
|QB| = √(2εsqNa · 2φF)
VT0 = VFB + 2φF + |QB| / Cox
Step 3 ｜ 超過 threshold 後，局部反轉電荷由 gate overdrive 決定
Qinv(x) = -Cox[VGS - VT - V(x)]

通道靠近 drain 的位置，V(x) 較大，所以反轉電荷較少。pinch-off 就是這樣長出來的。

Step 4 ｜ 沿通道積分得到線性區公式
ID = -W μnQinv(x) dV/dx
IDL = WμnCox ∫0VDS [(VGS-VT) - V] dV
ID = μnCox(W/L)[(VGS-VT)VDS - VDS2/2]
Step 5 ｜ 飽和區是通道末端先消失
VDS,sat = VGS - VT
ID,sat = (1/2)μnCox(W/L)(VGS - VT)2
橋接：第 4 章的表面位能與第 5 章的漂移傳輸，在第 10-11 章合成 MOSFET 電流模型。

推導 D：BJT 的放大機制

BJT 的核心不是「三層摻雜」這句廢話，而是：怎樣讓大部分注入載子穿過基區，只有很少一部分在 base 內復合。

Step 1 ｜ 定義 forward-active 模式
VBE > 0, VBC < 0

BE 正偏負責注入，BC 反偏負責收集。

Step 2 ｜ 基區邊界的少數電子濃度由 VBE 抬高
np(0) = np0 exp(VBE / VT)
np0 = ni2 / Na,B
Step 3 ｜ 薄基區近似把 collector 電流寫出來
IC = qADn(dn/dx)
若 WB ≪ Ln，則 (dn/dx) ≈ np(0)/WB
IC ≈ qADnnp0 exp(VBE/VT) / WB = IS exp(VBE/VT)
Step 4 ｜ 大 β 來自 αF 非常接近 1
αF = γ · αT
γ = InE / (InE + IpE)
αT = IC / InE
β = αF / (1 - αF)
Step 5 ｜ 把設計語言翻回物理
重摻雜 emitter ⇒ γ ↑
薄 base、低復合 ⇒ αT ↑
所以 αF → 1 ⇒ β ≫ 1
橋接：第 6 章的少數載子與復合概念，直接長成第 12-13 章的 BJT 放大機制。

推導 E：bandgap → 吸收 → 光電流 → 發光 → 雷射

這一段最容易被學生切碎背誦，但其實它根本是一條連續物理鏈。

Step 1 ｜ 光子是否能被吸收，先看能量
Ephoton = hν = hc / λ
吸收條件：hν ≥ Eg
Step 2 ｜ 吸收後，光子通量與產生率 obey 指數律
Φ(x) = Φ0 exp(-αx)
G(x) = -dΦ/dx = αΦ0 exp(-αx)
Step 3 ｜ 光生載子若被接面收集，就形成光電流
I = IL - IS[exp(V / VT) - 1]
VOC = VT ln(IL / IS + 1)

solar cell 本質上就是被光生電流推到第四象限的 pn 接面。

Step 4 ｜ 主動注入並以輻射復合為主，就是 LED

關鍵不只是有 pn 接面，而是材料必須讓 radiative recombination 夠有效。直接能隙材料因此特別重要。

Step 5 ｜ 再加上人口反轉與回授，LED 才會變成 laser
條件語言：optical gain > internal loss + mirror loss
橋接：第 6 章的 generation-recombination，第 7-8 章的接面場，第 14 章的光電元件，其實都在同一條鏈上。
