2.1.1 能量量子化與光電效應

Planck 量子假說 (1900)
$$E = h\nu = \frac{hc}{\lambda}$$
$E(\text{eV}) = 12400 / \lambda(\text{Å})$
🔍 光電效應推導 (2.1)
1能量守恆
光子能量 = 功函數 + 光電子動能$$h\nu = \phi + T_{\text{max}}$$
2功函數 $\phi = h\nu_0$
$\nu_0$ = 截止頻率。$\nu < \nu_0$ → 無光電子。
3光電子最大動能 (2.1)
$$\boxed{T_{\text{max}} = \frac{1}{2}mv^2 = h\nu - \phi = h(\nu - \nu_0)} \tag{2.1}$$
📐 例題 2.1 — X 射線光子能量
$\lambda = 0.708\ \text{Å}$
$E = hc/\lambda = 2.81 \times 10^{-15}\ \text{J} = 1.75 \times 10^4\ \text{eV} = 17.5\ \text{keV}$
（速算：$E = 12400/0.708 = 17500\ \text{eV}$）

2.1.2 波粒二象性

光子動量 (2.2)
$$p = \frac{h}{\lambda} \tag{2.2}$$
De Broglie 波長 (1924) (2.3)
$$\boxed{\lambda = \frac{h}{p} = \frac{h}{mv}} \tag{2.3}$$
動能形式：$p = \sqrt{2mE_k}$ → $\lambda = h/\sqrt{2mE_k}$
📐 例題 2.2 — 電子 de Broglie 波長
$v = 10^5\ \text{m/s}$，$p = mv = 9.11 \times 10^{-26}\ \text{kg·m/s}$
$\lambda = h/p = 7.27 \times 10^{-9}\ \text{m} = 72.7\ \text{Å}$（紫外線範圍）
⚡ Davisson–Germer 實驗 (1927)

2.1.3 海森堡不確定性原理

位置–動量 (2.4)
$$\Delta p \cdot \Delta x \geq \hbar \tag{2.4}$$
能量–時間 (2.5)
$$\Delta E \cdot \Delta t \geq \hbar \tag{2.5}$$
$\hbar = h/2\pi = 1.054 \times 10^{-34}\ \text{J·s}$
🔍 TYU 2.1 — 不確定性計算
$\Delta x = 8\ \text{Å}$，$\Delta p_{\text{min}} = \hbar/\Delta x = 1.32 \times 10^{-25}\ \text{kg·m/s}$
若 $p = 1.2 \times 10^{-23}$，$\Delta E_k \approx (p/m)\Delta p = 8.5\ \text{eV}$
🔍 TYU 2.2 — 能量–時間不確定性
$\Delta E = 0.8\ \text{eV} = 1.28 \times 10^{-19}\ \text{J}$
$\Delta t_{\text{min}} = \hbar/\Delta E = 8.23 \times 10^{-16}\ \text{s}$（質子與電子相同）
⚡ 核心推論
無法同時精確測定 $x$ 和 $p$ → 只能以機率描述 → Born：$|\psi|^2$ = 機率密度。

2.2 薛丁格波動方程式 (2.6–2.18)

時間相關薛丁格方程 (2.6)
$$-\frac{\hbar^2}{2m}\frac{\partial^2\Psi(x,t)}{\partial x^2} + V(x)\Psi(x,t) = j\hbar\frac{\partial\Psi(x,t)}{\partial t} \tag{2.6}$$
🔍 分離變數法 — (2.6) → (2.13) ⭐ 必考
1假設分離 (2.7)
$$\Psi(x,t) = \psi(x)\,\phi(t) \tag{2.7}$$
2代入 (2.6) 得 (2.8)
$$-\frac{\hbar^2}{2m}\phi\frac{d^2\psi}{dx^2} + V\psi\phi = j\hbar\psi\frac{d\phi}{dt} \tag{2.8}$$
3除以 $\psi\phi$，分離變數 (2.9)
$$-\frac{\hbar^2}{2m}\frac{1}{\psi}\frac{d^2\psi}{dx^2} + V = j\hbar\frac{1}{\phi}\frac{d\phi}{dt} \tag{2.9}$$左僅與 $x$ 有關，右僅與 $t$ 有關 → 等於常數 $E$。
4時間部分 (2.10) → (2.11)
$$j\hbar\frac{1}{\phi}\frac{d\phi}{dt} = E \tag{2.10}$$$$\phi(t) = e^{-j(E/\hbar)t} = e^{-j\omega t} \tag{2.11}$$$\omega = E/\hbar$，$E$ = 粒子總能量。
5時間無關薛丁格方程 (2.12) → (2.13)
$$-\frac{\hbar^2}{2m}\frac{1}{\psi}\frac{d^2\psi}{dx^2} + V = E \tag{2.12}$$$$\boxed{\frac{d^2\psi}{dx^2} + \frac{2m}{\hbar^2}[E - V(x)]\psi = 0} \tag{2.13}$$
Born 機率詮釋 (2.14–2.17)
$$\Psi(x,t) = \psi(x)e^{-j\omega t} \tag{2.14}$$
(2.15) $|\Psi|^2 = \Psi\Psi^*$
(2.16) $\Psi\Psi^* = \psi e^{-j\omega t} \cdot \psi^* e^{+j\omega t} = \psi\psi^*$
(2.17) $\boxed{|\Psi(x,t)|^2 = |\psi(x)|^2}$（與時間無關！）
歸一化條件 (2.18)
$$\int_{-\infty}^{\infty} |\psi(x)|^2\,dx = 1 \tag{2.18}$$
邊界條件
① $\psi$ 有限、單值、連續 ② $\psi'$ 有限、單值、連續（$V$ 有限時）

4.1.1–4.1.2 熱平衡載子濃度

⭐ 電子與電洞濃度（Boltzmann 近似）
$$\boxed{n_0 = N_c\,\exp\!\left(-\frac{E_c - E_F}{kT}\right)} \tag{4.11}$$
$$\boxed{p_0 = N_v\,\exp\!\left(-\frac{E_F - E_v}{kT}\right)} \tag{4.19}$$
$n_0$ = 熱平衡電子濃度, $p_0$ = 熱平衡電洞濃度
$N_c$, $N_v$ = 導帶/價帶有效態密度（見 (4.13), (4.21)）
🔍 推導 — $n_0$ 從積分得出
1電子濃度積分
$$n_0 = \int_{E_c}^{\infty} g_c(E)\,f_F(E)\,dE \tag{4.1}$$$g_c(E)=\frac{4\pi(2m_n^*)^{3/2}}{h^3}\sqrt{E-E_c}$（態密度）
$f_F(E)=\frac{1}{1+e^{(E-E_F)/kT}}$（Fermi-Dirac）
2Boltzmann 近似（$E_c-E_F \gg kT$）
$$f_F(E) \approx \exp\!\left(-\frac{E-E_F}{kT}\right) \tag{4.3}$$
3積分結果 (4.11)
令 $\eta = (E-E_c)/kT$：$$n_0 = N_c\,e^{-(E_c-E_F)/kT},\quad N_c = 2\left(\frac{2\pi m_n^* kT}{h^2}\right)^{3/2} \tag{4.11,4.13}$$
有效態密度 (4.13, 4.21)
$$N_c = 2\left(\frac{2\pi m_n^* kT}{h^2}\right)^{3/2} \tag{4.13}$$
$$N_v = 2\left(\frac{2\pi m_p^* kT}{h^2}\right)^{3/2} \tag{4.21}$$
Si (300K): $N_c=2.8\times10^{19}$ cm⁻³, $N_v=1.04\times10^{19}$ cm⁻³

4.1.3 本徵載子濃度 $n_i$

⭐ 質量作用定律 (4.23)
$$\boxed{n_0 p_0 = n_i^2} \tag{4.23}$$
熱平衡下，$n_0p_0$ 為常數，僅取決於溫度和材料（與摻雜無關）。
⭐ $n_i$ 公式 (4.20, 4.24)
$$n_i = \sqrt{N_c N_v}\,\exp\!\left(-\frac{E_g}{2kT}\right) \tag{4.20}$$
$$n_i = 2\left(\frac{2\pi kT}{h^2}\right)^{3/2}(m_n^* m_p^*)^{3/4}\exp\!\left(-\frac{E_g}{2kT}\right) \tag{4.24}$$
$n_i$ 隨溫度指數增加。Si 每升 11°C，$n_i$ 約翻倍。
📐 例題 4.1–4.2 — $n_i$ 計算
Si (300K): $n_i = \sqrt{(2.8\times10^{19})(1.04\times10^{19})}\,e^{-1.12/(2\cdot0.0259)}=1.5\times10^{10}$ cm⁻³

4.1.4 本徵 Fermi 能階

⭐ 本徵 Fermi 能階 $E_{Fi}$ (4.25)
$$\boxed{E_{Fi} = \frac{E_c + E_v}{2} + \frac{3}{4}kT\ln\!\left(\frac{m_p^*}{m_n^*}\right)} \tag{4.25}$$
$E_{Fi}$ 非常接近能隙中央。Si: $E_{Fi} \approx E_{\text{midgap}} - 0.013$ eV。
🔍 推導 — $n_0=p_0=n_i$ 代入求解
由 $n_0=N_c e^{-(E_c-E_{Fi})/kT}=p_0=N_v e^{-(E_{Fi}-E_v)/kT}$，取對數相減：$$E_{Fi} = \frac{E_c+E_v}{2} + \frac{kT}{2}\ln\frac{N_v}{N_c} = \frac{E_c+E_v}{2} + \frac{3}{4}kT\ln\frac{m_p^*}{m_n^*}$$
⭐ 用 $n_i$ 和 $E_{Fi}$ 重寫 $n_0$, $p_0$ (4.26–4.27)
$$\boxed{n_0 = n_i\,\exp\!\left(\frac{E_F - E_{Fi}}{kT}\right)} \tag{4.26}$$
$$\boxed{p_0 = n_i\,\exp\!\left(-\frac{E_F - E_{Fi}}{kT}\right)} \tag{4.27}$$
$E_F > E_{Fi}$ → n 型 ($n_0 > n_i > p_0$)；$E_F < E_{Fi}$ → p 型。

4.3 外質半導體

$n_0p_0$ 積在外質半導體中仍成立
$$n_0 p_0 = n_i^2 \quad\text{（摻雜不改變此關係）} \tag{4.40}$$
⚡ 簡併與非簡併
非簡併：$E_F$ 在能隙中，距能帶邊緣 $>3kT$ → Boltzmann 近似有效。
簡併：$E_F$ 進入導帶或價帶（$E_F > E_c$ 或 $E_F < E_v$）→ 必須用 Fermi-Dirac 積分。

4.4 施體與受體的統計

游離施體濃度 (4.53–4.55)
$$N_d^+ = \frac{N_d}{1 + g_d\,\exp\!\left(\dfrac{E_F - E_d}{kT}\right)} \tag{4.53}$$
$g_d=2$（施體基態簡併度）。$E_F \ll E_d$ → $N_d^+ \approx N_d$（完全游離）。
游離受體濃度 (4.56)
$$N_a^- = \frac{N_a}{1 + g_a\,\exp\!\left(\dfrac{E_a - E_F}{kT}\right)} \tag{4.56}$$
$g_a=4$（受體基態簡併度，價帶頂部雙重簡併）。

4.5 電荷中性 ⭐

⭐ 電荷中性條件 (4.60)
$$\boxed{n_0 + N_a^- = p_0 + N_d^+} \tag{4.60}$$
總負電荷 = 總正電荷。$N_a^-$ = 游離受體濃度, $N_d^+$ = 游離施體濃度。
🔍 推導 — 完全游離下的 $n_0$, $p_0$
1n 型半導體（$N_d \gg N_a \approx 0$）
$N_d^+ \approx N_d$, $N_a^- \approx 0$。由 (4.60): $n_0 = p_0 + N_d$
代入 $p_0 = n_i^2/n_0$：$n_0 = n_i^2/n_0 + N_d$
$$n_0 = \frac{N_d}{2} + \sqrt{\left(\frac{N_d}{2}\right)^2 + n_i^2} \approx N_d \quad (N_d \gg n_i) \tag{4.62}$$$$p_0 = \frac{n_i^2}{n_0} \approx \frac{n_i^2}{N_d} \tag{4.64}$$
2p 型半導體（$N_a \gg N_d \approx 0$）
$$p_0 = \frac{N_a}{2} + \sqrt{\left(\frac{N_a}{2}\right)^2 + n_i^2} \approx N_a \quad (N_a \gg n_i) \tag{4.66}$$$$n_0 = \frac{n_i^2}{p_0} \approx \frac{n_i^2}{N_a} \tag{4.68}$$
3補償半導體（$N_d, N_a$ 均存在）
$$n_0 = \frac{N_d - N_a}{2} + \sqrt{\left(\frac{N_d-N_a}{2}\right)^2 + n_i^2} \quad (N_d > N_a) \tag{4.69}$$

4.6 Fermi 能階位置 ⭐

⭐ Fermi 能階與摻雜的關係 (4.73–4.75)
$$\boxed{E_F - E_{Fi} = kT\ln\!\left(\frac{n_0}{n_i}\right)} \tag{4.73}$$
n 型 ($n_0 \approx N_d$)：$E_F - E_{Fi} = kT\ln(N_d/n_i)$ → $E_F$ 向 $E_c$ 靠近
p 型 ($p_0 \approx N_a$)：$E_{Fi} - E_F = kT\ln(N_a/n_i)$ → $E_F$ 向 $E_v$ 靠近
Fermi 能階相對於能帶邊緣
$$E_c - E_F = kT\ln\!\left(\frac{N_c}{N_d}\right) \quad\text{（n 型，$N_d \gg n_i$）} \tag{4.79}$$
$N_d$ 越大 → $E_F$ 越靠近 $E_c$。
Si: $N_d=10^{16}$ cm⁻³ → $E_c-E_F\approx0.21$ eV。
📐 例題 4.12 — Fermi 能階位置
Si, $N_d=10^{16}$ cm⁻³, $T=300$ K：
$n_0\approx N_d=10^{16}$ cm⁻³
$E_F-E_{Fi}=kT\ln(10^{16}/1.5\times10^{10})=0.347$ eV
（$E_{Fi}$ 在能隙中央下方 0.013 eV → $E_c-E_F\approx0.21$ eV）

5.1.1 漂移電流密度 (5.1–5.9)

🔍 完整推導——從單一載子受力到總漂移電流（9 條公式）
1一般定義 (5.1a, 5.1b)
$$J_{\text{drf}} = \rho\,v_d \tag{5.1a}$$$\rho$ = 電荷密度 (C/cm³)，$v_d$ = 平均漂移速度 (cm/s)。單位驗證：$J$ = (C/cm³)(cm/s) = A/cm² (5.1b)。
2電洞漂移電流 (5.2–5.5)
電洞電荷密度 $\rho = ep$：$$J_p = (ep)v_{dp} \tag{5.2}$$電洞運動方程 $F = m^*_{cp}a = eE$：$$F = m^*_{cp}\,a = eE \tag{5.3}$$碰撞後平均漂移速度 $\propto E$：$$v_{dp} = \mu_p E \tag{5.4}$$代入得電洞漂移電流：$$J_p = e\mu_p\,p\,E \tag{5.5}$$
3電子漂移電流 (5.6–5.8)
電子電荷密度 $\rho = -en$：$$J_n = (-en)v_{dn} \tag{5.6}$$電子帶負電，漂移方向與電場相反：$$v_{dn} = -\mu_n E \tag{5.7}$$代入得（負負得正，電流與電場同向）：$$J_n = (-en)(-\mu_n E) = e\mu_n\,n\,E \tag{5.8}$$
4⭐ 總漂移電流密度 (5.9)
$$\boxed{J_{\text{drf}} = e(\mu_n n + \mu_p p)E} \tag{5.9}$$
📐 例題 5.1 — GaAs 漂移電流
n 型 GaAs，$N_d=10^{16}$ cm⁻³，$E=10$ V/cm，$\mu_n=8500$ cm²/V·s。
$n_0\approx N_d=10^{16}$ cm⁻³，$p_0=n_i^2/n_0\approx3.2\times10^{-4}$ cm⁻³（可忽略）。
$J_{\text{drf}}\approx e\mu_n N_d E = (1.6\times10^{-19})(8500)(10^{16})(10)=136$ A/cm²。
極小的電場（10 V/cm）就能在半導體中產生可觀的電流密度。

5.1.2 遷移率效應 (5.10–5.16)

🔍 完整推導——散射、平均自由時間、Matthiessen 規則（7 條公式）
1受力與加速 (5.10–5.11)
$$F = m^*_{cp}\frac{dv}{dt} = eE \tag{5.10}$$設 $E$ 和 $m^*_{cp}$ 為常數，積分（初始 $v=0$）：$$v = \frac{eEt}{m^*_{cp}} \tag{5.11}$$
2平均漂移速度 (5.12a, 5.12b)
碰撞之間的平均自由時間 $\tau_{cp}$。峰值速度在 $t=\tau_{cp}$：$$v_{d,\text{peak}} = \frac{eE\tau_{cp}}{m^*_{cp}} \tag{5.12a}$$統計平均（均勻分布在 $0$ 到 $\tau_{cp}$）：$$\langle v_d \rangle = \frac{1}{2}\frac{eE\tau_{cp}}{m^*_{cp}} \tag{5.12b}$$較精確的統計分析給出 $\langle v_d \rangle = eE\tau_{cp}/m^*_{cp}$（無 1/2 因子）。
3遷移率與散射 (5.13–5.15)
由 $\mu = v_d/E$ 和 $v_d = eE\tau/m^*$：$\mu = e\tau/m^*$。
散射率相加（Matthiessen 規則）：$$\frac{1}{\mu} = \frac{1}{\mu_L} + \frac{1}{\mu_I} \tag{5.13}$$晶格散射（聲子）：$$\mu_L \propto T^{-3/2} \tag{5.14}$$雜質散射（游離雜質）：$$\mu_I \propto \frac{T^{3/2}}{N_I} \tag{5.15}$$其中 $N_I = N_d^+ + N_a^-$ 為總游離雜質濃度。

2高電場飽和 (5.28–5.31)

5.2 載子擴散 (5.35–5.45)

🔍 完整推導——擴散電流 + 總電流（11 條公式）
1擴散電流密度 (5.35–5.38)
電子擴散（從高濃度往低濃度）：$$J_n = e D_n \frac{dn}{dx} \tag{5.35}$$電洞擴散（注意負號！）：$$J_p = -e D_p \frac{dp}{dx} \tag{5.36}$$$D_n, D_p$ = 擴散係數 (cm²/s)。(5.37–5.38) 為一維形式的完整表述。
2總電流密度 (5.39–5.41)
漂移 + 擴散：$$J = e n \mu_n E + e D_n \frac{dn}{dx} + e p \mu_p E - e D_p \frac{dp}{dx} \tag{5.39}$$電子總電流 (5.40)：$J_n = e\mu_n n E + e D_n (dn/dx)$。
電洞總電流 (5.41)：$J_p = e\mu_p p E - e D_p (dp/dx)$。
總電流 (5.42)：$J = J_n + J_p$。
3擴散係數與遷移率 (5.43–5.45)
擴散係數與平均自由路徑 $\ell$ 的關係 (5.43)：$D = \ell^2/(2\tau)$ 或 $D = v_{th}\ell$。
與遷移率的初步連結 (5.44–5.45)：預示 Einstein 關係。

5.3 Einstein 關係 (5.46–5.51)

⭐ Einstein 關係 (5.46a, 5.46b)
$$\boxed{\frac{D_n}{\mu_n} = \frac{kT}{e}} \tag{5.46a}$$
$$\boxed{\frac{D_p}{\mu_p} = \frac{kT}{e}} \tag{5.46b}$$
室溫 $kT/e = 0.0259$ V。漂移和擴散由同一熱運動驅動，非獨立參數。
🔍 推導——從熱平衡條件出發 (5.47–5.51)
1梯度摻雜產生內建電場 (5.47–5.48)
不均勻摻雜→載子擴散→電荷分離→內建電場 (5.47)。
熱平衡下總電流為零 (5.48)：$J_p = e\mu_p p E - e D_p (dp/dx) = 0$。
2推導 Einstein 關係 (5.49a–5.49b)
由 $J_p=0$：$\mu_p p E = D_p (dp/dx)$。$p = n_i e^{-(E_F-E_{Fi})/kT}$，而 $E = (1/e)dE_{Fi}/dx$。

6.1 載子產生與復合 (6.1–6.23)

🔍 完整推導——過量載子、產生率、復合率（23 條公式）
1過量載子定義 (6.1–6.6)
$$\delta n = n - n_0,\quad \delta p = p - p_0 \tag{6.1}$$低階注入條件：$\delta n, \delta p \ll n_0$ (n 型) 或 $\delta n, \delta p \ll p_0$ (p 型) (6.2–6.4)。
電中性：過量電子和電洞成對產生→$\delta n = \delta p$ (6.5–6.6)。
2產生率與復合率 (6.7–6.14)
熱產生率 $G_{n0}=G_{p0}=G_0$ (6.7)。熱平衡復合率 $R_{n0}=R_{p0}=G_0$ (6.8)。
過量載子復合率 (6.9–6.12)：$R_n' = R_n - R_{n0}$，$R_p' = R_p - R_{p0}$。
直接能帶間復合 (6.13)：$R = \beta n p$。熱平衡：$R_0 = \beta n_0 p_0 = \beta n_i^2$ (6.14)。
3⭐ 過量載子復合率（低階注入）(6.15–6.19)
淨復合率：$R_n' = R - G = \beta(np - n_i^2)$ (6.15)。
低階注入 n 型 ($n\approx n_0$)：$$R_n' = \beta n_0\,\delta p \quad\Rightarrow\quad \tau_{p0} = \frac{1}{\beta n_0} \tag{6.17}$$p 型同理 (6.18)：$\tau_{n0} = 1/(\beta p_0)$。
$$R_n' = \frac{\delta p}{\tau_{p0}} \quad\text{（n 型）}\qquad R_p' = \frac{\delta n}{\tau_{n0}} \quad\text{（p 型）} \tag{6.19}$$
4SRH 復合理論 (6.20–6.23)
Shockley-Read-Hall 透過陷阱輔助復合：$$R = \frac{C_n C_p N_t(np - n_i^2)}{C_n(n+n_1) + C_p(p+p_1)} \tag{6.23}$$$n_1 = n_i e^{(E_t-E_{Fi})/kT}$，$p_1 = n_i e^{-(E_t-E_{Fi})/kT}$。
陷阱靠近能隙中央時復合最有效→$E_t\approx E_{Fi}$ 時 $R$ 最大。

6.2 過量載子的特性 (6.24–6.39)

🔍 連續性方程與擴散長度（16 條公式）
1連續性方程 (6.24–6.28)
電洞連續性方程（一維）：$$\frac{\partial p}{\partial t} = -\frac{1}{e}\frac{\partial J_p}{\partial x} + G_p - R_p \tag{6.24}$$電子同理 (6.25)。物理意義：濃度變化率 = 流入淨值 + 產生 − 消滅。
2擴散方程（無電場、均勻摻雜）(6.29–6.33)
代換 $J_p = -eD_p(\partial p/\partial x)$ 和 $R_p' = \delta p/\tau_{p0}$：$$D_p\frac{\partial^2\delta p}{\partial x^2} + G' - \frac{\delta p}{\tau_{p0}} = \frac{\partial\delta p}{\partial t} \tag{6.29}$$穩態且 $G'=0$ (6.30)：$d^2\delta p/dx^2 = \delta p/(D_p\tau_{p0})$。
3⭐ 擴散長度 (6.34–6.37)
定義：$$\boxed{L_p = \sqrt{D_p\tau_{p0}}} \quad\text{（電洞擴散長度）} \tag{6.34}$$$$L_n = \sqrt{D_n\tau_{n0}} \quad\text{（電子擴散長度）} \tag{6.35}$$穩態擴散解 (6.36)：$\delta p(x) = Ae^{-x/L_p} + Be^{+x/L_p}$。
物理意義：過量載子濃度每經過一個 $L_p$ 衰減 $e$ 倍。
4Haynes-Shockley 實驗 (6.38–6.39)
脈衝產生→量測到達時間→得漂移遷移率；量測脈衝展寬→得擴散係數 (6.38–6.39)。

6.3 雙極傳輸 ⭐ (6.40–6.82)

⭐ 雙極傳輸方程（Ambipolar Transport Equation）
$$D'\frac{\partial^2\delta p}{\partial x^2} - \mu'\mathcal{E}\frac{\partial\delta p}{\partial x} + G' - \frac{\delta p}{\tau_{p0}} = \frac{\partial\delta p}{\partial t} \tag{6.67}$$
過量電子和電洞一起移動，不是獨立地！雙極擴散係數 $D'$ 和雙極遷移率 $\mu'$ 取決於兩種類型載子的濃度。
🔍 完整推導——從 Poisson 方程到雙極傳輸（43 條公式）
1雙極擴散係數 (6.56)
$$D' = \frac{\mu_n n D_p + \mu_p p D_n}{\mu_n n + \mu_p p} \tag{6.56}$$n 型 ($n\gg p$)：$D' \approx D_p$（電洞擴散主導少數載子傳輸）。

2雙極遷移率 (6.61)
$$\mu' = \frac{\mu_n\mu_p(p-n)}{\mu_n n + \mu_p p} \tag{6.61}$$n 型 ($n\gg p$)：$\mu' \approx \mu_p$（少數載子電洞的遷移率）。p 型：$\mu' \approx -\mu_n$。
3介電弛豫時間 (6.74–6.76)

4Haynes-Shockley 實驗分析 (6.77–6.82)

6.4 準 Fermi 能階 (6.83–6.95)

$$x_n = \sqrt{\frac{2\epsilon_s V_{bi}}{e}\left(\frac{N_a}{N_d(N_a+N_d)}\right)} \tag{7.32}$$
重摻雜側空乏區更窄：若 $N_a\gg N_d$（p⁺n 接面），$x_p\ll x_n$。

7.3 反向偏壓 (7.35–7.52)

⭐ 反向偏壓下的空乏區寬度
$$\boxed{W = \sqrt{\frac{2\epsilon_s(V_{bi}+V_R)}{e}\left(\frac{N_a+N_d}{N_a N_d}\right)}} \tag{7.36}$$
反向偏壓增加 $V_R$→空乏區變寬。$W \propto \sqrt{V_{bi}+V_R}$。
⭐ 接面電容（Junction Capacitance）
$$\boxed{C_j = \frac{\epsilon_s}{W} = \epsilon_s\sqrt{\frac{e}{2\epsilon_s(V_{bi}+V_R)}\frac{N_a N_d}{N_a+N_d}}} \tag{7.40, 7.42}$$
$C_j \propto 1/\sqrt{V_{bi}+V_R}$。反向偏壓越大→$W$ 越大→$C_j$ 越小。
單側突變接面 ($p^+n$) (7.45)：$C_j = A\sqrt{e\epsilon_s N_d/[2(V_{bi}+V_R)]}$。

7.4 接面崩潰 (7.53–7.66)

崩潰電壓（單側突變接面）
$$V_B \approx \frac{\epsilon_s E_{\text{crit}}^2}{2e N_B} \tag{7.60}$$
$E_{\text{crit}}$ = 臨界電場（Si ∼ $3\times10^5$ V/cm）。$N_B$ = 輕摻雜側濃度。
$V_B \propto 1/N_B$——輕摻雜→高崩潰電壓（功率元件）；重摻雜→低崩潰（高速開關）。

7.5 非均勻摻雜接面 (7.67–7.82)

線性梯度接面
$$W \propto (V_{bi}+V_R)^{1/3},\quad C_j \propto (V_{bi}+V_R)^{-1/3} \tag{7.75, 7.78}$$
與突變接面的 $W\propto V^{1/2}$、$C_j\propto V^{-1/2}$ 不同。
超突變接面 (7.80–7.82)：$C_j \propto (V_{bi}+V_R)^{-m}$，$m>1/2$→壓控振盪器用。

8.1.4 理想二極體電流-電壓關係 ⭐ (8.21–8.38)

⭐ 理想二極體方程（Shockley 方程）
$$\boxed{I = I_S\left(e^{eV_a/kT} - 1\right)} \tag{8.27}$$
$$I_S = eA\left(\frac{D_p p_{n0}}{L_p} + \frac{D_n n_{p0}}{L_n}\right) = eA n_i^2\left(\frac{D_p}{L_p N_d} + \frac{D_n}{L_n N_a}\right) \tag{8.28}$$

🔍 推導——從擴散方程到 I-V 關係（18 條公式）
1少數載子擴散方程 (8.21–8.24)
n 側中性區中電洞的穩態擴散方程：$d^2\delta p_n/dx^2 = \delta p_n/L_p^2$。

3⭐ 總電流 (8.27–8.28)

8.2 產生-復合電流 (8.53–8.66)

復合電流（順偏）
$$I_{\text{rec}} \propto e^{eV_a/2kT} \tag{8.58}$$

8.3 小訊號模型 ⭐ (8.67–8.75)

⭐ 擴散電導（Diffusion Conductance）
$$\boxed{g_d = \frac{dI}{dV} = \frac{e}{kT}(I + I_S) \approx \frac{eI}{kT}} \tag{8.68}$$
⭐ 擴散電容（Diffusion Capacitance）
$$\boxed{C_d = \frac{e}{kT}I\,\tau_{p0}} \quad\text{（n⁺p 接面）} \tag{8.75}$$
$C_d \propto I$——順向電流越大，儲存的過量少數載子越多→等效電容越大。
接面電容 $C_j$（第七章）主導反向偏壓；擴散電容 $C_d$ 主導順向偏壓。

8.4–8.5 電荷儲存與穿隧二極體 (8.76–8.85)

二極體反向恢復
$$t_s \approx \tau_{p0}\,\ln\!\left(1 + \frac{I_F}{I_R}\right) \tag{8.79}$$
$t_s$ = 儲存時間。從順向切換到反向時，儲存電荷需時間消散。
穿隧二極體（Tunnel Diode）

10.1.3–11.1.5 能帶彎曲·空乏·反轉

🔍 完整推導——從閘極電壓方程式到 $V_T$

3⭐ 臨界電壓完整表達式

離子植入調整 $V_T$
$$V_T = V_{FB} + 2\phi_f + \frac{\sqrt{4\epsilon_s e N_a\phi_f} + e D_I}{C_{ox}}$$
$D_I$ = 通道離子植入劑量（cm⁻²）。正 $D_I$（p 型植入）→提高 $V_T$；負 $D_I$（n 型植入，如埋入通道）→降低 $V_T$。在現代 CMOS 中，通道植入是精確調控 $V_T$ 的主要手段。

10.3.1–11.3.3 線性區 I-V 推導 ⭐

🔍 完整推導——從通道電荷到線性區 $I_D$
1通道中的電荷密度
在通道中任意點 $y$，有效閘極控制電壓 $=V_{GS}-V_T-V(y)$（$V(y)$ = 該點的通道電位）。
單位面積反轉層電荷：$Q_n(y) = -C_{ox}[V_{GS}-V_T-V(y)]$（只計入超過 $V_T$ 的部分）。
2漂移電流
通道中電場 $E_y = -dV/dy$。電流 $I_D = W\mu_n Q_n(y)E_y$（$W$ = 通道寬度）。
$I_D = W\mu_n C_{ox}[V_{GS}-V_T-V(y)]\,dV/dy$。
3積分求 $I_D$
$I_D\,dy = W\mu_n C_{ox}[V_{GS}-V_T-V]\,dV$。從 $y=0$ ($V=0$) 積到 $y=L$ ($V=V_{DS}$)：$$I_D\,L = W\mu_n C_{ox}\left[(V_{GS}-V_T)V_{DS} - \frac{V_{DS}^2}{2}\right]$$$$\boxed{I_D = \mu_n C_{ox}\frac{W}{L}\left[(V_{GS}-V_T)V_{DS} - \frac{V_{DS}^2}{2}\right]}$$
4極低 $V_{DS}$ 的簡化
$V_{DS}\ll V_{GS}-V_T$→$I_D\approx\mu_n C_{ox}(W/L)(V_{GS}-V_T)V_{DS}$。通道電阻 $R_{ch}=V_{DS}/I_D = 1/[\mu_n C_{ox}(W/L)(V_{GS}-V_T)]$。

10.3.4–11.3.5 飽和區 I-V ⭐

⭐ 飽和區汲極電流
$$\boxed{I_D = \frac{\mu_n C_{ox}}{2}\frac{W}{L}(V_{GS}-V_T)^2}$$
夾止條件：$V_{DS}=V_{GS}-V_T$（汲極端通道厚度→0）。夾止後 $V_{DS}$ 繼續增加→夾止點向源極移動，但 $I_D$ 基本不變（飽和）。
🔍 夾止機制與通道長度調變
1夾止發生的條件
$V_{DS}=V_{GS}-V_T$，此時 $V(L)=V_{DS}$，$Q_n(L)=0$→通道在汲極端厚度為零。
2飽和 $I_D$ 推導
將 $V_{DS}=V_{GS}-V_T$ 代入線性區公式：$I_D(\text{sat})=\mu_n C_{ox}(W/L)[(V_{GS}-V_T)^2-(V_{GS}-V_T)^2/2]=\frac{1}{2}\mu_n C_{ox}(W/L)(V_{GS}-V_T)^2$。
3通道長度調變
$V_{DS}>V_{GS}-V_T$→夾止點距汲極 $\Delta L$→有效通道長度 $=L-\Delta L$。$$I_D = \frac{\mu_n C_{ox}}{2}\frac{W}{L}(V_{GS}-V_T)^2\!\left(1 + \frac{V_{DS}}{V_A}\right)$$$V_A$ 類似 BJT 的 Early 電壓。

10.4 小訊號參數與頻率響應

⭐ 跨導 $g_m$（飽和區）
$$\boxed{g_m = \frac{\partial I_D}{\partial V_{GS}} = \mu_n C_{ox}\frac{W}{L}(V_{GS}-V_T) = \frac{2I_D}{V_{GS}-V_T} = \sqrt{2\mu_n C_{ox}\frac{W}{L}I_D}}$$
BJT: $g_m=I_C/V_T=38.5$ mS/mA（線性）。MOSFET: $g_m=2I_D/(V_{GS}-V_T)$。若 $V_{GS}-V_T=0.3$ V→$g_m=6.7$ mS/mA。MOSFET 的 $g_m$ 效率約為 BJT 的 1/6。
輸出電阻 $r_o$
$$r_o = \frac{\partial V_{DS}}{\partial I_D} = \frac{V_A}{I_D}$$
截止頻率 $f_T$
$$f_T = \frac{g_m}{2\pi(C_{gs}+C_{gd})} \approx \frac{\mu_n(V_{GS}-V_T)}{2\pi L^2}$$
$f_T\propto1/L^2$→通道長度縮小一半→$f_T$ 增為四倍。現代 FinFET ($L_g\sim14$ nm): $f_T\sim300$–$400$ GHz。

10.5 短通道效應與次臨界導通

⭐ 次臨界斜率（Subthreshold Swing）
$$\boxed{S = \frac{dV_{GS}}{d(\log I_D)} \approx \frac{kT}{e}\ln10\left(1 + \frac{C_d}{C_{ox}}\right) \approx 60\left(1 + \frac{C_d}{C_{ox}}\right)\ \text{mV/dec}}$$
$C_d$ = 空乏層電容。室溫物理下限 $=60$ mV/dec。$C_d/C_{ox}$ 越小→$S$ 越接近 60→開關越陡峭→OFF 電流越小。
DIBL（汲極引發能障降低）
$$V_T = V_{T0} - \delta\,V_{DS}$$
短通道時汲極電場穿透到源極側→降低源極端能障→$V_T$ 隨 $V_{DS}$ 增加而下降。DIBL 是限制 $L$ 縮小的關鍵因素。

11.2.1 遷移率退化 (12.14–12.15)

⭐ 遷移率退化公式 (12.14a, 12.14b)

$$\theta = \frac{1}{t_{ox}}\sqrt{\frac{\epsilon_{ox}}{\epsilon_s}}\cdot\frac{\hbar^2}{2m^* kT} \tag{11.14b}$$

11.2.2 速度飽和效應的完整分析

🔍 速度飽和對 $I_D$, $g_m$, 飽和電壓的全面影響
1速度飽和下的跨導
從 (12.15a) 微分：$g_m = \partial I_D/\partial V_{GS}$。短通道極限：$g_m = W C_{ox} v_{\text{sat}}$（常數！）。
2速度飽和下的飽和電壓
長通道：$V_{DS,\text{sat}} = V_{GS}-V_T$（夾止條件）。速度飽和：$V_{DS,\text{sat}} < V_{GS}-V_T$（載子在通道中途已達 $v_{\text{sat}}$→提前夾止）。
3速度飽和對輸出電阻的影響

11.3 熱載子效應·DIBL·次臨界特性

11.4–12.5 短通道臨界電壓·穿通效應

短通道 $V_T$ 模型 (12.77)
$$\Delta V_T \propto -\exp\!\left(-\frac{L}{2l}\right) \tag{11.77}$$
$l$ = 特徵長度（取決於 $t_{ox}$, $x_j$, $x_d$）。$L$ 越小→$\Delta V_T$ 負得越多→$V_T$ roll-off。這是 $L$ 不能無限縮小的根本原因。
穿通效應 (Punch-through)
源極和汲極空乏區在通道下方相遇→形成直接穿通電流路徑→$I_D$ 不再由閘極控制。
閘極漏電流——Fowler-Nordheim 穿隧 (12.87–12.88)

等效氧化層厚度 (12.97)
$$t_{ox,\text{eq}} = \frac{\kappa_{\text{SiO}_2}}{\kappa_{\text{high-}\kappa}}\,t_{\text{high-}\kappa} \tag{11.97}$$
量子力學 $V_T$ 偏移 (12.99)
$$\Delta V_T^{QM} = \frac{\Delta E_0}{e} \tag{11.99}$$
$\Delta E_0$ = 反轉層中最低次能帶的能量（來自垂直方向的量子侷限）。使 $V_T$ 增加 ∼0.1–0.2 V。

12.2.1–10.2.2 電流增益參數 ⭐ (10.19–10.36)

12.1 BJT 作用回顧與多邊結構

BJT 核心參數回顧
$$\alpha = \gamma \cdot \alpha_T,\quad \beta = \frac{\alpha}{1-\alpha}$$
$$I_C = I_S e^{V_{BE}/V_T},\quad I_E = I_B + I_C$$

12.2 BJT 頻率限制——四個時間常數 ⭐

🔍 $f_T$ 的完整推導——四個延遲成分的物理來源
1⭐ 截止頻率定義 (12.97)
$$\boxed{f_T = \frac{1}{2\pi\tau_{ec}}},\quad \tau_{ec} = \tau_e + \tau_b + \tau_d + \tau_c \tag{12.97}$$
2① 射極充電時間 $\tau_e = r_e C_{je}$
$r_e = V_T/I_E \approx 25.9/I_E(\text{mA})$ Ω。$C_{je}$ = BE 接面電容。充放電 BE 接面的 RC 延遲。
3② 基極傳輸時間 $\tau_b = W_B^2/(2D_B)$
⭐ 最重要的延遲成分。來自擴散方程：粒子從 $x=0$ 擴散到 $x=W_B$ 的平均時間。
數值：$W_B=0.1$ µm, $D_B=30$ cm²/s→$\tau_b=1.7$ ps→$f_T\approx100$ GHz（僅此項）。
4③ 集極空乏層渡越時間 $\tau_d = x_d/(2v_{\text{sat}})$
$v_{\text{sat}}\approx10^7$ cm/s。$x_d$ 隨 $V_{CB}$ 增加→高速 BJT 需低 $V_{CB}$ 操作。
5④ 集極充電時間 $\tau_c = r_c C_{jc}$
$r_c$ = 集極串聯電阻。$C_{jc}$ = BC 接面電容。
⭐ 最大振盪頻率 $f_{\max}$ (12.102)
$$\boxed{f_{\max} = \sqrt{\frac{f_T}{8\pi\,r_b\,C_{jc}}}} \tag{12.102}$$
功率增益 = 1 的頻率。$r_b$（基極電阻）和 $C_{jc}$（BC 接面電容）形成輸入-輸出耦合的 RC 濾波器。$r_b C_{jc}$ 是功率增益的瓶頸。
共射極截止頻率 $f_\beta$ (12.105–12.106)
$$f_\beta = \frac{f_T}{\beta_0} \tag{12.105}$$
$$\beta(f) = \frac{\beta_0}{1 + j(f/f_\beta)} \tag{12.106a, 12.106b}$$
$f_\beta$ = 電流增益 $\beta$ 降至低頻值 $\beta_0$ 的 $1/\sqrt{2}$ 時的頻率。$f_\beta \ll f_T$（差 $\beta_0$ 倍）。
射極-集極延遲完整表達式 (12.107–12.108)
$$\tau_{ec} = \frac{V_T}{I_E}C_{je} + \frac{W_B^2}{2D_B} + \frac{x_d}{2v_{\text{sat}}} + r_c C_{jc} \tag{12.107}$$
(12.108) 給出各成分的相對重要性——基極傳輸時間通常主導。

12.3 大訊號開關

開關延遲時間
$$t_s \approx \tau_s\ln\!\left(1 + \frac{I_F}{I_R}\right)$$
$\tau_s$ = 飽和區等效壽命。$I_F$ = 順向基極電流，$I_R$ = 反向抽取電流。
蕭特基箝位防止飽和→$t_s$ 大幅降低（$\tau_s$ 變為主動區壽命，短得多）。

12.4 小訊號模型

⭐ 完整混合-π 模型（含電容）
$$g_m = \frac{I_C}{V_T},\quad r_\pi = \frac{\beta}{g_m},\quad r_o = \frac{V_A}{I_C}$$
$$C_\pi = C_{je} + \tau_F g_m,\quad C_\mu = C_{jc}$$
$C_\pi$（輸入電容）= 接面電容 $C_{je}$ + 擴散電容 $\tau_F g_m$（$\propto I_C$）。
$C_\mu$ = BC 接面電容（Miller 效應→輸入端放大 $(1+g_m R_L)$ 倍）。

12.5 HBT 與 SiGe 技術

HBT 射極注入效率的突破
$$\gamma_{\text{HBT}} \approx 1,\quad \beta_{\text{HBT}} \gg \beta_{\text{BJT}}$$
寬能隙射極→價帶能障阻止電洞從基極注入射極→$I_{pE}\approx0$→$\gamma\approx1$ 即使 $N_E$ 不重於 $N_B$。
可重摻雜基極→$r_b$ 降低→$f_{\max}$ 提升。
