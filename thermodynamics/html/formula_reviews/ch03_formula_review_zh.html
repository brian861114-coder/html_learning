3.3 熵的定義與第二定律不等式 (3.1–3.5)

⭐ 熵的定義 (3.1)
$$\boxed{dS = \frac{\delta q_{\text{rev}}}{T}} \tag{3.1}$$
$S$ = 熵（狀態函數）。$\delta q_{\text{rev}}$ 是可逆過程中交換的熱。$T$ 是絕對溫度。
有限過程的熵變 (3.2)
$$\Delta S = \int_{A}^{B} \frac{\delta q_{\text{rev}}}{T} \tag{3.2}$$
Clausius 不等式 — 不可逆過程 (3.3)
$$dS > \frac{\delta q_{\text{irrev}}}{T} \tag{3.3}$$
⭐ 第二定律的通用形式 (3.4)
$$\boxed{dS \geq \frac{\delta q}{T}} \tag{3.4}$$
等號：可逆過程。不等號：不可逆過程。這是第二定律最根本的數學表述。
⭐ 孤立系統 (3.5)
$$\boxed{dS_{\text{isolated}} \geq 0} \tag{3.5}$$
$\delta q = 0$（孤立）。自發過程 → $dS > 0$；平衡 → $dS = 0$，$S$ 最大。
🔍 為什麼 $\delta q_{\text{rev}}/T$ 是正合微分？
1卡諾定理
所有可逆熱機的效率相同，只取決於溫度：$\eta = 1 - T_c/T_h$。
2對可逆循環
$$\oint \frac{\delta q_{\text{rev}}}{T} = 0$$繞任意可逆循環一周，$\delta q_{\text{rev}}/T$ 的積分為零。
3正合微分判據
繞閉合路徑的積分為零 → $\delta q_{\text{rev}}/T$ 是正合微分 → 存在狀態函數 $S$ 使得 $dS = \delta q_{\text{rev}}/T$。

3.10–3.11 熱機與熱力學溫標 (3.6–3.8)

⭐ 卡諾效率 (3.6)
$$\boxed{\eta = \frac{w_{\text{net}}}{q_h} = 1 - \frac{T_c}{T_h}} \tag{3.6}$$
$T_h$、$T_c$ = 高溫熱庫和低溫熱庫的絕對溫度。任何在 $T_h$ 和 $T_c$ 間運行的熱機效率 ≤ 卡諾效率。
熱力學溫標 (3.7)
$$\frac{|q_h|}{|q_c|} = \frac{T_h}{T_c} \tag{3.7}$$
可逆熱機中，熱量的比值等於絕對溫度的比值——這定義了熱力學溫標。
可逆循環的積分 (3.8)
$$\oint \frac{\delta q_{\text{rev}}}{T} = 0 \tag{3.8}$$
這是 $S$ 為狀態函數的數學證明。
🔍 卡諾效率推導
1卡諾循環四步驟
等溫膨脹（$T_h$）→ 絕熱膨脹 → 等溫壓縮（$T_c$）→ 絕熱壓縮
2淨功
$w_{\text{net}} = q_h + q_c$（$q_c < 0$，向低溫放熱）。
3效率
$$\eta = \frac{w_{\text{net}}}{q_h} = \frac{q_h + q_c}{q_h} = 1 + \frac{q_c}{q_h}$$對可逆循環，$q_h/T_h + q_c/T_c = 0$ → $q_c/q_h = -T_c/T_h$ → $\eta = 1 - T_c/T_h$。

3.13–3.14 最大功與平衡判據 (3.9–3.11)

最大功的微分 (3.9)
$$\delta w_{\text{max}} = dU - T\,dS \tag{3.9}$$
⭐ 等溫最大功 = $\Delta A$ (3.10)
$$w_{\text{max}} = \Delta U - T\Delta S = \Delta A \tag{3.10}$$
$A = U - TS$（亥姆霍茲自由能）。等溫過程中系統能做的最大功等於 $A$ 的減少。
⭐ 孤立系統平衡判據 (3.11)
$$dS_{U,V} = 0 \quad \text{（平衡時 $S$ 最大）} \tag{3.11}$$
⚡ 平衡判據對照表
約束熱力學位平衡條件
$U$, $V$ 固定（孤立）$S$$dS = 0$（$S$ 最大）
$T$, $V$ 固定$A = U - TS$$dA = 0$（$A$ 最小）
$T$, $P$ 固定 ⭐$G = H - TS$$dG = 0$（$G$ 最小）

3.15 第一與第二定律的結合 (3.12–3.15)

⭐ 組合基本方程式 (3.12)
$$\boxed{dU = T\,dS - P\,dV} \tag{3.12}$$
合併 $dU = \delta q + \delta w$、$dS = \delta q_{\text{rev}}/T$、$\delta w_{\text{rev}} = -P\,dV$。整個平衡熱力學的基石。
⭐ 焓的基本方程式 (3.13)
$$\boxed{dH = T\,dS + V\,dP} \tag{3.13}$$
$H = U + PV$ → $dH = dU + P\,dV + V\,dP = (T\,dS - P\,dV) + P\,dV + V\,dP$。
⭐ 亥姆霍茲自由能 (3.14)
$$\boxed{dA = -S\,dT - P\,dV} \tag{3.14}$$
$A = U - TS$。自然變數：$T$ 和 $V$。
⭐ 吉布斯自由能 (3.15)
$$\boxed{dG = -S\,dT + V\,dP} \tag{3.15}$$
$G = H - TS$。自然變數：$T$ 和 $P$——最常用的熱力學位。
🔍 從 $U$ 到四個基本方程式的 Legendre 變換
1起點：$U(S,V)$
$dU = T\,dS - P\,dV$，自然變數 $S$ 和 $V$。
2$H = U + PV$（交換 $V \leftrightarrow P$）
$dH = dU + P\,dV + V\,dP = T\,dS + V\,dP$。自然變數 $S$ 和 $P$。
3$A = U - TS$（交換 $S \leftrightarrow T$）
$dA = dU - T\,dS - S\,dT = -S\,dT - P\,dV$。自然變數 $T$ 和 $V$。
4$G = H - TS$（雙重交換）
$dG = dH - T\,dS - S\,dT = -S\,dT + V\,dP$。自然變數 $T$ 和 $P$。
