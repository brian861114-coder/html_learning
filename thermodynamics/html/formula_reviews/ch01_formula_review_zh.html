1.4 理想氣體狀態方程式 (1.1–1.4)

⭐ 理想氣體定律 (1.1)
$$PV = nRT \tag{1.1}$$
$P$ = 壓力，$V$ = 體積，$n$ = 莫耳數，$R = 8.3144\ \text{J/mol·K}$，$T$ = 絕對溫度 (K)
莫耳形式 (1.2)
$$Pv = RT \tag{1.2}$$
$v = V/n$ = 莫耳體積。此形式只涉及強度量。
STP 下的莫耳體積 (1.3)
$$v = \frac{RT}{P} = \frac{0.082057 \times 273.15}{1} = 22.414\ \text{L/mol} \tag{1.3}$$
理想氣體定律的微分形式 (1.4)
$$P\,dV + V\,dP = nR\,dT \tag{1.4}$$
用於分析 $P$、$V$、$T$ 同時變化的過程。
🔍 理想氣體定律的物理基礎
1波以耳定律（$T$ 固定）
$PV = \text{constant}$（1662）——壓力與體積成反比
2查理定律（$P$ 固定）
$V \propto T$（1787）——體積與絕對溫度成正比
3亞佛加厥假說
相同 $T$、$P$ 下，等體積氣體含等數量分子 → $V \propto n$
4合併 → 理想氣體定律
$$PV = nRT$$$R$ 是比例常數，由實驗測定。
⚡ $R$ 的單位選擇
使用 $PV = nRT$ 時，$R$ 的數值取決於 $P$ 和 $V$ 的單位。最常用：
• SI：$R = 8.3144\ \text{J/mol·K}$（$P$ in Pa, $V$ in m³）
• 大氣壓：$R = 0.082057\ \text{L·atm/mol·K}$（$P$ in atm, $V$ in L）
• 熱化學：$R = 1.987\ \text{cal/mol·K}$

1.5 能量與功的定義 (1.7)

⭐ 壓力–體積功 (1.7)
$$w = \int_{V_1}^{V_2} P_{\text{ext}}\ dV \tag{1.7}$$
系統對抗外壓 $P_{\text{ext}}$ 膨脹時所做的功。膨脹（$dV > 0$）→ 系統對外做功 → $w < 0$（依本書符號慣例）。
🔍 功的符號慣例
1本書慣例
系統對外做功 → $w$ 為負（系統失去能量）
外界對系統做功 → $w$ 為正（系統獲得能量）
2與第一定律的關係
$$\Delta U = q + w$$熱 $q$ 和功 $w$ 都是從外界輸入系統的能量。

1.6 廣延量與強度量 (1.5–1.6)

狀態方程式的一般形式 (1.5)
$$f(P, V, T) = 0 \tag{1.5}$$
三個狀態變數中只有兩個是獨立的——指定任意兩個，第三個由狀態方程式確定。
全微分關係 (1.6)
$$dV = \left(\frac{\partial V}{\partial T}\right)_P dT + \left(\frac{\partial V}{\partial P}\right)_T dP \tag{1.6}$$
$V$ 的微小變化可表示為 $T$ 和 $P$ 變化的線性組合。這是狀態函數的核心數學性質。
🔍 廣延量 vs 強度量判別法
性質廣延量強度量
與系統大小成正比無關
系統均分兩半減半不變
範例$V$, $m$, $U$, $H$, $S$, $G$$T$, $P$, $\rho$, $v = V/n$
兩個廣延量之比→ 強度量（如 $\rho = m/V$）
