第十章

吉布斯自由能–組成圖與二元系統相圖

David R. Gaskell·《Introduction to the Thermodynamics of Materials》·第 342–399 頁
📋 10.0 本章預覽

第九章建立了溶液熱力學的理論基礎——活性、理想溶液、正規溶液模型。第十章是應用章節：展示如何使用 $G$–$x$ 曲線（吉布斯自由能對組成的圖）來建構和詮釋二元相圖。核心工具是共切線法則（Common Tangent Rule）——兩相平衡對應於兩條 $G$–$x$ 曲線的共切線，切點給出平衡相的組成。這使我們能夠從純熱力學數據「計算」出相圖，而不只是「觀察」它。

⚡ 第十章是材料科學家「讀懂相圖」的理論基礎。為什麼共晶系統有 V 形液相線？為什麼混溶間隙在等莫耳組成處最寬？為什麼某些系統有金屬間化合物而其他沒有？所有這些相圖特徵都源自 $G$–$x$ 曲線的形狀——而 $G$–$x$ 曲線的形狀又源自交互作用參數 $\Omega$ 和溫度 $T$ 的競爭。本章教你從源頭理解相圖。

在本章中，我們將學習：

  • $G$–$x$ 曲線的幾何意義——截距給出化學位，曲率決定穩定性
  • 共切線法則——兩相平衡條件 $\mu_A^\alpha = \mu_A^\beta$、$\mu_B^\alpha = \mu_B^\beta$ 的圖形詮釋
  • 三種基本二元相圖——完全互溶、共晶、混溶間隙——的 $G$–$x$ 曲線起源
  • 正規溶液模型在相圖計算中的應用：$\Omega$、$T_c$、和相邊界
  • 液相和固相標準態的選擇對 $G$–$x$ 曲線的影響

10.1–10.2 導論：$G$–$x$ 曲線與化學位的幾何意義

對於一個二元系統，在定溫定壓下，吉布斯自由能是組成的函數：$G = G(x_B)$。這條曲線包含了決定相行為的所有熱力學資訊。

化學位可以從 $G$–$x$ 曲線的幾何構造中直接讀出。在任意組成 $x_B$ 處，畫一條切線——這條切線在 $x_B = 0$（純 A）和 $x_B = 1$（純 B）軸上的截距就是 $\mu_A$ 和 $\mu_B$：

$$\boxed{\mu_A = G - x_B\frac{dG}{dx_B}, \quad \mu_B = G + (1-x_B)\frac{dG}{dx_B}} \tag{10.1}$$

這個「截距法」極其有用：

  • 對於純 A（$x_B = 0$）：$\mu_A = G_A^\circ$（純 A 的 $G$），$\mu_B$ 為有限值（但在此極限下 $x_B \to 0$，$\mu_B \to -\infty$ 如 $RT\ln x_B$）
  • 對於理想溶液：$G = x_A G_A^\circ + x_B G_B^\circ + RT(x_A\ln x_A + x_B\ln x_B)$，代入截距公式可得 $\mu_A = G_A^\circ + RT\ln x_A$——完全符合預期。
💡 截距法的物理直覺：想像 $G$–$x$ 曲線是一道彎曲的繩索。繩索在兩端的「高度」就是純成分的 $G^\circ$。在任何中間組成，繩索的切線在兩端的截距告訴你該成分在此溶液中的化學位——這就是偏莫耳自由能。如果切線的一端高於純成分的 $G^\circ$，該成分在溶液中「不舒服」，傾向離開（$\gamma > 1$）。

10.2A 化學位截距公式的推導

截距公式（10.1）可以從偏莫耳量的定義嚴格導出。對於二元溶液 A–B，總吉布斯自由能可以寫為各成分化學位的加權和：

$$G = n_A\mu_A + n_B\mu_B \tag{10.1a}$$

其中 $n_A$ 和 $n_B$ 是莫耳數。對 (10.1a) 取全微分：

$$dG = \mu_A\,dn_A + \mu_B\,dn_B + n_A\,d\mu_A + n_B\,d\mu_B \tag{10.1b}$$

但 Gibbs–Duhem 關係告訴我們，在定溫定壓下：

$$n_A\,d\mu_A + n_B\,d\mu_B = 0 \tag{10.1c}$$

因此 (10.1b) 化簡為：

$$dG = \mu_A\,dn_A + \mu_B\,dn_B \tag{10.1d}$$

將組成用莫耳分數表達：$x_B = n_B/(n_A+n_B)$，$n = n_A+n_B$。則 $n_A = n(1-x_B)$，$n_B = n x_B$。固定總莫耳數 $n$（即 $dn = 0$），有 $dn_A = -n\,dx_B$，$dn_B = n\,dx_B$。代入 (10.1d)：

$$dG = -\mu_A n\,dx_B + \mu_B n\,dx_B = n(\mu_B - \mu_A)\,dx_B \tag{10.1e}$$

所以：

$$\frac{1}{n}\frac{dG}{dx_B} = \frac{d(G/n)}{dx_B} = \mu_B - \mu_A \tag{10.1f}$$

令 $G_m = G/n$ 為莫耳吉布斯自由能，則：

$$\frac{dG_m}{dx_B} = \mu_B - \mu_A \tag{10.1g}$$

結合 $G_m = (1-x_B)\mu_A + x_B\mu_B$，解出 $\mu_A$ 和 $\mu_B$：

$$\boxed{\mu_A = G_m - x_B\frac{dG_m}{dx_B}}, \qquad \boxed{\mu_B = G_m + (1-x_B)\frac{dG_m}{dx_B}} \tag{10.1h}$$

這就完成了截距公式的嚴格導出。其幾何意義非常清晰：在 $G_m$–$x_B$ 圖上，任意點 $(x_B, G_m)$ 處的切線斜率是 $dG_m/dx_B = \mu_B - \mu_A$，而該切線在 $x_B=0$ 和 $x_B=1$ 的截距分別為 $\mu_A$ 和 $\mu_B$。

🔗 跨章連結 — 第九章：這個推導完全基於第九章中的偏莫耳量關係。回顧 Ch9 §9.1，我們定義了偏莫耳吉布斯自由能 $\bar{G}_i = \mu_i$，並證明了 Gibbs–Duhem 方程 $\sum n_i d\mu_i = 0$。這裡的截距法是 Gibbs–Duhem 方程在幾何上的直接體現。

10.3–10.4 常見二元相圖類型與標準態

定性上，二元相圖的形狀完全由液相和固相的 $G$–$x$ 曲線的相對位置決定。三種最基本的類型：

1. 完全互溶系統（Isomorphous）

液相和固相的 $G$–$x$ 曲線都是簡單的 U 形（$\Omega_{\text{liq}}$ 和 $\Omega_{\text{sol}}$ 都小於臨界值）。兩條曲線在不同的溫度範圍內交叉——在高溫時液相的 $G$ 最低（所有組成下液態穩定），在低溫時固相的 $G$ 最低。中間溫度時，存在一個組成範圍內兩相共存——由共切線決定。

經典例子：Cu–Ni（銅–鎳）、Ag–Au（銀–金）、Mo–W（鉬–鎢）。這些系統中兩種原子的晶體結構相同、原子半徑相近、化學性質相似，因此無論液態還是固態都能完全互溶。

2. 共晶系統（Eutectic）

液相 $G$–$x$ 曲線為簡單 U 形，但固相的 $G$–$x$ 曲線有很高的 $\Omega_{\text{sol}} > 0$——在全組成範圍內，固相的 $G$ 都很高（因為異類原子尺寸或化學性質不匹配）。液相在組成範圍的中間部分「勝出」到遠低於純成分熔點的溫度。這導致 V 形液相線在共晶點交匯。

經典例子：Pb–Sn（鉛–錫，焊料）、Al–Si（鋁–矽，鑄造合金）、Fe–C（鐵–碳，在 1147°C 的萊氏體反應）。共晶反應：$L \rightleftharpoons \alpha + \beta$。

3. 包晶系統（Peritectic）

當固相有較大的負混合焓（$\Omega < 0$）時，可能會出現包晶反應。在包晶溫度，液體和一個固相反應生成另一個固相：$L + \alpha \rightleftharpoons \beta$。

經典例子：Fe–C（鐵–碳，1495°C 的包晶反應）、Pt–Ag（鉑–銀）、Cu–Zn（銅–鋅，黃銅）。

4. 混溶間隙（Miscibility Gap）

當 $\Omega > 2RT$ 時，固相（或液相）的 $G$–$x$ 曲線出現雙凹形狀——在中間組成，$\partial^2 G/\partial x^2 < 0$。這意味著均勻溶液不穩定——它會自發分為兩個不同組成的相（由共切線的兩個切點給出）。這是第九章正規溶液模型的直接應用。

經典例子：Al–Zn（鋁–鋅，室溫附近）、Cu–Co（銅–鈷）、Fe–Cu（鐵–銅）、以及許多氧化物和聚合物混合物。

標準態的選擇

在繪製 $G$–$x$ 曲線時，標準態的選擇至關重要。如果液相和固相使用不同的標準態，兩條 $G$–$x$ 曲線的相對位置會發生位移。通常，液相的標準態選為純液態成分，固相的標準態選為純固態成分——兩者之差就是熔化自由能 $\Delta G_{\text{fus}}(T)$。這決定了 $G$–$x$ 曲線在純成分端的間距。

💡 從 $G$–$x$ 曲線到相圖——一步步：(1) 選擇溫度，(2) 繪製液相和固相的 $G$–$x$ 曲線，(3) 畫出兩條曲線的共切線，(4) 記錄兩個切點的組成。對一系列溫度重複——你就得到了完整的相圖。這就是 CALPHAD（CALculation of PHAse Diagrams）方法的本質——只是使用更複雜的模型來描述 $G$–$x$。

10.5–10.6 正規溶液的相穩定性準則

對於正規溶液，我們可以從第九章的結果精確推導相穩定性的條件。$\Delta G_{\text{mix}}$ 曲線的穩定性由其二階導數決定。回顧第九章中正規溶液的混合自由能：

$$\Delta G_{\text{mix}} = \Omega x_A x_B + RT(x_A\ln x_A + x_B\ln x_B) \tag{10.2}$$

對 $x_B$ 求二階導數（令 $x_A = 1 - x_B$）：

$$\frac{\partial^2 \Delta G_{\text{mix}}}{\partial x_B^2} = -2\Omega + RT\left(\frac{1}{1-x_B} + \frac{1}{x_B}\right) \tag{10.2a}$$

或等價地：

$$\boxed{\frac{\partial^2 \Delta G_{\text{mix}}}{\partial x_B^2} = -2\Omega + \frac{RT}{x_B(1-x_B)}} \tag{10.2b}$$

穩定性判據：

  • $\partial^2 G/\partial x^2 > 0$：溶液對組成漲落穩定——均勻相
  • $\partial^2 G/\partial x^2 < 0$：溶液對組成漲落不穩定——自發分相（spinodal 分解）
  • $\partial^2 G/\partial x^2 = 0$：定義了 spinodal 曲線——穩定與不穩定的邊界

對於對稱的正規溶液（$\Omega$ 與組成無關），spinodal 曲線的解析形式為：

$$T_s = \frac{2\Omega}{R}x_B(1-x_B) \tag{10.3}$$

這是一條拋物線，在 $x_B = 0.5$ 處達到最高點 $T_s = \Omega/2R$——這恰好是臨界溫度 $T_c$。在 $T_s$ 以下，中間組成區域不穩定。

Binodal（共存曲線）由共切線條件決定，位於 spinodal 外部。在 binodal 和 spinodal 之間，溶液處於亞穩（metastable）狀態——對小漲落穩定，但可以通過成核–成長機制分相。

⚡ Spinodal 分解 vs 成核–成長——兩種完全不同的微觀機制：Spinodal 分解產生週期性的組成調幅（如調幅分解玻璃 Vycor 中的互連奈米孔結構），因為任何波長的漲落都不穩定，最快增長的波長主導了最終結構。成核–成長則產生離散的顆粒——需要克服界面能障，結構由形核速率和擴散控制。兩種機制在工業上都有應用——spinodal 用於製造奈米多孔材料，成核–成長用於時效硬化合金。

10.5A 混溶間隙與臨界條件的完整推導

本節對混溶間隙的臨界條件進行完整的數學推導，展示 $\Omega_c = 2RT_c$ 的來源。

臨界點條件

在臨界點，binodal 和 spinodal 重合。數學上，這要求：

$$\frac{\partial^2 \Delta G_{\text{mix}}}{\partial x_B^2} = 0 \quad \text{（spinodal 條件）} \tag{10.4a}$$
$$\frac{\partial^3 \Delta G_{\text{mix}}}{\partial x_B^3} = 0 \quad \text{（臨界點）} \tag{10.4b}$$

從 (10.2b) 出發，令其二階導數為零：

$$-2\Omega + \frac{RT}{x_B(1-x_B)} = 0 \quad \Rightarrow \quad T = \frac{2\Omega}{R}x_B(1-x_B) \tag{10.4c}$$

這是 spinodal 曲線。再求三階導數：

$$\frac{\partial^3 \Delta G_{\text{mix}}}{\partial x_B^3} = RT\frac{-(1-2x_B)}{x_B^2(1-x_B)^2} \tag{10.4d}$$

令其為零：$-RT(1-2x_B)/[x_B^2(1-x_B)^2] = 0$，解得 $x_B = 0.5$。

代入 (10.4c)：

$$T_c = \frac{2\Omega}{R} \cdot 0.5 \cdot 0.5 = \frac{\Omega}{2R} \tag{10.4e}$$

反解 $\Omega$：

$$\boxed{\Omega_c = 2RT_c} \tag{10.4f}$$

這就是正規溶液模型給出的臨界交互作用參數。如果系統的 $\Omega > 2RT_c$，就會出現混溶間隙。

數值實例

系統$\Omega$ (kJ/mol)$T_c$ (K)$2RT_c$ (kJ/mol)實際行為
Cu–Ni≈ 0 (近似理想)——完全互溶
Al–Zn~13~620~10.3$\Omega > 2RT_c$ → 混溶間隙
Au–Pt~20~1150~19.1$\Omega > 2RT_c$ → 高溫混溶間隙

Binodal（共存曲線）的求解

Binodal 由共切線條件決定。對於對稱正規溶液，由於曲線對稱於 $x_B=0.5$，平衡的兩相組成 $x_B^\alpha$ 和 $x_B^\beta$ 滿足 $x_B^\alpha + x_B^\beta = 1$。共切線條件給出：

$$\mu_B^\alpha - \mu_A^\alpha = \mu_B^\beta - \mu_A^\beta \tag{10.4g}$$

對於正規溶液，$\mu_B - \mu_A = (G_B^\circ - G_A^\circ) + \Omega(1-2x_B) + RT\ln[x_B/(1-x_B)]$。

利用對稱性 $x_B^\beta = 1 - x_B^\alpha$，可以得到：

$$\ln\left(\frac{x_B^\alpha}{1-x_B^\alpha}\right) = \frac{\Omega}{RT}(1-2x_B^\alpha) \tag{10.4h}$$

這是一個超越方程，需要數值求解。在 $T$ 接近 $T_c$ 時，binodal 近似為：

$$x_B^\alpha \approx \frac{1}{2} - \frac{1}{2}\sqrt{\frac{T_c - T}{T_c}}, \quad x_B^\beta \approx \frac{1}{2} + \frac{1}{2}\sqrt{\frac{T_c - T}{T_c}} \tag{10.4i}$$

這是典型 mean-field 臨界指數 $1/2$ 的體現。

💡 $T_0$ 曲線的物理意義：$T_0$ 曲線上每一點滿足 $G^\alpha = G^\beta$。它是無成分變化的馬氏體相變的熱力學上限——如果實際溫度高於 $T_0$，即使無擴散也無法發生相變。$T_0$ 曲線在快速凝固和相變誘導塑性（TRIP）鋼的設計中非常重要。

10.7–10.8 從 $G$–$x$ 曲線建構相圖

這是最終的綜合步驟。對於一個二元系統，在任何給定溫度 $T$：

  1. 繪製所有可能相的 $G$–$x$ 曲線（液相、固相 $\alpha$、固相 $\beta$……）
  2. 找出最低的 $G$ 包絡線——在所有 $x_B$ 處，最穩定的狀態是 $G$ 最低的那個（或那幾個相的混合物）。
  3. 如果存在兩相共存區，畫出共切線——兩個切點就是該溫度下的平衡相組成。
  4. 在不同溫度重複——將共切線切點的組成對溫度作圖，就是相圖的邊界。

這整套流程可以完全自動化（現代 CALPHAD 軟體就是這樣做的），但理解其背後的幾何原理至關重要——否則你只是在點擊按鈕，而不理解輸出結果的物理意義。

共切線條件的嚴格推導

當兩相 $\alpha$ 和 $\beta$ 在定溫定壓下平衡時，對每一個成分，化學位必須相等：

$$\mu_A^\alpha(T,P,x^\alpha) = \mu_A^\beta(T,P,x^\beta) \tag{10.5a}$$
$$\mu_B^\alpha(T,P,x^\alpha) = \mu_B^\beta(T,P,x^\beta) \tag{10.5b}$$

從截距公式 (10.1)，$\mu_A$ 和 $\mu_B$ 分別是 $G$–$x$ 曲線上 $x_B=0$ 和 $x_B=1$ 處的切線截距。因此，兩個切點 $(x^\alpha, G^\alpha)$ 和 $(x^\beta, G^\beta)$ 在相同的直線上（共切線），當且僅當：

$$\begin{aligned} G^\alpha - x^\alpha\frac{dG^\alpha}{dx} &= G^\beta - x^\beta\frac{dG^\beta}{dx} \\ G^\alpha + (1-x^\alpha)\frac{dG^\alpha}{dx} &= G^\beta + (1-x^\beta)\frac{dG^\beta}{dx} \end{aligned} \tag{10.5c}$$

這兩個條件等價於以下兩個幾何條件：

$$\frac{G^\beta - G^\alpha}{x^\beta - x^\alpha} = \frac{dG^\alpha}{dx}\bigg|_{x^\alpha} = \frac{dG^\beta}{dx}\bigg|_{x^\beta} \tag{10.5d}$$

即：兩點連線的斜率等於各自曲線在該點的切線斜率。這是共切線法則的標準表述。

數值實例：正規溶液共晶系統

考慮一個假想二元系統：液相互溶（$\Omega_{\text{liq}} \approx 0$），固相有正混合焓（$\Omega_{\text{sol}} = +20\ \text{kJ/mol}$）。純 A 的熔點 $T_A = 1200\ \text{K}$，純 B 的熔點 $T_B = 900\ \text{K}$。熔化熵取 Richard 規則值 $\Delta S_{\text{fus}} \approx R$。

在 800 K（低於兩個純成分的熔點），固相的 $G$ 在純 A 和純 B 端低於液相——但中間組成處，固相的 $G$–$x$ 由於 $\Omega_{\text{sol}} > 0$ 而向上彎曲，液相的 $G$ 反而更低。這導致典型的 V 形共晶相圖——共晶點估計在約 600 K、$x_B \approx 0.4$。

💡 數值直覺——$\Omega$ 如何影響共晶溫度：$\Omega$ 越大，固相 $G$–$x$ 曲線在中間彎曲越厲害，共晶點越低。這就是為什麼某些系統形成「深共晶」（deep eutectic）——交互作用參數大導致固相極不穩定，液相穩定到遠低於純成分熔點的溫度。深共晶溶劑（DES）在綠色化學中是新興的研究熱點。

10.7A 液相線與固相線的計算：正規溶液模型

本節展示如何從正規溶液模型定量計算液–固相邊界。

問題設定

考慮一個二元系統 A–B，其中：

  • 液體為理想溶液，$\Omega_{\text{liq}} = 0$
  • 固體為正規溶液，$\Omega_{\text{sol}} = \Omega_0$
  • 純 A 的熔點 $T_{m,A}$，熔化熵 $\Delta S_{\text{fus},A}$
  • 純 B 的熔點 $T_{m,B}$，熔化熵 $\Delta S_{\text{fus},B}$

液相吉布斯自由能

$$G_{\text{liq}}(x,T) = (1-x)G_A^{\circ,\text{liq}} + x G_B^{\circ,\text{liq}} + RT[(1-x)\ln(1-x) + x\ln x] \tag{10.6a}$$

固相吉布斯自由能

$$G_{\text{sol}}(x,T) = (1-x)G_A^{\circ,\text{sol}} + x G_B^{\circ,\text{sol}} + \Omega_0 x(1-x) + RT[(1-x)\ln(1-x) + x\ln x] \tag{10.6b}$$

兩者之差涉及純成分的熔化自由能：

$$\Delta G_{\text{fus},A}(T) = G_A^{\circ,\text{liq}} - G_A^{\circ,\text{sol}} \approx \Delta S_{\text{fus},A}(T_{m,A} - T) \tag{10.6c}$$
$$\Delta G_{\text{fus},B}(T) = G_B^{\circ,\text{liq}} - G_B^{\circ,\text{sol}} \approx \Delta S_{\text{fus},B}(T_{m,B} - T) \tag{10.6d}$$

其中採用了近似 $\Delta H_{\text{fus}}$ 和 $\Delta S_{\text{fus}}$ 與溫度無關。

平衡條件

在液相線和固相線上，液–固兩相平衡，滿足共切線條件：

$$\mu_A^{\text{liq}}(x_{\text{liq}},T) = \mu_A^{\text{sol}}(x_{\text{sol}},T) \tag{10.6e}$$
$$\mu_B^{\text{liq}}(x_{\text{liq}},T) = \mu_B^{\text{sol}}(x_{\text{sol}},T) \tag{10.6f}$$

代入化學位表達式：

$$\mu_A^{\text{liq}} = G_A^{\circ,\text{liq}} + RT\ln(1-x_{\text{liq}}) \tag{10.6g}$$
$$\mu_A^{\text{sol}} = G_A^{\circ,\text{sol}} + \Omega_0 x_{\text{sol}}^2 + RT\ln(1-x_{\text{sol}}) \tag{10.6h}$$
$$\mu_B^{\text{liq}} = G_B^{\circ,\text{liq}} + RT\ln x_{\text{liq}} \tag{10.6i}$$
$$\mu_B^{\text{sol}} = G_B^{\circ,\text{sol}} + \Omega_0 (1-x_{\text{sol}})^2 + RT\ln x_{\text{sol}} \tag{10.6j}$$

整理得到一組耦合超越方程：

$$\begin{aligned} \Delta S_{\text{fus},A}(T_{m,A} - T) &= \Omega_0 x_{\text{sol}}^2 + RT\ln\frac{1-x_{\text{sol}}}{1-x_{\text{liq}}} \\ \Delta S_{\text{fus},B}(T_{m,B} - T) &= \Omega_0 (1-x_{\text{sol}})^2 + RT\ln\frac{x_{\text{sol}}}{x_{\text{liq}}} \end{aligned} \tag{10.6k}$$

對給定的 $x_{\text{sol}}$，可以數值求解 $x_{\text{liq}}$ 和 $T$。這就是 CALPHAD 核心計算的簡化版。

極限情況

  • 理想系統（$\Omega_0 = 0$）：方程簡化為 $x_{\text{liq}}$ 和 $x_{\text{sol}}$ 透過熔化自由能直接關聯。這是標準的液–固理想溶液模型。
  • 稀釋極限（$x_B \ll 1$）：液相線斜率由 van't Hoff 關係給出 $dT/dx_{\text{B}} \approx RT_m^2/\Delta H_{\text{fus}}$。
  • 共晶極限：當 $\Omega_0$ 足夠大時，兩個固相溶解度極限在低溫相交，形成共晶點。
🔗 跨章連結 — 第七章：純成分的熔化自由能 $\Delta G_{\text{fus}}(T)$ 來自 Clapeyron 方程（Ch7 §7.3）。Clapeyron 方程給出 $dP/dT = \Delta S/\Delta V$，對固–液平衡積分得到熔化曲線的 $T$–$P$ 關係。在此處，我們在定壓下使用 $\Delta G_{\text{fus}}(T) \approx \Delta S_{\text{fus}}(T_m - T)$。

🔗 跨章節聯繫：從單組元到多組元相圖

本章的內容不是孤立的——它建立在之前章節的基礎上，並為後續章節鋪平道路。以下是主要的跨章連結：

與第七章（Clapeyron 方程與單組元相圖）的聯繫

第七章（Ch7）介紹了單組元系統的相平衡——Clapeyron 方程 $dP/dT = \Delta S/\Delta V$ 描述了純物質的相邊界。我們在第十章中遇到的每個二元相圖都「繼承」了其純成分的單組元相圖特徵：

  • 純 A 和純 B 的熔點 $T_{m,A}$、$T_{m,B}$ 是二元相圖的錨點——共晶和包晶溫度由這些純成分熔點和交互作用參數共同決定。
  • 熔化自由能 $\Delta G_{\text{fus}}(T) = \Delta H_{\text{fus}} - T\Delta S_{\text{fus}}$ 直接來自 Clapeyron 關係對液–固平衡的積分（Ch7 §7.4）。
  • 在二元相圖的邊界（純 A 或純 B 附近），相邊界斜率可以與 Clapeyron 方程的推廣形式——van't Hoff 關係——連結。

與第九章（溶液熱力學）的聯繫

第九章是第十章最直接的理論基礎。以下是從第九章「繼承」的核心概念：

  • 正規溶液模型（Ch9 §9.5）：$\Delta G_{\text{mix}} = \Omega x_A x_B + RT(x_A\ln x_A + x_B\ln x_B)$ 是第十章的「原料」。第十章中所有 $G$–$x$ 曲線都基於這個表達式（或其推廣形式）。
  • 活性與活度係數（Ch9 §9.2–9.3）：$\mu_i = \mu_i^\circ + RT\ln a_i$。活性 $a_i$ 偏離理想值的程度決定了 $G$–$x$ 曲線偏離理想曲線的程度，進而決定相圖形狀。
  • 偏莫耳量與 Gibbs–Duhem 方程（Ch9 §9.1）：化學位截距法（10.1）和共切線條件（10.5）都是 Gibbs–Duhem 方程的直接應用。
  • Richard 規則（Ch9 §9.6）：許多金屬的熔化熵 $\Delta S_{\text{fus}} \approx R$（或更準確地，約 8–10 J/mol·K），這簡化了相圖計算。

與第十三章（反應系統的相穩定性）的聯繫

第十三章（Ch13）將相圖的概念推廣到化學反應系統：

  • 在第十章中，我們討論了物理平衡——液–固、液–液、固–固相變，不涉及化學反應。
  • 在第十三章中，引入了化學反應的平衡——例如氧化物、硫化物的還原反應。$G$–$x$ 曲線推廣為 $G$–組成圖，其中「成分」包括化合物。
  • 第十章中的共切線法在第十三章中推廣為優勢區圖（Predominance Diagram）和相穩定性圖（Phase Stability Diagram）——在 $T$ 和 $P$ 或 $T$ 和化學位空間中繪製穩定相區域。
  • 混合焓 $\Omega$ 的概念在第十三章中推廣為化合物的生成自由能 $\Delta G_f^\circ$。
📚 總結——從第七章到第十三章的脈絡：
  1. Ch7：單組元相圖 ← Clapeyron 方程（$dP/dT$）
  2. Ch9：溶液熱力學 ← 活性、正規溶液模型（$\Omega$、$\Delta G_{\text{mix}}$）
  3. Ch10：二元相圖 ← 共切線法 + $G$–$x$ 曲線 ← 當前章節
  4. Ch13：反應系統相穩定性 ← 優勢區圖、化學位空間
這一系列是熱力學從「純物質」到「溶液」再到「化學反應系統」的自然延伸。

本章摘要

  1. 化學位截距法：$\mu_A = G - x_B(dG/dx_B)$。$G$–$x$ 曲線的切線截距直接給出各成分的化學位。推導基於偏莫耳量定義和 Gibbs–Duhem 關係。
  2. 相穩定性判據：$\partial^2 G/\partial x^2 > 0$→穩定；$<0$→不穩定（spinodal 分解）。臨界條件 $\Omega_c = 2RT_c$ 來自正規溶液模型。
  3. 共切線法則：兩相共存 ⇔ 共切線。切點 → 平衡相的組成。這是建構相圖的幾何核心。嚴格推導給出 $\mu_A^\alpha = \mu_A^\beta$、$\mu_B^\alpha = \mu_B^\beta$。
  4. 四種基本相圖：完全互溶（$\Omega$ 小）、共晶（$\Omega_{\text{sol}} \gg 0$）、包晶（$\Omega < 0$）、混溶間隙（$\Omega > 2RT$→分相）。
  5. 從 $G$–$x$ 到相圖：在不同溫度畫 $G$–$x$ 曲線→找共切線→記錄切點→繪製 $T$–$x$ 圖。CALPHAD 方法的原理。
  6. 液固相線計算：正規溶液模型給出耦合超越方程 (10.6k)，可數值求解液相線和固相線。
  7. 跨章節聯繫：Ch7 提供純物質熔點和 Clapeyron 基礎，Ch9 提供溶液模型和活性框架，Ch10 整合為二元相圖，Ch13 推廣到化學反應系統。
