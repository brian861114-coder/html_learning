第三章：導數（Derivatives）

Calculus Vol 1 · OpenStax · Gilbert Strang & Edwin Herman

導數的定義

⚡ 核心概念
導數 $f'(x)$ 測量函數在某一點的瞬時變化率，即切線斜率。導數是微積分的核心運算，連結變化量與函數行為。
定義 — 差商極限
$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$
若極限存在，則稱 $f$ 在 $x$ 處可微（differentiable）。可微 ⇒ 連續，但連續 ⇏ 可微。
等價定義 — 另一種寫法
$$f'(a) = \lim_{x \to a} \frac{f(x) - f(a)}{x - a}$$
此形式在計算特定點 $x=a$ 的導數時特別方便。
導數的記號
$$f'(x) = y' = \frac{dy}{dx} = \frac{df}{dx} = \frac{d}{dx}f(x) = D_x f(x)$$
Leibniz 記號 $\frac{dy}{dx}$ 強調導數是 $y$ 對 $x$ 的變化率比值，在連鎖律中特別有用。
🔍 定義推導 — 從割線到切線
1割線斜率（平均變化率）
過點 $(x, f(x))$ 和 $(x+h, f(x+h))$ 的割線斜率：$$m_{\text{sec}} = \frac{f(x+h) - f(x)}{h}$$
2取極限 → 切線斜率（瞬時變化率）
令 $h \to 0$，割線趨近切線：$$f'(x) = \lim_{h \to 0} m_{\text{sec}} = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$
3切線方程式
在 $x=a$ 處的切線：$$y - f(a) = f'(a)(x - a)$$
📐 例題 — 用定義求 $f(x)=x^2$ 在 $x=3$ 的導數
$f'(3) = \lim_{h \to 0} \frac{(3+h)^2 - 9}{h} = \lim_{h \to 0} \frac{9 + 6h + h^2 - 9}{h} = \lim_{h \to 0} (6 + h) = 6$
∴ 在 $x=3$ 處的切線斜率為 6，切線方程式：$y - 9 = 6(x-3)$

基本微分規則 — 冪法則·常數法則·和差法則

常數法則（Constant Rule）
$$\frac{d}{dx}[c] = 0, \quad c \in \mathbb{R}$$
常數函數的圖形是水平線，斜率處處為 0。
⭐ 冪法則（Power Rule）
$$\boxed{\frac{d}{dx}[x^n] = n x^{n-1}}, \quad n \in \mathbb{R}$$
適用於任意實數冪次 $n$（正整數、負整數、分數……）。例：$\frac{d}{dx}[x^5]=5x^4$，$\frac{d}{dx}[x^{-2}]=-2x^{-3}$，$\frac{d}{dx}[\sqrt{x}]=\frac{1}{2\sqrt{x}}$。
常數倍法則（Constant Multiple Rule）
$$\frac{d}{dx}[c \cdot f(x)] = c \cdot f'(x)$$
常數可以提出微分運算之外。
和差法則（Sum/Difference Rule）
$$\frac{d}{dx}[f(x) \pm g(x)] = f'(x) \pm g'(x)$$
微分是線性運算：和的導數 = 導數的和。
🔍 冪法則推導（正整數 $n$）
1二項式展開
$(x+h)^n = x^n + n x^{n-1}h + \binom{n}{2}x^{n-2}h^2 + \cdots + h^n$
2代入差商
$$\frac{(x+h)^n - x^n}{h} = \frac{n x^{n-1}h + \binom{n}{2}x^{n-2}h^2 + \cdots + h^n}{h} = n x^{n-1} + h \cdot (\cdots)$$
3取極限 $h \to 0$
所有含 $h$ 的項 → 0，僅剩 $n x^{n-1}$。$$\therefore \frac{d}{dx}[x^n] = n x^{n-1}$$
📐 例題 — $f(x)=3x^4-5x^2+2x-7$ 的導數
$f'(x) = 3 \cdot 4x^3 - 5 \cdot 2x + 2 \cdot 1 - 0 = 12x^3 - 10x + 2$

積法則與商法則

⚡ 關鍵提醒
積的導數 ≠ 導數的積！$(fg)' \neq f'g'$。同樣，商的導數 ≠ 導數的商。
⭐ 積法則（Product Rule）
$$\boxed{\frac{d}{dx}[f(x) g(x)] = f'(x) g(x) + f(x) g'(x)}$$
口訣：「前微後不微，前不微後微，兩者相加。」
⭐ 商法則（Quotient Rule）
$$\boxed{\frac{d}{dx}\!\left[\frac{f(x)}{g(x)}\right] = \frac{f'(x) g(x) - f(x) g'(x)}{[g(x)]^2}}$$
口訣：「下微上不微，減上微下不微，除以下平方。」（分子：$g f' - f g'$）
注意次序：分子是 $g f'$ 減 $f g'$，不是反過來的。
🔍 積法則推導
1寫出差商
$$\frac{f(x+h)g(x+h) - f(x)g(x)}{h}$$
2加減 $f(x+h)g(x)$（巧妙拆項）
$$= \frac{f(x+h)g(x+h) - f(x+h)g(x) + f(x+h)g(x) - f(x)g(x)}{h}$$$$= f(x+h)\frac{g(x+h)-g(x)}{h} + g(x)\frac{f(x+h)-f(x)}{h}$$
3取極限 $h \to 0$
$\lim f(x+h)=f(x)$（連續性），$\lim \frac{g(x+h)-g(x)}{h}=g'(x)$，$\lim \frac{f(x+h)-f(x)}{h}=f'(x)$。$$\therefore (fg)' = f g' + g f'$$
📐 例題 — $h(x) = (x^2+1)(x^3-2x)$ 的導數
$f(x)=x^2+1$, $f'(x)=2x$
$g(x)=x^3-2x$, $g'(x)=3x^2-2$
$h'(x) = 2x(x^3-2x) + (x^2+1)(3x^2-2) = 2x^4-4x^2 + 3x^4 - 2x^2 + 3x^2 - 2 = 5x^4 - 3x^2 - 2$
📐 例題 — $h(x) = \frac{x^2}{x+1}$ 的導數
$f(x)=x^2$, $f'(x)=2x$；$g(x)=x+1$, $g'(x)=1$
$h'(x) = \frac{2x(x+1) - x^2(1)}{(x+1)^2} = \frac{2x^2+2x - x^2}{(x+1)^2} = \frac{x^2+2x}{(x+1)^2}$

三角函數導數 — sin / cos / tan

⚡ 重要前提
以下公式在 $x$ 以弧度為單位時成立。若使用度數，需先轉換。兩個基本極限：$\lim_{h\to 0}\frac{\sin h}{h}=1$，$\lim_{h\to 0}\frac{\cos h - 1}{h}=0$。
⭐ sin x 的導數
$$\boxed{\frac{d}{dx}[\sin x] = \cos x}$$
⭐ cos x 的導數
$$\boxed{\frac{d}{dx}[\cos x] = -\sin x}$$
注意負號！$\cos$ 的導數是 $-\sin$，不是 $+\sin$。
⭐ tan x 的導數
$$\boxed{\frac{d}{dx}[\tan x] = \sec^2 x}$$
可由 $\tan x = \frac{\sin x}{\cos x}$ 及商法則推出。
🔍 sin 與 cos 導數推導
1sin 導數 — 使用和角公式
$$\frac{d}{dx}\sin x = \lim_{h \to 0} \frac{\sin(x+h)-\sin x}{h} = \lim_{h \to 0} \frac{\sin x \cos h + \cos x \sin h - \sin x}{h}$$$$= \sin x \cdot \lim_{h\to 0}\frac{\cos h - 1}{h} + \cos x \cdot \lim_{h\to 0}\frac{\sin h}{h}$$$$= \sin x \cdot 0 + \cos x \cdot 1 = \cos x$$
2cos 導數 — 同樣手法
$$\frac{d}{dx}\cos x = \lim_{h \to 0} \frac{\cos(x+h)-\cos x}{h} = \lim_{h \to 0} \frac{\cos x \cos h - \sin x \sin h - \cos x}{h}$$$$= \cos x \cdot 0 - \sin x \cdot 1 = -\sin x$$
3tan 導數 — 商法則
$$\frac{d}{dx}\tan x = \frac{d}{dx}\frac{\sin x}{\cos x} = \frac{\cos x \cdot \cos x - \sin x \cdot (-\sin x)}{\cos^2 x} = \frac{\cos^2 x + \sin^2 x}{\cos^2 x} = \frac{1}{\cos^2 x} = \sec^2 x$$
📐 例題 — $f(x)=x^2\sin x$ 的導數
使用積法則：$f'(x) = 2x\sin x + x^2\cos x$

三角函數導數 — cot / sec / csc

cot x 的導數
$$\boxed{\frac{d}{dx}[\cot x] = -\csc^2 x}$$
$\cot x = \frac{\cos x}{\sin x}$，使用商法則可得。
sec x 的導數
$$\boxed{\frac{d}{dx}[\sec x] = \sec x \tan x}$$
$\sec x = \frac{1}{\cos x}$，使用商法則或直接由 $(\cos x)^{-1}$ 用連鎖律。
csc x 的導數
$$\boxed{\frac{d}{dx}[\csc x] = -\csc x \cot x}$$
注意負號！$\csc x = \frac{1}{\sin x}$。
🔍 sec 與 csc 導數推導
1sec 導數 — 使用商法則
$$\frac{d}{dx}\sec x = \frac{d}{dx}\frac{1}{\cos x} = \frac{0\cdot\cos x - 1\cdot(-\sin x)}{\cos^2 x} = \frac{\sin x}{\cos^2 x} = \frac{1}{\cos x}\cdot\frac{\sin x}{\cos x} = \sec x \tan x$$
2csc 導數 — 同樣手法
$$\frac{d}{dx}\csc x = \frac{d}{dx}\frac{1}{\sin x} = \frac{0\cdot\sin x - 1\cdot\cos x}{\sin^2 x} = -\frac{\cos x}{\sin^2 x} = -\csc x \cot x$$
3cot 導數 — 商法則
$$\frac{d}{dx}\cot x = \frac{d}{dx}\frac{\cos x}{\sin x} = \frac{(-\sin x)\sin x - \cos x(\cos x)}{\sin^2 x} = -\frac{\sin^2 x + \cos^2 x}{\sin^2 x} = -\frac{1}{\sin^2 x} = -\csc^2 x$$
📐 例題 — $f(x)=\sec x \tan x$ 的導數
使用積法則：$f'(x) = (\sec x \tan x)\tan x + \sec x(\sec^2 x) = \sec x \tan^2 x + \sec^3 x$

⭐ 連鎖律（Chain Rule）

⚡ 最重要的微分規則
連鎖律是處理複合函數微分的核心工具。若 $y=f(u)$ 且 $u=g(x)$，則 $y$ 對 $x$ 的變化率 = $y$ 對 $u$ 的變化率 × $u$ 對 $x$ 的變化率。
⭐ 連鎖律 — 標準形式
$$\boxed{\frac{d}{dx}\bigl[f(g(x))\bigr] = f'(g(x)) \cdot g'(x)}$$
「外函數微分（保持內函數不變）× 內函數微分」
Leibniz 記號形式
$$\boxed{\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}}$$
此形式直觀地展示了「變化率相乘」的概念，可推廣到多層複合。
多層連鎖律
$$\frac{d}{dx}\bigl[f(g(h(x)))\bigr] = f'(g(h(x))) \cdot g'(h(x)) \cdot h'(x)$$
從最外層微分到最內層，每一層的導數相乘。
🔍 連鎖律推導（直觀版）
1設 $u = g(x)$，$y = f(u)$
當 $x$ 變化 $\Delta x$，$u$ 變化 $\Delta u = g(x+\Delta x)-g(x)$，$y$ 變化 $\Delta y$。
2變化率相乘
$$\frac{\Delta y}{\Delta x} = \frac{\Delta y}{\Delta u} \cdot \frac{\Delta u}{\Delta x}$$取極限 $\Delta x \to 0$（則 $\Delta u \to 0$）：$$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$$
📐 例題 1 — $h(x)=\sin(x^2)$
外函數 $f(u)=\sin u$，$f'(u)=\cos u$。內函數 $g(x)=x^2$，$g'(x)=2x$。
$h'(x) = \cos(x^2) \cdot 2x = 2x\cos(x^2)$
📐 例題 2 — $h(x)=\sqrt{3x^2+1}$
$h(x) = (3x^2+1)^{1/2}$
$h'(x) = \frac{1}{2}(3x^2+1)^{-1/2} \cdot 6x = \frac{3x}{\sqrt{3x^2+1}}$

隱函數微分（Implicit Differentiation）

⚡ 當 $y$ 無法（或不方便）寫成 $y=f(x)$ 的顯式時使用
隱函數微分步驟
對方程式兩邊同時對 $x$ 微分，將 $y$ 視為 $x$ 的函數，每遇到 $y$ 就乘以 $\frac{dy}{dx}$（連鎖律）。
方程式 $F(x,y)=0$ 定義隱函數 $y=y(x)$。對 $F(x,y(x))=0$ 兩邊對 $x$ 微分：$\frac{\partial F}{\partial x} + \frac{\partial F}{\partial y}\frac{dy}{dx}=0$，因此 $\frac{dy}{dx} = -\frac{F_x}{F_y}$。
🔍 隱函數微分推導與公式
1基本原則
$$\frac{d}{dx}[y^n] = n y^{n-1} \cdot \frac{dy}{dx}$$對 $y$ 的任何函數微分時，都要用連鎖律乘上 $\frac{dy}{dx}$。
2隱函數定理
若 $F(x,y)=0$ 且 $F_y \neq 0$：$$\boxed{\frac{dy}{dx} = -\frac{F_x}{F_y}}$$
📐 例題 — 求圓 $x^2+y^2=25$ 在點 $(3,4)$ 的切線斜率
兩邊對 $x$ 微分：$2x + 2y\frac{dy}{dx} = 0$
⇒ $\frac{dy}{dx} = -\frac{x}{y}$
在 $(3,4)$：$\frac{dy}{dx} = -\frac{3}{4}$

反三角函數導數 — arcsin / arccos / arctan

⚡ 反三角函數的定義域限制
反三角函數的主值範圍限制了其導數的定義域。例如 $\arcsin x$ 定義在 $[-1,1]$，導數在 $x=\pm 1$ 處不存在。
⭐ arcsin x 的導數
$$\boxed{\frac{d}{dx}[\arcsin x] = \frac{1}{\sqrt{1-x^2}}}, \quad |x| < 1$$
arccos x 的導數
$$\boxed{\frac{d}{dx}[\arccos x] = -\frac{1}{\sqrt{1-x^2}}}, \quad |x| < 1$$
注意負號！$\arccos$ 導數與 $\arcsin$ 導數差一個負號。
⭐ arctan x 的導數
$$\boxed{\frac{d}{dx}[\arctan x] = \frac{1}{1+x^2}}$$
定義在全實數軸上，導數恆正。
🔍 arcsin 導數推導（隱函數法）
1設 $y = \arcsin x$
則 $\sin y = x$，且 $y \in [-\pi/2, \pi/2]$。
2隱函數微分
$$\frac{d}{dx}[\sin y] = \frac{d}{dx}[x] \quad\Rightarrow\quad \cos y \cdot \frac{dy}{dx} = 1 \quad\Rightarrow\quad \frac{dy}{dx} = \frac{1}{\cos y}$$
3用 $\cos y = \sqrt{1-\sin^2 y} = \sqrt{1-x^2}$
因 $y \in [-\pi/2, \pi/2]$，$\cos y \geq 0$，取正根。$$\therefore \frac{dy}{dx} = \frac{1}{\sqrt{1-x^2}}$$
📐 例題 — $f(x)=\arcsin(3x)$ 的導數
使用連鎖律：$f'(x) = \frac{1}{\sqrt{1-(3x)^2}} \cdot 3 = \frac{3}{\sqrt{1-9x^2}}$

反三角函數導數 — arccot / arcsec / arccsc

arccot x 的導數
$$\boxed{\frac{d}{dx}[\operatorname{arccot} x] = -\frac{1}{1+x^2}}$$
arcsec x 的導數
$$\boxed{\frac{d}{dx}[\operatorname{arcsec} x] = \frac{1}{|x|\sqrt{x^2-1}}}, \quad |x| > 1$$
絕對值！分子是 $|x|$，不是 $x$。
arccsc x 的導數
$$\boxed{\frac{d}{dx}[\operatorname{arccsc} x] = -\frac{1}{|x|\sqrt{x^2-1}}}, \quad |x| > 1$$
🔍 arcsec 導數推導
1設 $y = \operatorname{arcsec} x$
則 $\sec y = x$，$y \in [0,\pi/2) \cup (\pi/2,\pi]$。
2隱函數微分
$$\frac{d}{dx}[\sec y] = \frac{d}{dx}[x] \quad\Rightarrow\quad \sec y \tan y \cdot \frac{dy}{dx} = 1$$$$\frac{dy}{dx} = \frac{1}{\sec y \tan y}$$
3化簡
$\sec y = x$，$\tan y = \sqrt{\sec^2 y - 1} = \sqrt{x^2-1}$（值域內 $\tan y \geq 0$ 當 $y\in[0,\pi/2)$，$\tan y \leq 0$ 當 $y\in(\pi/2,\pi]$），搭配 $\sec y$ 的符號得 $|x|$）。$$\therefore \frac{dy}{dx} = \frac{1}{|x|\sqrt{x^2-1}}$$

指數函數與對數函數的導數

⭐ eˣ 的導數
$$\boxed{\frac{d}{dx}[e^x] = e^x}$$
$e^x$ 是唯一導數等於自身的函數（不考慮常數倍）。這是自然指數的特殊性質。
aˣ 的導數（$a>0, a\neq 1$）
$$\boxed{\frac{d}{dx}[a^x] = a^x \ln a}$$
可由 $a^x = e^{x\ln a}$ 及連鎖律推出。
⭐ ln x 的導數
$$\boxed{\frac{d}{dx}[\ln x] = \frac{1}{x}}, \quad x > 0$$
一般對數的導數
$$\boxed{\frac{d}{dx}[\log_a x] = \frac{1}{x \ln a}}, \quad x > 0$$
🔍 ln x 導數推導
1使用定義
$$\frac{d}{dx}\ln x = \lim_{h \to 0} \frac{\ln(x+h)-\ln x}{h} = \lim_{h \to 0} \frac{1}{h}\ln\!\left(1+\frac{h}{x}\right)$$
2使用 $\lim_{n\to\infty}(1+1/n)^n = e$
令 $n=x/h$：$$= \lim_{n\to\infty} \frac{n}{x}\ln\!\left(1+\frac{1}{n}\right) = \frac{1}{x}\lim_{n\to\infty}\ln\!\left[\left(1+\frac{1}{n}\right)^n\right] = \frac{1}{x}\ln e = \frac{1}{x}$$
📐 例題 — $f(x)=e^{x^2}\ln x$ 的導數
$f'(x) = e^{x^2}\cdot 2x \cdot \ln x + e^{x^2} \cdot \frac{1}{x} = e^{x^2}\!\left(2x\ln x + \frac{1}{x}\right)$

高階導數（Higher-Order Derivatives）

二階導數
$$f''(x) = \frac{d^2y}{dx^2} = \frac{d}{dx}\!\left[\frac{dy}{dx}\right]$$
二階導數測量變化率的變化率，即加速度。$f''(x)>0$ ⇒ 凹向上（concave up）；$f''(x)<0$ ⇒ 凹向下（concave down）。
n 階導數記號
$$f^{(n)}(x) = \frac{d^n y}{dx^n}$$
$f^{(4)}$ 表示四階導數（括號避免與冪次混淆）。
📐 例題 — $f(x)=x^4$ 的各階導數
$f'(x)=4x^3$，$f''(x)=12x^2$，$f'''(x)=24x$，$f^{(4)}(x)=24$，$f^{(5)}(x)=0$

對數微分法（Logarithmic Differentiation）

⚡ 處理複雜乘積、冪指函數的技巧
對 $y=f(x)$ 取 $\ln$，利用 $\ln(AB)=\ln A+\ln B$ 將乘積轉為和，然後隱函數微分。
對數微分法步驟
1. 取 $\ln y = \ln f(x)$
2. 化簡右側（利用對數性質）
3. 兩邊對 $x$ 微分：$\frac{1}{y}\frac{dy}{dx} = \cdots$
4. 解得 $\frac{dy}{dx} = y \cdot (\cdots)$
📐 例題 — $y = x^x$ 的導數
$\ln y = \ln(x^x) = x\ln x$
$\frac{1}{y}\frac{dy}{dx} = \ln x + x\cdot\frac{1}{x} = \ln x + 1$
$\frac{dy}{dx} = x^x(\ln x + 1)$
📐 例題 — $y = \frac{(x^2+1)^3\sqrt{x-1}}{x^4}$ 的導數
$\ln y = 3\ln(x^2+1) + \frac{1}{2}\ln(x-1) - 4\ln x$
$\frac{y'}{y} = 3\cdot\frac{2x}{x^2+1} + \frac{1}{2}\cdot\frac{1}{x-1} - \frac{4}{x}$
$y' = \frac{(x^2+1)^3\sqrt{x-1}}{x^4}\left(\frac{6x}{x^2+1} + \frac{1}{2(x-1)} - \frac{4}{x}\right)$

sin$\frac{d}{dx}[\sin x]=\cos x$⭐
cos$\frac{d}{dx}[\cos x]=-\sin x$⭐ 注意負號
tan$\frac{d}{dx}[\tan x]=\sec^2 x$⭐
cot$\frac{d}{dx}[\cot x]=-\csc^2 x$
sec$\frac{d}{dx}[\sec x]=\sec x\tan x$
csc$\frac{d}{dx}[\csc x]=-\csc x\cot x$
arcsin$\frac{d}{dx}[\arcsin x]=\frac{1}{\sqrt{1-x^2}}$⭐ $|x|<1$
arccos$\frac{d}{dx}[\arccos x]=-\frac{1}{\sqrt{1-x^2}}$$|x|<1$
arctan$\frac{d}{dx}[\arctan x]=\frac{1}{1+x^2}$⭐ 全實數定義
arccot$\frac{d}{dx}[\operatorname{arccot} x]=-\frac{1}{1+x^2}$
arcsec$\frac{d}{dx}[\operatorname{arcsec} x]=\frac{1}{|x|\sqrt{x^2-1}}$$|x|>1$
arccsc$\frac{d}{dx}[\operatorname{arccsc} x]=-\frac{1}{|x|\sqrt{x^2-1}}$$|x|>1$
