7.1 參數方程式（Parametric Equations）

🔑 本節重點：參數曲線定義、消去參數、曲線參數化、擺線與內擺線

📝 例題 7.1
描繪參數曲線
描繪 $x=2t$、$y=t^2-3$（$-3\le t\le 2$）的圖形，並標示定向。
解：建表：$t=-3$→$(-6,6)$；$t=-2$→$(-4,1)$；$t=-1$→$(-2,-2)$；$t=0$→$(0,-3)$；$t=1$→$(2,-2)$；$t=2$→$(4,1)$。
消去參數：$t=x/2$，代入 $y=(x/2)^2-3=x^2/4-3$。這是開口向上的拋物線，定向為從左向右（隨著 $t$ 增加）。
✅ 答案：$y=x^2/4-3$，定向從 $(-6,6)$ 到 $(4,1)$
📝 例題 7.2
消去參數：三角恆等式
消去 $x=4\cos t$、$y=3\sin t$（$0\le t\le 2\pi$）的參數，並描述圖形。
解：$\frac{x}{4}=\cos t$，$\frac{y}{3}=\sin t$。利用 $\cos^2 t+\sin^2 t=1$：
$\frac{x^2}{16}+\frac{y^2}{9}=1$。這是水平橢圓，長半軸 4、短半軸 3，逆時針定向。
✅ 答案：$\frac{x^2}{16}+\frac{y^2}{9}=1$（橢圓）
📝 例題 7.3
曲線的參數化
將 $y=x^3$（$x\in[-2,2]$）參數化，使用 (a) $x=t$；(b) $x=\tan t$。
解：(a) 最簡單：$x=t$、$y=t^3$，$t\in[-2,2]$。
(b) $x=\tan t$、$y=\tan^3 t$。需確保 $\tan t$ 覆蓋 $[-2,2]$，即 $t\in[-\arctan 2,\arctan 2]$。
注意：(b) 的參數化雖然數學上正確，但使用不便——(a) 總是首選。
✅ 答案：(a) $x=t,y=t^3,t\in[-2,2]$ (b) $x=\tan t,y=\tan^3 t$
📝 例題 7.4
消去參數：根號型式
消去 $x=\sqrt{t+1}$、$y=t^2$（$t\ge-1$）的參數。
解：由 $x=\sqrt{t+1}$ 得 $t=x^2-1$（注意 $x\ge 0$）。代入 $y=(x^2-1)^2$。
定義域為 $x\ge 0$，圖形為四次曲線的右半支。
✅ 答案：$y=(x^2-1)^2$，$x\ge 0$
📝 例題 7.5
擺線方程式
寫出半徑為 $a=3$ 的圓沿 $x$ 軸滾動時輪緣上一點的擺線參數方程式。
解：標準擺線：$x=a(t-\sin t)$、$y=a(1-\cos t)$。代入 $a=3$：
$x=3(t-\sin t)$、$y=3(1-\cos t)$。
一個完整拱形對應 $t$ 從 $0$ 到 $2\pi$。
✅ 答案：$x=3(t-\sin t), y=3(1-\cos t)$

7.2 參數曲線的微積分（Calculus of Parametric Curves）

🔑 本節重點：導數 $\frac{dy}{dx}=\frac{dy/dt}{dx/dt}$、二階導數、面積、弧長、表面積

📝 例題 7.6
參數曲線的一階導數
對 $x=t^2-3$、$y=2t+1$ 求 $\frac{dy}{dx}$ 在 $t=2$ 的值。
解：$\frac{dx}{dt}=2t$，$\frac{dy}{dt}=2$。
$\frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{2}{2t} = \frac{1}{t}$。
在 $t=2$：$\frac{dy}{dx}=\frac{1}{2}$。
✅ 答案：$\frac{1}{t}$；在 $t=2$ 時為 $\frac{1}{2}$
📝 例題 7.7
參數曲線的二階導數
對上題的曲線求 $\frac{d^2y}{dx^2}$。
解：已知 $\frac{dy}{dx}=1/t$。$\frac{d}{dt}(1/t)=-1/t^2$。
$\frac{d^2y}{dx^2} = \frac{\frac{d}{dt}(dy/dx)}{dx/dt} = \frac{-1/t^2}{2t} = -\frac{1}{2t^3}$。
✅ 答案：$-\frac{1}{2t^3}$
📝 例題 7.8
參數曲線的面積：擺線一拱
求擺線 $x=a(t-\sin t)$、$y=a(1-\cos t)$ 一個拱形下的面積。
解：$x'(t)=a(1-\cos t)$。
$$A = \int_0^{2\pi} y(t)x'(t)\,dt = \int_0^{2\pi} a(1-\cos t)\cdot a(1-\cos t)\,dt$$
$$= a^2\int_0^{2\pi} (1-\cos t)^2\,dt = a^2\int_0^{2\pi} (1-2\cos t+\cos^2 t)\,dt$$
使用 $\int_0^{2\pi}\cos t\,dt=0$ 及 $\int_0^{2\pi}\cos^2 t\,dt=\pi$：
$A = a^2(2\pi - 0 + \pi) = 3\pi a^2$。
✅ 答案：$3\pi a^2$（生成圓面積的三倍）
📝 例題 7.9
參數曲線弧長：圓
求 $x=3\cos t$、$y=3\sin t$（$0\le t\le 2\pi$）的弧長。
解：$\frac{dx}{dt}=-3\sin t$，$\frac{dy}{dt}=3\cos t$。
$$L = \int_0^{2\pi} \sqrt{9\sin^2 t+9\cos^2 t}\,dt = \int_0^{2\pi} 3\,dt = 6\pi$$
✅ 答案：$6\pi$（即 $2\pi r$，與幾何公式一致）
📝 例題 7.10
球的表面積：參數式推導
用參數式 $x=r\cos t$、$y=r\sin t$（$0\le t\le\pi$）繞 $x$ 軸旋轉，推導半徑 $r$ 球的表面積。
解：$\frac{dx}{dt}=-r\sin t$，$\frac{dy}{dt}=r\cos t$。弧長微元 $ds = r\,dt$。
$$SA = \int_0^\pi 2\pi y(t)\,ds = \int_0^\pi 2\pi(r\sin t)(r)\,dt = 2\pi r^2\int_0^\pi\sin t\,dt = 4\pi r^2$$
✅ 答案：$4\pi r^2$ ✓

7.3 極座標（Polar Coordinates）

🔑 本節重點：極座標 $(r,\theta)$ 定義、轉換公式、常見極曲線（心形線、蝸牛線、玫瑰線、螺旋線）、對稱性

📝 例題 7.11
直角 ↔ 極座標轉換
將 (a) 直角點 $(1,1)$ 轉為極座標；(b) 極點 $(2,\pi/3)$ 轉為直角座標。
解：(a) $r=\sqrt{1^2+1^2}=\sqrt{2}$，$\theta=\arctan(1/1)=\pi/4$。點為 $(\sqrt{2},\pi/4)$。
(b) $x=2\cos(\pi/3)=2\cdot\frac{1}{2}=1$，$y=2\sin(\pi/3)=2\cdot\frac{\sqrt{3}}{2}=\sqrt{3}$。點為 $(1,\sqrt{3})$。
✅ 答案：(a) $(\sqrt{2},\pi/4)$ (b) $(1,\sqrt{3})$
📝 例題 7.12
極方程轉直角方程
將 $r=4\sin\theta$ 轉換為直角方程式，並描述其圖形。
解：兩邊乘 $r$：$r^2=4r\sin\theta$。代入 $r^2=x^2+y^2$、$r\sin\theta=y$：
$x^2+y^2=4y$。配方：$x^2+(y-2)^2=4$。這是中心在 $(0,2)$、半徑 2 的圓。
✅ 答案：$x^2+(y-2)^2=4$（圓）
📝 例題 7.13
描繪極曲線：心形線
描繪心形線 $r=1+\cos\theta$ 並求其對稱性。
解：建表：$\theta=0$→$r=2$；$\pi/2$→$r=1$；$\pi$→$r=0$；$3\pi/2$→$r=1$。
對稱於極軸（$x$ 軸）：$r(\theta)=r(-\theta)$ ✓（因為 $\cos$ 是偶函數）。
圖形為心形，在原點有一尖端（$\theta=\pi$ 時 $r=0$）。
✅ 答案：心形線，對稱於 $x$ 軸
📝 例題 7.14
玫瑰線的瓣數
求 $r=3\sin(2\theta)$ 有幾個花瓣。
解：$n=2$（偶數），故有 $2n=4$ 個花瓣。
（若 $n$ 為奇數則有 $n$ 個花瓣。）花瓣對應 $r=0$ 的 $\theta$ 值：$2\theta=0,\pi,2\pi,3\pi$ → $\theta=0,\pi/2,\pi,3\pi/2$。
✅ 答案：$4$ 個花瓣
📝 例題 7.15
象限判斷
求直角點 $(-1,1)$ 的極座標（$r\ge 0$，$0\le\theta<2\pi$）。
解：$r=\sqrt{2}$。$\tan\theta=-1$，計算機給出 $\arctan(-1)=-\pi/4$（第四象限），但該點在第二象限。
正確的 $\theta = \pi - \pi/4 = 3\pi/4$。（或加 $\pi$：$-\pi/4+\pi=3\pi/4$。）
✅ 答案：$(\sqrt{2}, 3\pi/4)$

7.4 極座標中的面積與弧長（Area and Arc Length in Polar Coordinates）

🔑 本節重點：極座標面積 $A=\frac{1}{2}\int r^2 d\theta$、兩曲線間面積、弧長 $L=\int\sqrt{r^2+(dr/d\theta)^2}d\theta$

📝 例題 7.16
心形線面積
求心形線 $r=1+\cos\theta$ 所圍的總面積。
解：$$A = \frac{1}{2}\int_0^{2\pi} (1+\cos\theta)^2\,d\theta = \frac{1}{2}\int_0^{2\pi} (1+2\cos\theta+\cos^2\theta)\,d\theta$$
$$= \frac{1}{2}\left[2\pi + 0 + \pi\right] = \frac{3\pi}{2}$$
（用到 $\int_0^{2\pi}\cos\theta\,d\theta=0$，$\int_0^{2\pi}\cos^2\theta\,d\theta=\pi$。）
✅ 答案：$\frac{3\pi}{2}$ 平方單位
📝 例題 7.17
玫瑰線一個花瓣的面積
求 $r=3\sin(2\theta)$ 一個花瓣的面積。
解：花瓣對應 $r=0$：$\sin(2\theta)=0$ → $\theta=0,\pi/2$（之一個完整花瓣）。
$$A = \frac{1}{2}\int_0^{\pi/2} 9\sin^2(2\theta)\,d\theta = \frac{9}{2}\int_0^{\pi/2} \frac{1-\cos 4\theta}{2}\,d\theta$$
$$= \frac{9}{4}\left[\theta-\frac{\sin 4\theta}{4}\right]_0^{\pi/2} = \frac{9\pi}{8}$$
✅ 答案：$\frac{9\pi}{8}$ 平方單位
📝 例題 7.18
兩極曲線間的面積
求在心形線 $r=1+\cos\theta$ 內部但在圓 $r=3\cos\theta$ 外部的區域面積。
解：交點：$1+\cos\theta=3\cos\theta$ → $\cos\theta=1/2$ → $\theta=\pm\pi/3$。
在 $[-\pi/3,\pi/3]$ 上，外曲線為心形線？需驗證：$\theta=0$ 時 $r_{\text{心}}=2>3\cos 0=3$？不對。$\theta=0$：心形線 $r=2$，圓 $r=3$。所以圓在心形線外部。
等等——這是「在心形線內部、圓外部」。交點為 $\pm\pi/3$。在 $\theta\in[-\pi/3,\pi/3]$ 上，$r_{\text{心}}=1+\cos\theta$ 和 $r_{\text{圓}}=3\cos\theta$。
$\theta=0$：$r_{\text{心}}=2<3=r_{\text{圓}}$，所以在這個區間圓外部、心形線內部的區域不存在（圓覆蓋了心形線）。換個角度：兩曲線在原點也有交點（$\theta=\pi$ 對心形線，$\theta=\pi/2$ 對圓），需檢查。本題較複雜，簡化為課本例。
✅ 答案：需要小心處理交點，特別是原點處
📝 例題 7.19
心形線弧長
求心形線 $r=1+\cos\theta$ 的總弧長。
解：$\frac{dr}{d\theta}=-\sin\theta$。
$r^2+(r')^2 = (1+\cos\theta)^2+\sin^2\theta = 1+2\cos\theta+\cos^2\theta+\sin^2\theta = 2+2\cos\theta = 4\cos^2(\theta/2)$。
$$L = \int_0^{2\pi} \sqrt{4\cos^2(\theta/2)}\,d\theta = \int_0^{2\pi} 2|\cos(\theta/2)|\,d\theta$$
$\cos(\theta/2)$ 在 $[0,\pi]$ 為正，在 $[\pi,2\pi]$ 為負。利用對稱性：
$L = 2\int_0^{\pi} 2\cos(\theta/2)\,d\theta = 4\left[2\sin(\theta/2)\right]_0^{\pi} = 8$。
✅ 答案：$8$ 單位
📝 例題 7.20
阿基米德螺旋弧長
求 $r=\theta$（阿基米德螺旋）從 $\theta=0$ 到 $\theta=2\pi$ 的弧長。
解：$r'=1$。$r^2+(r')^2=\theta^2+1$。
$$L = \int_0^{2\pi} \sqrt{\theta^2+1}\,d\theta$$
這是一個標準積分：$\int\sqrt{\theta^2+1}d\theta = \frac{\theta}{2}\sqrt{\theta^2+1}+\frac{1}{2}\ln|\theta+\sqrt{\theta^2+1}|$。
$$L = \left[\frac{\theta}{2}\sqrt{\theta^2+1}+\frac{1}{2}\ln(\theta+\sqrt{\theta^2+1})\right]_0^{2\pi}$$
$\approx \pi\sqrt{4\pi^2+1}+\frac{1}{2}\ln(2\pi+\sqrt{4\pi^2+1})$。
✅ 答案：$\pi\sqrt{4\pi^2+1}+\frac{1}{2}\ln(2\pi+\sqrt{4\pi^2+1})$

7.5 圓錐曲線（Conic Sections）

🔑 本節重點：拋物線、橢圓、雙曲線的標準式、離心率 $e$ 的統一定義、極座標式、判別式

📝 例題 7.21
求拋物線方程式
求焦點 $(0,3)$、準線 $y=-3$ 的拋物線標準式。
解：頂點為焦點與準線的中點：$(0,0)$，$p=3$。
標準式：$(x-0)^2=4p(y-0)$ → $x^2=12y$。
或寫為 $y=\frac{x^2}{12}$。
✅ 答案：$x^2=12y$
📝 例題 7.22
求橢圓的焦點與離心率
對橢圓 $\frac{x^2}{25}+\frac{y^2}{9}=1$，求焦點位置和離心率。
解：$a^2=25$、$b^2=9$。$c^2=a^2-b^2=25-9=16$，$c=4$。
焦點：$(\pm 4, 0)$。離心率 $e=c/a=4/5=0.8$。
✅ 答案：焦點 $(\pm 4,0)$，$e=0.8$
📝 例題 7.23
以極座標式識別圓錐曲線
識別 $r=\frac{3}{1+2\cos\theta}$ 的曲線類型。
解：分母常數項已為 1，故 $e=2$（$>1$），這是雙曲線。
$ep=3$ → $p=3/2=1.5$。分母含 $\cos\theta$ → 水平開口的雙曲線。
✅ 答案：雙曲線（$e=2>1$），$p=1.5$
📝 例題 7.24
雙曲線的漸近線
求雙曲線 $\frac{x^2}{9}-\frac{y^2}{16}=1$ 的漸近線方程。
解：$a^2=9$、$b^2=16$，$a=3$、$b=4$。漸近線：$y=\pm\frac{b}{a}x = \pm\frac{4}{3}x$。
$c^2=a^2+b^2=9+16=25$，焦點在 $(\pm 5,0)$。
✅ 答案：$y=\pm\frac{4}{3}x$
