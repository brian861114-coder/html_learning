1.1 函數回顧（Review of Functions）

例題 1.1 — 求函數值（Evaluating Functions）
題目：給定函數 $f(x) = x^2 + 3x - 4$，求 (a) $f(0)$、(b) $f(2)$、(c) $f(a)$。
解：將給定的 $x$ 值代入 $f(x) = x^2 + 3x - 4$。
(a) $f(0) = 0^2 + 3(0) - 4 = -4$
(b) $f(2) = 2^2 + 3(2) - 4 = 4 + 6 - 4 = 6$
(c) $f(a) = a^2 + 3a - 4$
答：$f(0)=-4,\ f(2)=6,\ f(a)=a^2+3a-4$
例題 1.2 — 求定義域與值域（Finding Domain and Range）
題目：對以下各函數，求 (i) 定義域（domain）和 (ii) 值域（range）。
(a) $f(x) = (x-4)^2 + 5$　(b) $f(x) = \sqrt{3x+2}-1$　(c) $f(x) = \dfrac{3}{x-2}$
解：
(a) $f(x) = (x-4)^2 + 5$
　(i) 定義域：$(x-4)^2$ 對所有實數 $x$ 都有定義，故定義域為 $(-\infty, \infty)$。
　(ii) 值域：$(x-4)^2 \ge 0$，所以 $(x-4)^2 + 5 \ge 5$。當 $x=4$ 時取得最小值 $5$。故值域為 $[5, \infty)$。

(b) $f(x) = \sqrt{3x+2} - 1$
　(i) 定義域：根號內必須非負：$3x+2 \ge 0 \implies x \ge -\frac{2}{3}$。定義域為 $[-\frac{2}{3}, \infty)$。
　(ii) 值域：$\sqrt{3x+2} \ge 0$，所以 $\sqrt{3x+2} - 1 \ge -1$。值域為 $[-1, \infty)$。

(c) $f(x) = \dfrac{3}{x-2}$
　(i) 定義域：分母不為零：$x-2 \neq 0 \implies x \neq 2$。定義域為 $(-\infty, 2) \cup (2, \infty)$。
　(ii) 值域：$f(x) = \frac{3}{x-2}$ 可以取任何非零實數值（因為 $y = \frac{3}{x-2} \iff x = 2 + \frac{3}{y}$，對任意 $y \neq 0$ 都有解）。值域為 $(-\infty, 0) \cup (0, \infty)$。
例題 1.3 — 求函數的零點與 $x$ 截距（Finding Zeros and $x$-Intercepts）
題目：求函數 $f(x) = x^3 - 4x$ 的零點。
解：零點是滿足 $f(x) = 0$ 的 $x$ 值。
$x^3 - 4x = 0$
$x(x^2 - 4) = 0$
$x(x-2)(x+2) = 0$
由零因子定理：$x = 0$ 或 $x = 2$ 或 $x = -2$。
答：零點為 $x = -2, 0, 2$。這些也是 $x$ 截距。
例題 1.4 — 利用零點與 $x$ 截距繪圖（Using Zeros and $x$-Intercepts to Sketch a Graph）
題目：利用零點繪出 $f(x) = x^2 - 2x - 3$ 的圖形草圖。
解：
① 求零點：$x^2 - 2x - 3 = 0$
　$(x-3)(x+1) = 0 \implies x = -1, 3$
② $y$ 截距：$f(0) = -3$
③ 頂點（vertex）：$x = -\frac{b}{2a} = -\frac{-2}{2(1)} = 1$
　$f(1) = 1^2 - 2(1) - 3 = 1 - 2 - 3 = -4$
④ 此為開口向上的拋物線（$a=1>0$），頂點 $(1, -4)$，穿過 $(-1, 0)$ 和 $(3, 0)$，$y$ 截距 $(0,-3)$。
答：圖形為開口向上的拋物線，頂點 $(1,-4)$，$x$ 截距 $-1, 3$。
例題 1.5 — 自由落體高度（Finding the Height of a Free-Falling Object）
題目：一物體從高處落下，其高度函數為 $h(t) = -16t^2 + 144$（$t$：秒，$h$：英尺）。求 (a) $t=0$ 時的高度；(b) $t=1$ 時的高度；(c) 物體何時落地？
解：
(a) $h(0) = -16(0)^2 + 144 = 144$ 英尺
(b) $h(1) = -16(1)^2 + 144 = 128$ 英尺
(c) 落地時 $h(t) = 0$：
　$-16t^2 + 144 = 0$
　$16t^2 = 144$
　$t^2 = 9$
　$t = 3$（取正值）
答：(a) 144 ft；(b) 128 ft；(c) $t = 3$ 秒後落地。
例題 1.6 — 函數的算術組合（Combining Functions Using Mathematical Operations）
題目：給定 $f(x) = 2x^2 - 3$ 和 $g(x) = x + 1$，求 (a) $(f+g)(x)$；(b) $(f-g)(x)$；(c) $(fg)(x)$；(d) $\left(\frac{f}{g}\right)(x)$ 及其定義域。
解：
(a) $(f+g)(x) = f(x) + g(x) = (2x^2 - 3) + (x + 1) = 2x^2 + x - 2$
(b) $(f-g)(x) = f(x) - g(x) = (2x^2 - 3) - (x + 1) = 2x^2 - x - 4$
(c) $(fg)(x) = f(x) \cdot g(x) = (2x^2 - 3)(x + 1) = 2x^3 + 2x^2 - 3x - 3$
(d) $\left(\frac{f}{g}\right)(x) = \frac{f(x)}{g(x)} = \frac{2x^2 - 3}{x + 1}$，定義域為 $x \neq -1$，即 $(-\infty, -1) \cup (-1, \infty)$。
例題 1.7 — 公式定義的合成函數（Compositions of Functions Defined by Formulas）
題目：給定 $f(x) = x^2$ 和 $g(x) = 2x + 1$，求 (a) $(f \circ g)(x)$；(b) $(g \circ f)(x)$；(c) $(f \circ f)(x)$。
解：
(a) $(f \circ g)(x) = f(g(x)) = f(2x+1) = (2x+1)^2 = 4x^2 + 4x + 1$
(b) $(g \circ f)(x) = g(f(x)) = g(x^2) = 2x^2 + 1$
(c) $(f \circ f)(x) = f(f(x)) = f(x^2) = (x^2)^2 = x^4$
注意：一般來說 $f \circ g \neq g \circ f$，合成函數不滿足交換律。
例題 1.8 — 表格定義的合成函數（Composition of Functions Defined by Tables）
題目：下表給出 $f$ 和 $g$ 的取值。求 (a) $(f \circ g)(1)$；(b) $(g \circ f)(3)$。
表 1：1.1 函數回顧（Review of Functions）
$x$1234
$f(x)$3142
$g(x)$2413
解：
(a) $(f \circ g)(1) = f(g(1))$。查表得 $g(1) = 2$，再查 $f(2) = 1$。故 $(f \circ g)(1) = 1$。
(b) $(g \circ f)(3) = g(f(3))$。查表得 $f(3) = 4$，再查 $g(4) = 3$。故 $(g \circ f)(3) = 3$。
答：(a) $1$；(b) $3$。
例題 1.9 — 合成函數的應用（Application Involving a Composite Function）
題目：某商品的價格 $p$（美元）為需求 $x$（件）的函數：$p(x) = 50 - 0.1x$。收入 $R$ 為 $R = x \cdot p$。寫出 $R$ 作為 $x$ 的函數，並求賣出 $200$ 件時的收入。
解：
$R(x) = x \cdot p(x) = x(50 - 0.1x) = 50x - 0.1x^2$
當 $x = 200$：$R(200) = 50(200) - 0.1(200^2) = 10000 - 0.1(40000) = 10000 - 4000 = 6000$。
答：$R(x) = 50x - 0.1x^2$；賣出 $200$ 件時收入為 $\$6000$。
例題 1.10 — 偶函數與奇函數（Even and Odd Functions）
題目：判斷以下函數是偶函數、奇函數、或兩者皆非：(a) $f(x) = x^4 - 3x^2$；(b) $g(x) = x^3 - 2x$；(c) $h(x) = x^3 + 1$。
解：
(a) $f(-x) = (-x)^4 - 3(-x)^2 = x^4 - 3x^2 = f(x)$ → 偶函數（對稱於 $y$ 軸）
(b) $g(-x) = (-x)^3 - 2(-x) = -x^3 + 2x = -(x^3 - 2x) = -g(x)$ → 奇函數（對稱於原點）
(c) $h(-x) = (-x)^3 + 1 = -x^3 + 1$。這既不等于 $h(x)=x^3+1$，也不等于 $-h(x)=-x^3-1$。故 兩者皆非。
例題 1.11 — 絕對值函數（Working with the Absolute Value Function）
題目：畫出 $f(x) = |x-2| + 1$ 的圖形，並寫出其定義域與值域。
解：
$f(x) = |x-2| + 1$ 是基本絕對值函數 $y=|x|$ 向右平移 $2$ 單位、向上平移 $1$ 單位。
定義域：$(-\infty, \infty)$（所有實數）
值域：$|x-2| \ge 0$，所以 $|x-2|+1 \ge 1$。值域為 $[1, \infty)$。
頂點在 $(2, 1)$，呈 V 字形。

1.2 函數的基本類型（Basic Classes of Functions）

例題 1.12 — 求斜率與直線方程式（Finding the Slope and Equations of Lines）
題目：求通過點 $(2, 5)$ 和 $(6, 13)$ 的直線之斜率，並寫出點斜式和斜截式。
解：
斜率 $m = \dfrac{y_2 - y_1}{x_2 - x_1} = \dfrac{13 - 5}{6 - 2} = \dfrac{8}{4} = 2$
點斜式（使用點 $(2,5)$）：$y - 5 = 2(x - 2)$
斜截式：$y = 2x + 1$
答：$m = 2$；點斜式 $y - 5 = 2(x - 2)$；斜截式 $y = 2x + 1$。
例題 1.13 — 線性距離函數（A Linear Distance Function）
題目：一輛汽車以恒定速率 $60$ mph 行駛。若 $t=0$ 時已行駛 $20$ 英里，寫出距離 $d$（英里）作為時間 $t$（小時）的線性函數，並求 $t=2.5$ 時的距離。
解：
$d(t) = 60t + 20$（每小時增加 $60$ 英里，初始距離 $20$ 英里）
$d(2.5) = 60(2.5) + 20 = 150 + 20 = 170$ 英里
答：$d(t) = 60t + 20$；$t=2.5$ 時距離為 $170$ 英里。
例題 1.14 — 多項式函數繪圖（Graphing Polynomial Functions）
題目：畫出 $f(x) = (x-1)(x+2)(x-3)$ 的圖形草圖，標示 $x$ 截距和端點行為（end behavior）。
解：
① $x$ 截距：$f(x)=0 \implies x=1, -2, 3$
② $y$ 截距：$f(0)=(-1)(2)(-3)=6$
③ 此為三次多項式，首項係數為正（展開後 $x^3$ 項係數為 $+1$）：當 $x \to -\infty$，$f(x) \to -\infty$；當 $x \to \infty$，$f(x) \to \infty$。
④ 在各零點之間判斷正負：
　$x < -2$：三個因子皆負 → 乘積為負
　$-2 < x < 1$：兩個因子負、一個正 → 乘積為正
　$1 < x < 3$：一個因子負、兩個正 → 乘積為負
　$x > 3$：三個因子皆正 → 乘積為正
圖形從左下進入，依次穿過 $(-2,0)$、$(1,0)$、$(3,0)$，呈 S 形曲線。
例題 1.15 — 最大化營收（Maximizing Revenue）
題目：一公司的營收函數為 $R(x) = -0.5x^2 + 100x$（$x$：售出數量）。求最大營收及其對應的 $x$ 值。
解：
$R(x) = -0.5x^2 + 100x$ 為開口向下的二次函數。最大值發生在頂點：
$x = -\dfrac{b}{2a} = -\dfrac{100}{2(-0.5)} = -\dfrac{100}{-1} = 100$
最大營收：$R(100) = -0.5(100^2) + 100(100) = -5000 + 10000 = 5000$
答：售出 $100$ 件時達最大營收 $\$5000$。
例題 1.16 — 代數函數的定義域與值域（Finding Domain and Range for Algebraic Functions）
題目：求 $f(x) = \sqrt{9 - x^2}$ 的定義域與值域。
解：
定義域：根號內必須非負：$9 - x^2 \ge 0 \implies x^2 \le 9 \implies -3 \le x \le 3$。定義域為 $[-3, 3]$。
值域：當 $x = 0$ 時 $f(0) = \sqrt{9} = 3$（最大值）；當 $x = \pm 3$ 時 $f(\pm 3) = 0$（最小值）。值域為 $[0, 3]$。
註：此函數圖形為上半圓（半徑 $3$）。
例題 1.17 — 求代數函數的定義域（Finding Domains for Algebraic Functions）
題目：求以下各函數的定義域：
(a) $f(x) = \dfrac{1}{\sqrt{x-3}}$　(b) $g(x) = \dfrac{x+1}{x^2-4}$
解：
(a) 根號內 $x-3 \ge 0 \implies x \ge 3$；分母 $\sqrt{x-3} \neq 0 \implies x \neq 3$。綜合得定義域 $(3, \infty)$。
(b) 分母 $x^2 - 4 \neq 0 \implies x \neq \pm 2$。定義域為 $(-\infty, -2) \cup (-2, 2) \cup (2, \infty)$。
例題 1.18 — 代數函數與超越函數分類（Classifying Algebraic and Transcendental Functions）
題目：將以下函數分類為代數函數或超越函數：
(a) $f(x) = \dfrac{x^3+1}{x-2}$　(b) $g(x) = 2^x$　(c) $h(x) = \sin(x^2)$
解：
(a) $f(x) = \frac{x^3+1}{x-2}$ → 代數函數（有理函數，為多項式之商）
(b) $g(x) = 2^x$ → 超越函數（指數函數，非代數運算可表達）
(c) $h(x) = \sin(x^2)$ → 超越函數（三角函數為超越函數）
例題 1.19 — 分段定義函數繪圖（Graphing a Piecewise-Defined Function）
題目：畫出分段定義函數的圖形： $$f(x) = \begin{cases} x+2 & \text{if } x < 1 \\ 4 - x^2 & \text{if } x \ge 1 \end{cases}$$
解：
當 $x < 1$：$y = x+2$ 是一條斜率為 $1$ 的直線。在 $x=1$ 處（不含），$y = 1+2 = 3$（空心點）。
當 $x \ge 1$：$y = 4 - x^2$ 是開口向下的拋物線。在 $x=1$ 處，$y = 4 - 1 = 3$（實心點）。
在 $x=1$ 處，兩段的值相等，故函數在 $x=1$ 連續。
例題 1.20 — 停車費分段函數（Parking Fees Described by a Piecewise-Defined Function）
題目：某停車場收費規則：第一小時（或不足一小時）收 $\$3$，之後每增加一小時（或不足一小時）加收 $\$2$，每日最高收費 $\$15$。寫出停車 $t$ 小時的費用函數 $C(t)$（$0 < t \le 12$）。
解：
費用函數 $C(t)$ 為階梯函數（step function）：
　$0 < t \le 1$：$C(t) = 3$
　$1 < t \le 2$：$C(t) = 5$
　$2 < t \le 3$：$C(t) = 7$
　$3 < t \le 4$：$C(t) = 9$
　$4 < t \le 5$：$C(t) = 11$
　$5 < t \le 6$：$C(t) = 13$
　$6 < t \le 12$：$C(t) = 15$（達每日上限）
這是取上整函數（ceiling function）的應用。
例題 1.21 — 函數變換（Transforming a Function）
題目：利用基本函數 $y = \sqrt{x}$ 的變換，畫出 $f(x) = -2\sqrt{x-1} + 3$ 的圖形。
解：從 $y = \sqrt{x}$ 出發，依序變換：
① $x \to x-1$：向右平移 $1$ 單位 → $y = \sqrt{x-1}$
② 乘以 $2$：垂直拉伸 $2$ 倍 → $y = 2\sqrt{x-1}$
③ 加上負號：對 $x$ 軸反射 → $y = -2\sqrt{x-1}$
④ $+3$：向上平移 $3$ 單位 → $y = -2\sqrt{x-1} + 3$
最終圖形從 $(1, 3)$ 開始向右下方延伸（開口向左的平方根曲線翻轉）。

1.3 三角函數（Trigonometric Functions）

例題 1.22 — 弧度與度數互換（Converting between Radians and Degrees）
題目：(a) 將 $210^\circ$ 轉換為弧度。(b) 將 $\frac{5\pi}{6}$ 弧度轉換為度數。
解：
使用轉換關係：$\pi\text{ rad} = 180^\circ$
(a) $210^\circ \times \dfrac{\pi}{180^\circ} = \dfrac{210\pi}{180} = \dfrac{7\pi}{6}$ 弧度
(b) $\dfrac{5\pi}{6} \times \dfrac{180^\circ}{\pi} = \dfrac{5 \times 180^\circ}{6} = 150^\circ$
答：(a) $\frac{7\pi}{6}$ rad；(b) $150^\circ$。
例題 1.23 — 求三角函數值（Evaluating Trigonometric Functions）
題目：求以下各值：(a) $\sin\frac{5\pi}{3}$；(b) $\cos\frac{3\pi}{4}$；(c) $\tan\frac{\pi}{6}$；(d) $\sec\frac{\pi}{3}$。
解：
(a) $\frac{5\pi}{3} = 300^\circ$（第四象限），$\sin\frac{5\pi}{3} = -\frac{\sqrt{3}}{2}$
(b) $\frac{3\pi}{4} = 135^\circ$（第二象限），$\cos\frac{3\pi}{4} = -\frac{\sqrt{2}}{2}$
(c) $\tan\frac{\pi}{6} = \frac{\sin(\pi/6)}{\cos(\pi/6)} = \frac{1/2}{\sqrt{3}/2} = \frac{1}{\sqrt{3}} = \frac{\sqrt{3}}{3}$
(d) $\sec\frac{\pi}{3} = \frac{1}{\cos(\pi/3)} = \frac{1}{1/2} = 2$
例題 1.24 — 建造木製坡道（Constructing a Wooden Ramp）
題目：欲建造一個長 $12$ 英尺的坡道，使其與地面的夾角為 $15^\circ$。求坡道頂端距離地面的高度。
解：
坡道長為斜邊，高度為對邊。$\sin 15^\circ = \dfrac{h}{12}$
$h = 12 \sin 15^\circ = 12 \times \dfrac{\sqrt{6} - \sqrt{2}}{4} = 3(\sqrt{6} - \sqrt{2}) \approx 12 \times 0.2588 \approx 3.11$ 英尺
答：高度約 $3.11$ 英尺。
例題 1.25 — 解三角方程式（Solving Trigonometric Equations）
題目：在 $[0, 2\pi)$ 區間內，解 $\sin x = \frac{1}{2}$。
解：
$\sin x = \frac{1}{2}$ 在 $[0, 2\pi)$ 內的解為：
$x = \frac{\pi}{6}$（第一象限）和 $x = \frac{5\pi}{6}$（第二象限）。
驗證：$\sin\frac{\pi}{6} = \frac{1}{2}$，$\sin\frac{5\pi}{6} = \sin(\pi - \frac{\pi}{6}) = \sin\frac{\pi}{6} = \frac{1}{2}$。
答：$x = \frac{\pi}{6}, \frac{5\pi}{6}$。
例題 1.26 — 證明三角恆等式（Proving a Trigonometric Identity）
題目：證明 $\dfrac{1 - \cos^2 x}{\sin x} = \sin x$。
解：
利用畢氏恆等式 $\sin^2 x + \cos^2 x = 1$，得 $1 - \cos^2 x = \sin^2 x$。
因此 $\dfrac{1 - \cos^2 x}{\sin x} = \dfrac{\sin^2 x}{\sin x} = \sin x$（當 $\sin x \neq 0$ 時成立）。
例題 1.27 — 畫變形正弦曲線（Sketching the Graph of a Transformed Sine Curve）
題目：畫出 $f(x) = 3\sin\!\big(2x - \frac{\pi}{3}\big) + 1$ 的圖形，並給出振幅、週期、相位偏移和垂直偏移。
解：重寫為標準形式 $f(x) = 3\sin\!\big(2(x - \frac{\pi}{6})\big) + 1$
振幅 $|A| = 3$
週期 $= \dfrac{2\pi}{|B|} = \dfrac{2\pi}{2} = \pi$
相位偏移 $= \dfrac{\pi}{6}$（向右）
垂直偏移 $D = 1$（向上 $1$ 單位）
圖形是正弦曲線，在 $y = -2$ 和 $y = 4$ 之間振盪，中線為 $y = 1$。

1.4 反函數（Inverse Functions）

例題 1.28 — 判斷函數是否一對一（Determining Whether a Function Is One-to-One）
題目：判斷以下函數在其自然定義域上是否為一對一（one-to-one）：
(a) $f(x) = x^3$　(b) $g(x) = x^2$　(c) $h(x) = 2x + 1$。
解：
(a) $f(x) = x^3$：若 $a^3 = b^3$，則 $a = b$。→ 一對一（通過水平線檢驗）
(b) $g(x) = x^2$：$g(2) = 4 = g(-2)$ 但 $2 \neq -2$。→ 不是一對一
(c) $h(x) = 2x+1$：若 $2a+1 = 2b+1$，則 $2a = 2b$，即 $a = b$。→ 一對一
例題 1.29 — 求反函數（Finding an Inverse Function）
題目：求 $f(x) = \dfrac{2x-1}{x+3}$ 的反函數 $f^{-1}(x)$。
解：
① 設 $y = \dfrac{2x-1}{x+3}$
② 交換 $x$ 和 $y$：$x = \dfrac{2y-1}{y+3}$
③ 解出 $y$：$x(y+3) = 2y-1$
　$xy + 3x = 2y - 1$
　$xy - 2y = -3x - 1$
　$y(x-2) = -(3x+1)$
　$y = \dfrac{3x+1}{2-x}$
答：$f^{-1}(x) = \dfrac{3x+1}{2-x},\ x \neq 2$。
例題 1.30 — 畫反函數圖形（Sketching Graphs of Inverse Functions）
題目：給定 $f(x) = \sqrt{x-1}$（$x \ge 1$），在同一個坐標系中畫出 $f$ 和 $f^{-1}$ 的圖形。
解：
① 求反函數：$y = \sqrt{x-1} \implies y^2 = x-1 \implies x = y^2+1$
　交換 $x$ 和 $y$：$y = x^2+1$（其中 $x \ge 0$，因為原函數的值域為 $[0,\infty)$）
　故 $f^{-1}(x) = x^2 + 1,\ x \ge 0$
② $f(x)$ 的圖形從 $(1,0)$ 開始向右上方延伸。
③ $f^{-1}(x)$ 的圖形從 $(0,1)$ 開始，為開口向上的拋物線右半支。
④ 兩圖形對稱於直線 $y = x$。
例題 1.31 — 限制定義域使函數有反函數（Restricting the Domain）
題目：求 $f(x) = (x-2)^2$ 在限制定義域為 $x \ge 2$ 後的反函數。
解：
在 $x \ge 2$ 上，$f(x) = (x-2)^2$ 是一對一的（嚴格遞增）。
設 $y = (x-2)^2$，由於 $x \ge 2$，$x-2 \ge 0$，故 $\sqrt{y} = x-2$。
$x = \sqrt{y} + 2$
交換 $x$ 和 $y$：$y = \sqrt{x} + 2$
答：$f^{-1}(x) = \sqrt{x} + 2,\ x \ge 0$。注意定義域 $x \ge 0$ 來自原函數的值域。
例題 1.32 — 計算反三角函數值（Evaluating Expressions Involving Inverse Trigonometric Functions）
題目：求 (a) $\sin^{-1}\!\big(\frac{\sqrt{3}}{2}\big)$；(b) $\cos^{-1}\!\big(-\frac{1}{2}\big)$；(c) $\tan^{-1}(1)$。
解：
(a) $\sin^{-1}\!\big(\frac{\sqrt{3}}{2}\big)$：在 $[-\frac{\pi}{2}, \frac{\pi}{2}]$ 內，$\sin\frac{\pi}{3} = \frac{\sqrt{3}}{2}$。故答案為 $\frac{\pi}{3}$。
(b) $\cos^{-1}\!\big(-\frac{1}{2}\big)$：在 $[0, \pi]$ 內，$\cos\frac{2\pi}{3} = -\frac{1}{2}$。故答案為 $\frac{2\pi}{3}$。
(c) $\tan^{-1}(1)$：在 $(-\frac{\pi}{2}, \frac{\pi}{2})$ 內，$\tan\frac{\pi}{4} = 1$。故答案為 $\frac{\pi}{4}$。
答：(a) $\frac{\pi}{3}$；(b) $\frac{2\pi}{3}$；(c) $\frac{\pi}{4}$。

1.5 指數與對數函數（Exponential and Logarithmic Functions）

例題 1.33 — 細菌增長（Bacterial Growth）
題目：一細菌培養基初始有 $500$ 個細菌，每小時數量翻倍。寫出 $t$ 小時後的細菌數量 $N(t)$，並求 $4$ 小時後的數量。
解：
$N(t) = 500 \cdot 2^t$
$N(4) = 500 \cdot 2^4 = 500 \cdot 16 = 8000$
答：$N(t) = 500 \cdot 2^t$；$4$ 小時後有 $8000$ 個細菌。
例題 1.34 — 使用指數律（Using the Laws of Exponents）
題目：化簡以下各式：(a) $\dfrac{2^{3x} \cdot 2^{x}}{2^{2x}}$；(b) $(e^{2x})^3$。
解：
(a) $\dfrac{2^{3x} \cdot 2^{x}}{2^{2x}} = \dfrac{2^{3x+x}}{2^{2x}} = \dfrac{2^{4x}}{2^{2x}} = 2^{4x-2x} = 2^{2x}$
(b) $(e^{2x})^3 = e^{2x \cdot 3} = e^{6x}$
答：(a) $2^{2x}$；(b) $e^{6x}$。
例題 1.35 — 複利計算（Compounding Interest）
題目：若將 $\$1000$ 以年利率 $5\%$ 存入銀行，按年複利計算。求 (a) $3$ 年後的金額；(b) 若改為連續複利（continuous compounding），$3$ 年後的金額。
解：
(a) 年複利公式：$A = P(1 + r)^t$
　$A = 1000(1 + 0.05)^3 = 1000(1.157625) = 1157.63$
(b) 連續複利公式：$A = Pe^{rt}$
　$A = 1000e^{0.05 \times 3} = 1000e^{0.15} \approx 1000 \times 1.16183 = 1161.83$
答：(a) $\$1157.63$；(b) $\$1161.83$。
例題 1.36 — 解指數方程式（Solving Equations Involving Exponential Functions）
題目：解 $5 \cdot 2^{3x} = 80$。
解：
方法一：$2^{3x} = \frac{80}{5} = 16 = 2^4$，因為 $2^t$ 是一對一的，所以 $3x = 4$，$x = \frac{4}{3}$。
方法二（取對數）：$2^{3x} = 16$
　$\ln(2^{3x}) = \ln 16$
　$3x \ln 2 = \ln 16 = \ln(2^4) = 4\ln 2$
　$3x = 4 \implies x = \frac{4}{3}$
答：$x = \frac{4}{3}$。
例題 1.37 — 解對數方程式（Solving Equations Involving Logarithmic Functions）
題目：解 $\log_2(x+1) + \log_2(x-1) = 3$。
解：
① 合併對數：$\log_2[(x+1)(x-1)] = 3$
② $\log_2(x^2-1) = 3$
③ 轉為指數形式：$x^2 - 1 = 2^3 = 8$
④ $x^2 = 9 \implies x = \pm 3$
⑤ 檢驗定義域：對數要求 $x+1 > 0$ 且 $x-1 > 0$，即 $x > 1$。因此 $x = -3$ 不合。
答：$x = 3$。
例題 1.38 — 換底公式（Changing Bases）
題目：利用換底公式計算 $\log_5 20$（以 $3$ 位小數表示）。
解：
換底公式：$\log_a b = \dfrac{\ln b}{\ln a}$
$\log_5 20 = \dfrac{\ln 20}{\ln 5} \approx \dfrac{2.9957}{1.6094} \approx 1.861$
或使用常用對數：$\log_5 20 = \dfrac{\log_{10} 20}{\log_{10} 5} \approx \dfrac{1.3010}{0.6990} \approx 1.861$
答：$\log_5 20 \approx 1.861$。
例題 1.39 — 芮氏地震規模（The Richter Scale for Earthquakes）
題目：芮氏規模（Richter scale）公式為 $M = \log_{10}\!\big(\frac{I}{I_0}\big)$，其中 $I$ 為地震強度，$I_0$ 為參考強度。若地震 A 的規模為 $7.3$，地震 B 的規模為 $9.0$，求兩者的強度比值。
解：
$M_A = \log_{10}\!\big(\frac{I_A}{I_0}\big) = 7.3 \implies \frac{I_A}{I_0} = 10^{7.3}$
$M_B = \log_{10}\!\big(\frac{I_B}{I_0}\big) = 9.0 \implies \frac{I_B}{I_0} = 10^{9.0}$
$\dfrac{I_B}{I_A} = \dfrac{10^{9.0}}{10^{7.3}} = 10^{9.0-7.3} = 10^{1.7} \approx 50.1$
答：地震 B 的強度約為地震 A 的 $50$ 倍。規模每增加 $1$ 單位，強度增加約 $10$ 倍；增加 $1.7$ 單位約為 $50$ 倍。
例題 1.40 — 求雙曲函數值（Evaluating Hyperbolic Functions）
題目：求 (a) $\sinh 0$；(b) $\cosh(\ln 2)$；(c) $\tanh 1$。
解：
(a) $\sinh 0 = \dfrac{e^0 - e^{-0}}{2} = \dfrac{1 - 1}{2} = 0$
(b) $\cosh(\ln 2) = \dfrac{e^{\ln 2} + e^{-\ln 2}}{2} = \dfrac{2 + \frac{1}{2}}{2} = \dfrac{2.5}{2} = 1.25 = \frac{5}{4}$
(c) $\tanh 1 = \dfrac{\sinh 1}{\cosh 1} = \dfrac{e^1 - e^{-1}}{e^1 + e^{-1}} \approx \dfrac{2.7183 - 0.3679}{2.7183 + 0.3679} \approx \dfrac{2.3504}{3.0862} \approx 0.762$
答：(a) $0$；(b) $\frac{5}{4}$；(c) $\approx 0.762$。
例題 1.41 — 求反雙曲函數值（Evaluating Inverse Hyperbolic Functions）
題目：求 (a) $\sinh^{-1}(0)$；(b) $\cosh^{-1}(1)$。
解：
(a) $\sinh^{-1}(0)$：求 $x$ 使得 $\sinh x = 0$。由例題 1.40 知 $\sinh 0 = 0$，且 $\sinh$ 為嚴格遞增的一對一函數。故 $\sinh^{-1}(0) = 0$。
　也可用公式：$\sinh^{-1}(x) = \ln\!\big(x + \sqrt{x^2+1}\big)$
　$\sinh^{-1}(0) = \ln(0 + \sqrt{0+1}) = \ln 1 = 0$
(b) $\cosh^{-1}(1)$：求 $x \ge 0$ 使得 $\cosh x = \frac{e^x + e^{-x}}{2} = 1$。
　$e^x + e^{-x} = 2 \implies e^{2x} - 2e^x + 1 = 0 \implies (e^x - 1)^2 = 0 \implies e^x = 1 \implies x = 0$
答：(a) $0$；(b) $0$。
