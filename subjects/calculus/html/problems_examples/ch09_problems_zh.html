9.1 序列 (Sequences)

🔑 本節重點：序列的定義、通項公式、序列的極限、收斂與發散、單調有界定理

📝 例題 9.1
求序列的通項公式
求序列 $\{-\frac{1}{2},\frac{2}{3},-\frac{3}{4},\frac{4}{5},\ldots\}$ 的通項公式。
解：分子：$1,2,3,4,\ldots = n$。分母：$2,3,4,5,\ldots = n+1$。交錯符號：奇數項為負、偶數項為正，因子為 $(-1)^n$。
通項：$a_n = (-1)^n \frac{n}{n+1}$。
✅ 答案：$a_n = (-1)^n \frac{n}{n+1}$
📝 例題 9.2
序列極限：a_n=(n+1)/n
求 $\lim_{n\to\infty} \frac{n+1}{n}$。
解：$\lim_{n\to\infty} \frac{n+1}{n} = \lim_{n\to\infty} \left(1 + \frac{1}{n}\right) = 1 + 0 = 1$。序列收斂至 1。
✅ 答案：$1$（收斂）
📝 例題 9.3
序列極限：a_n=(-1)ⁿ
求 $\lim_{n\to\infty} (-1)^n$。
解：序列為 $-1, 1, -1, 1, \ldots$，在 $-1$ 與 $1$ 之間振盪，無唯一極限值。故序列發散。
✅ 答案：發散（振盪）
📝 例題 9.4
序列極限：aₙ=ln(n)/n
求 $\lim_{n\to\infty} \frac{\ln n}{n}$。
解：使用羅必達法則（$\infty/\infty$ 型）：$\lim_{n\to\infty} \frac{\ln n}{n} = \lim_{n\to\infty} \frac{1/n}{1} = 0$。
✅ 答案：$0$（收斂）
📝 例題 9.5
幾何序列的極限
討論 $\{r^n\}$ 的收斂性，其中 $r$ 為常數。
解：若 $|r| < 1$：$\lim_{n\to\infty} r^n = 0$（收斂）。
若 $r = 1$：$\lim_{n\to\infty} 1^n = 1$（收斂）。
若 $|r| > 1$ 或 $r = -1$：發散。
✅ 答案：$|r|<1$ 或 $r=1$ 收斂，其餘發散
📝 例題 9.6
單調有界定理應用
證明 $a_{n+1} = \frac{a_n}{2} + \frac{1}{a_n}$，$a_1 = 2$ 收斂，並求極限。
解：可證明 $a_n > \sqrt{2}$（下界）且遞減。由單調有界定理，序列收斂。設 $\lim a_n = L$，則 $L = \frac{L}{2} + \frac{1}{L}$。
$\frac{L}{2} = \frac{1}{L}$，$L^2 = 2$，$L = \sqrt{2}$（取正根）。
✅ 答案：$\sqrt{2}$

9.2 無窮級數 (Infinite Series)

🔑 本節重點：部分和、幾何級數、伸縮級數、調和級數

📝 例題 9.7
幾何級數的和
求 $\sum_{n=0}^{\infty} \left(\frac{1}{2}\right)^n$。
解：幾何級數，$a=1$、$r=1/2$、$|r|<1$。和 $= \frac{a}{1-r} = \frac{1}{1-1/2} = 2$。
✅ 答案：$2$
📝 例題 9.8
伸縮級數的和
求 $\sum_{n=1}^{\infty} \frac{1}{n(n+1)}$。
解：使用部分分式：$\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$。
部分和 $S_N = (1-\frac{1}{2}) + (\frac{1}{2}-\frac{1}{3}) + \cdots + (\frac{1}{N}-\frac{1}{N+1}) = 1 - \frac{1}{N+1}$。
$\lim_{N\to\infty} S_N = 1$。
✅ 答案：$1$
📝 例題 9.9
調和級數發散
證明 $\sum_{n=1}^{\infty} \frac{1}{n}$ 發散。
解：使用積分檢驗：$\int_1^\infty \frac{dx}{x}$ 發散（$p=1$），故調和級數發散。
✅ 答案：發散
📝 例題 9.10
級數部分和的計算
求 $\sum_{n=1}^{\infty} \frac{3}{10^n}$ 的和。
解：幾何級數：$a = 3/10$、$r = 1/10$。和 $= \frac{3/10}{1-1/10} = \frac{3/10}{9/10} = \frac{1}{3}$。
✅ 答案：$\frac{1}{3}$

9.3 發散檢驗與積分檢驗 (Divergence and Integral Tests)

🔑 本節重點：第 n 項發散檢驗、積分檢驗、p-級數

📝 例題 9.11
第 n 項檢驗
判斷 $\sum_{n=1}^{\infty} \frac{n}{n+1}$ 的斂散性。
解：$\lim_{n\to\infty} a_n = \lim_{n\to\infty} \frac{n}{n+1} = 1 \neq 0$。由第 n 項發散檢驗，級數發散。
✅ 答案：發散
📝 例題 9.12
p-級數：p=2
判斷 $\sum_{n=1}^{\infty} \frac{1}{n^2}$ 的斂散性。
解：使用積分檢驗：$\int_1^\infty \frac{dx}{x^2} = 1$，收斂。故級數收斂。這是一個 $p$-級數，$p=2>1$。
✅ 答案：收斂（和 $= \frac{\pi^2}{6}$）
📝 例題 9.13
p-級數：p=1/2
判斷 $\sum_{n=1}^{\infty} \frac{1}{\sqrt{n}}$ 的斂散性。
解：$p = 1/2 \leq 1$。$\int_1^\infty \frac{dx}{\sqrt{x}}$ 發散，故級數發散。
✅ 答案：發散
📝 例題 9.14
積分檢驗：∑(ln n)/n²
判斷 $\sum_{n=2}^{\infty} \frac{\ln n}{n^2}$ 的斂散性。
解：使用積分檢驗：$\int_2^\infty \frac{\ln x}{x^2}\,dx$。分部積分：$u=\ln x$、$dv=x^{-2}dx$，$du=dx/x$、$v=-1/x$。
$\int \frac{\ln x}{x^2}\,dx = -\frac{\ln x}{x} - \frac{1}{x} + C$。從 2 到 $\infty$ 收斂。故級數收斂。
✅ 答案：收斂

9.4 比較檢驗 (Comparison Tests)

🔑 本節重點：直接比較檢驗、極限比較檢驗

📝 例題 9.15
直接比較檢驗
判斷 $\sum_{n=1}^{\infty} \frac{1}{n^2+1}$ 的斂散性。
解：對 $n \geq 1$：$n^2+1 > n^2$，故 $\frac{1}{n^2+1} < \frac{1}{n^2}$。$\sum \frac{1}{n^2}$ 收斂（$p=2>1$），由直接比較檢驗，原級數收斂。
✅ 答案：收斂
📝 例題 9.16
極限比較檢驗
判斷 $\sum_{n=1}^{\infty} \frac{1}{2^n-1}$ 的斂散性。
解：與 $\sum \frac{1}{2^n}$ 比較：$\lim_{n\to\infty} \frac{1/(2^n-1)}{1/2^n} = \lim_{n\to\infty} \frac{2^n}{2^n-1} = 1$（有限正數）。已知 $\sum \frac{1}{2^n}$ 收斂（幾何級數，$r=1/2$），故原級數收斂。
✅ 答案：收斂
📝 例題 9.17
比較檢驗：∑1/√(n³+1)
判斷 $\sum_{n=1}^{\infty} \frac{1}{\sqrt{n^3+1}}$ 的斂散性。
解：$\frac{1}{\sqrt{n^3+1}} < \frac{1}{\sqrt{n^3}} = \frac{1}{n^{3/2}}$。$\sum \frac{1}{n^{3/2}}$ 為 $p$-級數，$p=3/2>1$，收斂。由比較檢驗，原級數收斂。
✅ 答案：收斂

9.5 交錯級數 (Alternating Series)

🔑 本節重點：交錯級數檢驗、絕對收斂與條件收斂

📝 例題 9.18
交錯調和級數
判斷 $\sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n}$ 的斂散性。
解：$b_n = 1/n$ 遞減且 $\lim b_n = 0$。由交錯級數檢驗，級數收斂。但 $\sum |a_n| = \sum \frac{1}{n}$ 發散，故為條件收斂。
✅ 答案：條件收斂（和 $= \ln 2$）
📝 例題 9.19
交錯級數檢驗應用
判斷 $\sum_{n=1}^{\infty} \frac{(-1)^n}{n^2}$ 的斂散性。
解：交錯級數檢驗：$b_n = 1/n^2$ 遞減且 $\lim b_n = 0$，收斂。$\sum 1/n^2$ 也收斂，故為絕對收斂。
✅ 答案：絕對收斂
📝 例題 9.20
交錯級數誤差估計
用 $\sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n^4}$ 的前 5 項近似，估計誤差。
解：交錯級數的誤差 $\leq$ 首個未取項的絕對值：$|R_5| \leq b_6 = \frac{1}{6^4} = \frac{1}{1296} \approx 0.00077$。
✅ 答案：$|R_5| \leq \frac{1}{1296} \approx 0.00077$

9.6 比值檢驗與根值檢驗 (Ratio and Root Tests)

🔑 本節重點：比值檢驗、根值檢驗

📝 例題 9.21
比值檢驗：∑n!/nⁿ
判斷 $\sum_{n=1}^{\infty} \frac{n!}{n^n}$ 的斂散性。
解：$\rho = \lim_{n\to\infty} \left|\frac{a_{n+1}}{a_n}\right| = \lim_{n\to\infty} \frac{(n+1)!}{(n+1)^{n+1}} \cdot \frac{n^n}{n!} = \lim_{n\to\infty} \frac{n+1}{n+1} \cdot \left(\frac{n}{n+1}\right)^n$
$= \lim_{n\to\infty} \left(\frac{n}{n+1}\right)^n = \frac{1}{e} < 1$。由比值檢驗，級數收斂。
✅ 答案：收斂（$\rho = 1/e < 1$）
📝 例題 9.22
比值檢驗：∑n/2ⁿ
判斷 $\sum_{n=1}^{\infty} \frac{n}{2^n}$ 的斂散性。
解：$\rho = \lim_{n\to\infty} \left|\frac{n+1}{2^{n+1}} \cdot \frac{2^n}{n}\right| = \lim_{n\to\infty} \frac{n+1}{2n} = \frac{1}{2} < 1$。由比值檢驗，收斂。
✅ 答案：收斂（和 $= 2$）
📝 例題 9.23
根值檢驗：∑(1/n)ⁿ
判斷 $\sum_{n=1}^{\infty} \left(\frac{1}{n}\right)^n$ 的斂散性。
解：$\rho = \lim_{n\to\infty} \sqrt[n]{|a_n|} = \lim_{n\to\infty} \frac{1}{n} = 0 < 1$。由根值檢驗，收斂。
✅ 答案：收斂
📝 例題 9.24
根值檢驗：∑(3ⁿ+1)/4ⁿ
判斷 $\sum_{n=1}^{\infty} \frac{3^n+1}{4^n}$ 的斂散性。
解：$\lim_{n\to\infty} \sqrt[n]{\frac{3^n+1}{4^n}} = \lim_{n\to\infty} \frac{\sqrt[n]{3^n+1}}{4} = \frac{3}{4} < 1$。由根值檢驗，收斂。
✅ 答案：收斂
📝 例題 9.25
比值檢驗失效：p-級數
對 $\sum \frac{1}{n^2}$ 使用比值檢驗，說明檢驗不確定。
解：$\rho = \lim_{n\to\infty} \frac{1/(n+1)^2}{1/n^2} = \lim_{n\to\infty} \left(\frac{n}{n+1}\right)^2 = 1$。$\rho = 1$ 時比值檢驗不確定，但我們已知此 $p$-級數收斂。
✅ 答案：比值檢驗不確定（$\rho=1$），但級數已知收斂
