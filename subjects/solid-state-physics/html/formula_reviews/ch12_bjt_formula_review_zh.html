12.1.1–12.1.4 BJT 結構、電流關係、操作模式

BJT 的兩種結構
npn：n⁺射極 / p 基極 / n 集極
pnp：p⁺射極 / n 基極 / p 集極
射極重摻雜（∼10¹⁸–10²⁰ cm⁻³），基極輕摻雜且極窄（$W_B\sim0.1$–1 µm），集極輕摻雜（∼10¹⁵–10¹⁶ cm⁻³）。
🔍 電流組成與操作模式（9 條公式）
1端點電流關係 (10.1–10.3)
$$I_E = I_B + I_C \tag{12.1}$$射極電流 = 基極電流 + 集極電流（Kirchhoff 電流律）。npn：$I_E$ 流出，$I_B$ 和 $I_C$ 流入。pnp 方向相反。
2四種操作模式 (10.4–10.7)

射極電流 $I_E$ = 電子注入基極的電流 + 電洞從基極注入射極的電流。

12.2 少數載子分布

🔍 基極中少數載子分布的推導（9 條公式）
1基極中的擴散方程 (10.10–10.13)
基極中無電場→純擴散。穩態擴散方程：$d^2\delta n_B/dx^2 = \delta n_B/L_B^2$。
因為 $W_B \ll L_B$（基極極窄），復合項可忽略→$d^2\delta n_B/dx^2 \approx 0$ (10.12)。
2線性濃度分布 (10.14–10.15)
無復合→擴散方程解為直線：$\delta n_B(x) = \delta n_B(0)(1-x/W_B)$。
邊界條件：$x=0$（BE 接面邊界）$\delta n_B(0)=n_{B0}(e^{eV_{BE}/kT}-1)$ (10.14)。$x=W_B$（BC 接面邊界）$\delta n_B(W_B)\approx0$（逆偏收集）(10.15)。
3擴散電流 (10.16–10.18)
$$J_n = e D_B\frac{d\delta n_B}{dx} = -e D_B\frac{\delta n_B(0)}{W_B} \tag{12.17}$$濃度梯度 = $\delta n_B(0)/W_B$（常數！→擴散電流均勻）。$W_B$ 越窄→梯度越大→電流越大。

12.3.1–12.3.4 電流增益參數 ⭐

🔍 完整推導——射極效率·基極傳輸因子·電流增益（18 條公式）
1射極注入效率 $\gamma$ (10.19–10.25)
$$\gamma = \frac{I_{nE}}{I_{nE}+I_{pE}} = \frac{1}{1 + \dfrac{D_E N_B W_B}{D_B N_E L_E}} \tag{12.24}$$$I_{nE}$ = 從射極注入基極的電子電流（有用）；$I_{pE}$ = 從基極注入射極的電洞電流（浪費）。
要使 $\gamma\approx1$：$N_E \gg N_B$（射極遠重於基極），$W_B$ 小，$L_E$ 大。
2基極傳輸因子 $\alpha_T$ (10.26–10.29)
$$\alpha_T = \frac{I_{nC}}{I_{nE}} \approx 1 - \frac{1}{2}\left(\frac{W_B}{L_B}\right)^2 \tag{12.29}$$$W_B \ll L_B$ → $\alpha_T \approx 1$。在基極中復合的電子比例 $\propto (W_B/L_B)^2$。
3⭐ 共基極電流增益 $\alpha$ (10.30–10.31)
$$\boxed{\alpha = \frac{I_C}{I_E} = \gamma \cdot \alpha_T} \tag{12.31}$$$\alpha$ 略小於 1（典型 0.98–0.998）。
4⭐ 共射極電流增益 $\beta$ (10.32–10.36)
$I_C = \alpha I_E + I_{CBO}$，$I_B = (1-\alpha)I_E - I_{CBO}$ (10.32–10.34)。$$\boxed{\beta = \frac{I_C}{I_B} = \frac{\alpha}{1-\alpha}} \tag{12.36}$$$\alpha=0.99\rightarrow\beta=99$；$\alpha=0.995\rightarrow\beta=199$；$\alpha=0.998\rightarrow\beta=499$。

12.5.1 Ebers-Moll 模型

⭐ Ebers-Moll 方程（完整耦合二極體模型）
$$\boxed{I_E = I_{ES}(e^{eV_{BE}/kT}-1) - \alpha_R I_{CS}(e^{eV_{BC}/kT}-1)} \tag{12.37a}$$
$$\boxed{I_C = \alpha_F I_{ES}(e^{eV_{BE}/kT}-1) - I_{CS}(e^{eV_{BC}/kT}-1)} \tag{12.37b}$$
$\alpha_F$ = 順向共基極增益，$\alpha_R$ = 反向共基極增益。$I_{ES}, I_{CS}$ = 射極和集極飽和電流。互易關係：$\alpha_F I_{ES} = \alpha_R I_{CS}$。

12.4 非理想效應

🔍 基極寬度調變·Early 效應·高階注入（11 條公式）
1基極寬度調變 (10.38–10.43)
$V_{CE}$ 增加→BC 接面逆偏增加→BC 空乏區變寬→有效基極寬度 $W_B$ 變窄 (10.38–10.40)。
$W_B$ 變窄→濃度梯度 $n_{B0}/W_B$ 變大→$I_C$ 增加 (10.41–10.43)。
2⭐ Early 效應 (10.44–10.48)
輸出特性上傾：$$\boxed{I_C = I_S\,e^{eV_{BE}/kT}\!\left(1 + \frac{V_{CE}}{V_A}\right)} \tag{12.48}$$$V_A$ = Early 電壓（通常 50–300 V）。$V_A$ 越大→輸出特性越平→$r_o$ 越大。

12.5.3 小訊號等效電路

⭐ 混合-π 模型參數
$$g_m = \frac{I_{CQ}}{kT/e} = \frac{I_{CQ}}{V_T} \tag{12.52a}$$
$$r_\pi = \frac{\beta}{g_m} = \frac{\beta V_T}{I_{CQ}} \tag{12.52b}$$
$$r_o = \frac{V_A}{I_{CQ}} \tag{12.52c}$$
$g_m$ = 跨導（控制量），$r_\pi$ = 輸入電阻（基極-射極），$r_o$ = 輸出電阻（集極-射極，來自 Early 效應）。$V_T = kT/e = 0.0259$ V (300 K)。
