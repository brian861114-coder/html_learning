3.1.1 能帶的形成

⚡ 核心概念
原子靠近→波函數重疊→分立能階分裂成能帶。Pauli 原理要求每個電子有不同量子態→准連續能帶。Si：3s(2電子)+3p(2電子)→價帶(4N態,全滿)+導帶(4N態,全空)，中間為能隙 $E_g$。
📐 例題 3.1 — 能態間距極小
$10^{19}$ 原子，能帶寬 1 eV→態間距 $10^{-19}$ eV。電子 $v=10^7$ cm/s，$\Delta v=1$ cm/s→$\Delta E=mv\Delta v=5.7\times10^{-9}$ eV ≫ $10^{-19}$ eV→能態可視為准連續。

3.1.2 Kronig-Penney 模型 (3.1–3.19)

⚡ Kronig-Penney 是能帶理論的數學基礎，週期性位能→允許/禁止能帶。
🔍 Bloch 定理 + Kronig-Penney 推導 — 19 條公式
1Bloch 定理 (3.1–3.3)
$$\psi(x)=u(x)e^{jkx}\tag{3.1}$$$u(x)$ 為週期函數，週期 $a+b$。$$\Psi(x,t)=u(x)e^{j(kx-(E/\hbar)t)}\tag{3.3}$$行進波解，振幅受晶格週期調制。
2區域 I ($V=0$) (3.4–3.5)
$$\frac{d^2u_1}{dx^2}+2jk\frac{du_1}{dx}-(k^2-\alpha^2)u_1=0\tag{3.4}$$$$\alpha^2=\frac{2mE}{\hbar^2}\tag{3.5}$$
3區域 II ($V=V_0$) (3.6–3.8)
$$\frac{d^2u_2}{dx^2}+2jk\frac{du_2}{dx}-\left(k^2-\alpha^2+\frac{2mV_0}{\hbar^2}\right)u_2=0\tag{3.6}$$$$\beta^2=\frac{2m(E-V_0)}{\hbar^2}=\alpha^2-\frac{2mV_0}{\hbar^2}\tag{3.7}$$$$\frac{d^2u_2}{dx^2}+2jk\frac{du_2}{dx}-(k^2-\beta^2)u_2=0\tag{3.8}$$

$x=0$：$u_1(0)=u_2(0)$→$A+B-C-D=0$ (3.11–3.12)

3.1.3 k 空間圖與能帶結構 (3.20–3.24)

⭐ E–k 關係（拋物線近似，能帶邊緣）
$$E = \frac{\hbar^2 k^2}{2m^*}\tag{3.24}$$
$m^*$ = 有效質量。導帶底部和價帶頂部可用拋物線近似。
🔍 Kronig-Penney → E–k 圖
1簡化 K-P 模型（$b→0, V_0→∞$，保持 $bV_0$ 有限）
$$P'\frac{\sin\alpha a}{\alpha a}+\cos\alpha a=\cos ka\tag{3.20}$$其中 $P'=\frac{mV_0ba}{\hbar^2}$ 為位能障壁強度參數。
2允帶條件
(3.20) 左邊必須在 $[-1,1]$ → 僅當 $\alpha a$（即 $E$）落在特定範圍才有解。
這些範圍即為Brillouin 區中的允帶。
3$k$ 的範圍—Brillouin 區
$k$ 的範圍：$-\pi/(a+b)\leq k\leq\pi/(a+b)$（第一 Brillouin 區）
能帶在 $k=\pm n\pi/(a+b)$ 處不連續→能隙 $E_g$。
📐 例題 3.2 — Kronig-Penney 能隙
$P'=10$，$a=5$ Å。由 (3.20)，$\cos ka$ 在允帶邊界 $=\pm1$。解得第一能隙 $E_g\approx 3.12$ eV。

3.2.1–3.2.2 電傳導與漂移電流 (3.25–3.33)

Drift 電流密度 (3.25–3.31)
$$J = -e\sum_{i=1}^{n}v_i\tag{3.25}$$
(3.26) $J = -env_d$ (3.27) $v_d = \mu E$（$\mu$=遷移率）
(3.31) $\boxed{J = e n \mu E}$（電子電流）
(3.32) $\boxed{\sigma = e n \mu}$（導電率）
$n$=電子濃度, $\mu$=遷移率 (cm²/V·s)
🔍 漂移電流推導
1電子受力 $F=-eE$，由牛頓第二定律
$$F = m^* a = -eE \quad\Rightarrow\quad a = -\frac{eE}{m^*}\tag{3.28}$$
2碰撞限制—平均漂移速度
電子在 $\tau$（平均自由時間）內加速，碰撞後歸零：$$v_d = \frac{1}{2}a\tau = \frac{1}{2}\frac{eE}{m^*}\tau\tag{3.29}$$
3遷移率定義 (3.30)
$$\mu = \frac{v_d}{E} = \frac{e\tau}{2m^*}\tag{3.30}$$更精確：$\mu = e\tau/m^*$（考慮統計分布）

3.2.3 電子有效質量 ⭐ (3.34–3.36)

⭐ 有效質量定義 (3.34)
$$\boxed{m^* = \frac{\hbar^2}{\dfrac{d^2E}{dk^2}}}\tag{3.34}$$
$E$–$k$ 曲線的曲率決定 $m^*$。
導帶底部 $d^2E/dk^2>0$ → $m^*>0$
價帶頂部 $d^2E/dk^2<0$ → $m^*<0$ → 引入電洞概念。
🔍 有效質量推導
1群速度 (3.33)
$$v_g = \frac{1}{\hbar}\frac{dE}{dk}\tag{3.33}$$
2加速度
$$a = \frac{dv_g}{dt} = \frac{1}{\hbar}\frac{d}{dt}\!\left(\frac{dE}{dk}\right) = \frac{1}{\hbar}\frac{d^2E}{dk^2}\frac{dk}{dt}$$外力 $F = \hbar\,dk/dt$（由 $p=\hbar k$, $F=dp/dt$）$$F = \hbar\frac{dk}{dt}\tag{3.35}$$
3$F=m^*a$ (3.36)
$$F = \frac{\hbar^2}{d^2E/dk^2}\cdot a \;\equiv\; m^*a \quad\Rightarrow\quad \boxed{m^* = \frac{\hbar^2}{d^2E/dk^2}}\tag{3.36}$$

3.4 態密度函數 ⭐ (3.44–3.58)

⚡ 態密度 $g(E)$ = 單位能量、單位體積中可用的量子態數目。
🔍 完整推導 — $k$ 空間計數→$g(E)$ — 15 條公式
1$k$ 空間中的態密度
3D 無限位能井：$k_x=n_x\pi/a$, $k_y=n_y\pi/a$, $k_z=n_z\pi/a$
每個量子態在 $k$ 空間佔體積 $(\pi/a)^3$，每單位 $k$ 空間體積的態數 = $V/\pi^3$。
考慮自旋簡併（×2）：$$\text{態密度 in k-space} = \frac{2V}{\pi^3}\tag{3.44}$$
2$k$ 空間球殼中的態數
半徑 $k$、厚度 $dk$ 的球殼體積 = $4\pi k^2 dk$：$$dN = \frac{2V}{\pi^3}\cdot 4\pi k^2 dk = \frac{8Vk^2}{\pi^2}dk$$
3⭐ 態密度 $g(E)$ (3.49)
由 $E=\hbar^2k^2/2m^*$ → $k=\sqrt{2m^*E}/\hbar$，$dk = \frac{m^*}{\hbar^2k}dE$$$g(E) = \frac{1}{V}\frac{dN}{dE} = \frac{4\pi(2m^*)^{3/2}}{h^3}\sqrt{E}\tag{3.49}$$
4半導體中的態密度 (3.56–3.58)
導帶（從 $E_c$ 起算）：$$\boxed{g_c(E) = \frac{4\pi(2m_n^*)^{3/2}}{h^3}\sqrt{E-E_c}}\tag{3.56}$$價帶（從 $E_v$ 往下算）：$$\boxed{g_v(E) = \frac{4\pi(2m_p^*)^{3/2}}{h^3}\sqrt{E_v-E}}\tag{3.58}$$$m_n^*$ = 電子 DOS 有效質量, $m_p^*$ = 電洞 DOS 有效質量。
📐 例題 3.3 — Si 導帶態密度
$m_n^*=1.08m_0$。在 $E-E_c=0.1$ eV 處：
$g_c = \frac{4\pi(2\cdot1.08\cdot9.11\times10^{-31})^{3/2}}{(6.625\times10^{-34})^3}\sqrt{0.1\cdot1.6\times10^{-19}} = 7.9\times10^{20}$ cm⁻³eV⁻¹

3.5 統計力學：Fermi-Dirac 分布 ⭐ (3.59–3.79)

⭐ Fermi-Dirac 機率函數 (3.63)
$$\boxed{f_F(E) = \frac{1}{1 + \exp\!\left(\dfrac{E - E_F}{kT}\right)}}\tag{3.63}$$
$E_F$ = Fermi 能階（化學勢）。$T=0$ K：$f_F=1$ ($EE_F$)。
🔍 Fermi-Dirac 推導與性質
1Pauli 原理+Fermi-Dirac 統計 (3.59–3.62)
電子為 Fermi 子（自旋 1/2），服從 Pauli 原理。$N(E)=g(E)f_F(E)$ = 佔據態密度。$$n = \int g(E)f_F(E)\,dE\tag{3.59}$$
2Fermi 能階的性質
$E=E_F$ 時 $f_F=1/2$（50% 佔據機率）
$\lim_{T\to0}f_F(E< E_F)=1$, $\lim_{T\to0}f_F(E>E_F)=0$（階梯函數）
3Boltzmann 近似 (3.68) — $E-E_F \gg kT$ 時
$$f_F(E) \approx \exp\!\left(-\frac{E-E_F}{kT}\right)\tag{3.68}$$適用於非簡併半導體（$E_F$ 在能隙中，遠離能帶邊緣）。
4$f_F$ 隨溫度變化
$T>0$ 時 Fermi 函數在 $E_F$ 附近「模糊化」，寬度 $\sim kT$。
室溫 $kT\approx0.026$ eV → 僅靠近 $E_F$ 的電子受熱激發。
電子/電洞濃度（Boltzmann 近似）
$$n_0 = N_c\,\exp\!\left(-\frac{E_c-E_F}{kT}\right),\quad N_c = 2\left(\frac{2\pi m_n^*kT}{h^2}\right)^{3/2}\tag{3.74}$$
$p_0 = N_v\exp[-(E_F-E_v)/kT]$，$N_v$ 同理。
(3.79) $n_0p_0 = N_cN_v\exp(-E_g/kT) = n_i^2$（質量作用定律）
📐 例題 3.4 — Fermi 函數計算
$T=300$ K。求 $E-E_F = 0.05$ eV 處的 $f_F$：
$f_F = 1/[1+\exp(0.05/0.0259)] = 1/[1+e^{1.93}] = 0.127$（約 12.7% 佔據機率）
