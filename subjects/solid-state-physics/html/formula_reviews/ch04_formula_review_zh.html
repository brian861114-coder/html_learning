4.1.1–4.1.2 熱平衡載子濃度

⭐ 電子與電洞濃度（Boltzmann 近似）
$$\boxed{n_0 = N_c\,\exp\!\left(-\frac{E_c - E_F}{kT}\right)} \tag{4.11}$$
$$\boxed{p_0 = N_v\,\exp\!\left(-\frac{E_F - E_v}{kT}\right)} \tag{4.19}$$
$n_0$ = 熱平衡電子濃度, $p_0$ = 熱平衡電洞濃度
$N_c$, $N_v$ = 導帶/價帶有效態密度（見 (4.13), (4.21)）
🔍 推導 — $n_0$ 從積分得出
1電子濃度積分
$$n_0 = \int_{E_c}^{\infty} g_c(E)\,f_F(E)\,dE \tag{4.1}$$$g_c(E)=\frac{4\pi(2m_n^*)^{3/2}}{h^3}\sqrt{E-E_c}$（態密度）
$f_F(E)=\frac{1}{1+e^{(E-E_F)/kT}}$（Fermi-Dirac）
2Boltzmann 近似（$E_c-E_F \gg kT$）
$$f_F(E) \approx \exp\!\left(-\frac{E-E_F}{kT}\right) \tag{4.3}$$
3積分結果 (4.11)
令 $\eta = (E-E_c)/kT$：$$n_0 = N_c\,e^{-(E_c-E_F)/kT},\quad N_c = 2\left(\frac{2\pi m_n^* kT}{h^2}\right)^{3/2} \tag{4.11,4.13}$$
有效態密度 (4.13, 4.21)
$$N_c = 2\left(\frac{2\pi m_n^* kT}{h^2}\right)^{3/2} \tag{4.13}$$
$$N_v = 2\left(\frac{2\pi m_p^* kT}{h^2}\right)^{3/2} \tag{4.21}$$
Si (300K): $N_c=2.8\times10^{19}$ cm⁻³, $N_v=1.04\times10^{19}$ cm⁻³

4.1.3 本徵載子濃度 $n_i$

⭐ 質量作用定律 (4.23)
$$\boxed{n_0 p_0 = n_i^2} \tag{4.23}$$
熱平衡下，$n_0p_0$ 為常數，僅取決於溫度和材料（與摻雜無關）。
⭐ $n_i$ 公式 (4.20, 4.24)
$$n_i = \sqrt{N_c N_v}\,\exp\!\left(-\frac{E_g}{2kT}\right) \tag{4.20}$$
$$n_i = 2\left(\frac{2\pi kT}{h^2}\right)^{3/2}(m_n^* m_p^*)^{3/4}\exp\!\left(-\frac{E_g}{2kT}\right) \tag{4.24}$$
$n_i$ 隨溫度指數增加。Si 每升 11°C，$n_i$ 約翻倍。
📐 例題 4.1–4.2 — $n_i$ 計算
Si (300K): $n_i = \sqrt{(2.8\times10^{19})(1.04\times10^{19})}\,e^{-1.12/(2\cdot0.0259)}=1.5\times10^{10}$ cm⁻³
GaAs (300K): $n_i = 1.8\times10^6$ cm⁻³（$E_g$ 較大）

4.1.4 本徵 Fermi 能階

⭐ 本徵 Fermi 能階 $E_{Fi}$ (4.25)
$$\boxed{E_{Fi} = \frac{E_c + E_v}{2} + \frac{3}{4}kT\ln\!\left(\frac{m_p^*}{m_n^*}\right)} \tag{4.25}$$
$E_{Fi}$ 非常接近能隙中央。Si: $E_{Fi} \approx E_{\text{midgap}} - 0.013$ eV。
🔍 推導 — $n_0=p_0=n_i$ 代入求解
由 $n_0=N_c e^{-(E_c-E_{Fi})/kT}=p_0=N_v e^{-(E_{Fi}-E_v)/kT}$，取對數相減：$$E_{Fi} = \frac{E_c+E_v}{2} + \frac{kT}{2}\ln\frac{N_v}{N_c} = \frac{E_c+E_v}{2} + \frac{3}{4}kT\ln\frac{m_p^*}{m_n^*}$$
⭐ 用 $n_i$ 和 $E_{Fi}$ 重寫 $n_0$, $p_0$ (4.26–4.27)
$$\boxed{n_0 = n_i\,\exp\!\left(\frac{E_F - E_{Fi}}{kT}\right)} \tag{4.26}$$
$$\boxed{p_0 = n_i\,\exp\!\left(-\frac{E_F - E_{Fi}}{kT}\right)} \tag{4.27}$$
$E_F > E_{Fi}$ → n 型 ($n_0 > n_i > p_0$)；$E_F < E_{Fi}$ → p 型。

4.3 外質半導體

$n_0p_0$ 積在外質半導體中仍成立
$$n_0 p_0 = n_i^2 \quad\text{（摻雜不改變此關係）} \tag{4.40}$$
⚡ 簡併與非簡併
非簡併：$E_F$ 在能隙中，距能帶邊緣 $>3kT$ → Boltzmann 近似有效。
簡併：$E_F$ 進入導帶或價帶（$E_F > E_c$ 或 $E_F < E_v$）→ 必須用 Fermi-Dirac 積分。

4.4 施體與受體的統計

游離施體濃度 (4.53–4.55)
$$N_d^+ = \frac{N_d}{1 + g_d\,\exp\!\left(\dfrac{E_F - E_d}{kT}\right)} \tag{4.53}$$
$g_d=2$（施體基態簡併度）。$E_F \ll E_d$ → $N_d^+ \approx N_d$（完全游離）。
游離受體濃度 (4.56)
$$N_a^- = \frac{N_a}{1 + g_a\,\exp\!\left(\dfrac{E_a - E_F}{kT}\right)} \tag{4.56}$$
$g_a=4$（受體基態簡併度，價帶頂部雙重簡併）。

4.5 電荷中性 ⭐

⭐ 電荷中性條件 (4.60)
$$\boxed{n_0 + N_a^- = p_0 + N_d^+} \tag{4.60}$$
總負電荷 = 總正電荷。$N_a^-$ = 游離受體濃度, $N_d^+$ = 游離施體濃度。
🔍 推導 — 完全游離下的 $n_0$, $p_0$
1n 型半導體（$N_d \gg N_a \approx 0$）
$N_d^+ \approx N_d$, $N_a^- \approx 0$。由 (4.60): $n_0 = p_0 + N_d$
代入 $p_0 = n_i^2/n_0$：$n_0 = n_i^2/n_0 + N_d$
$$n_0 = \frac{N_d}{2} + \sqrt{\left(\frac{N_d}{2}\right)^2 + n_i^2} \approx N_d \quad (N_d \gg n_i) \tag{4.62}$$$$p_0 = \frac{n_i^2}{n_0} \approx \frac{n_i^2}{N_d} \tag{4.64}$$
2p 型半導體（$N_a \gg N_d \approx 0$）
$$p_0 = \frac{N_a}{2} + \sqrt{\left(\frac{N_a}{2}\right)^2 + n_i^2} \approx N_a \quad (N_a \gg n_i) \tag{4.66}$$$$n_0 = \frac{n_i^2}{p_0} \approx \frac{n_i^2}{N_a} \tag{4.68}$$
3補償半導體（$N_d, N_a$ 均存在）
$$n_0 = \frac{N_d - N_a}{2} + \sqrt{\left(\frac{N_d-N_a}{2}\right)^2 + n_i^2} \quad (N_d > N_a) \tag{4.69}$$

4.6 Fermi 能階位置 ⭐

⭐ Fermi 能階與摻雜的關係 (4.73–4.75)
$$\boxed{E_F - E_{Fi} = kT\ln\!\left(\frac{n_0}{n_i}\right)} \tag{4.73}$$
n 型 ($n_0 \approx N_d$)：$E_F - E_{Fi} = kT\ln(N_d/n_i)$ → $E_F$ 向 $E_c$ 靠近
p 型 ($p_0 \approx N_a$)：$E_{Fi} - E_F = kT\ln(N_a/n_i)$ → $E_F$ 向 $E_v$ 靠近
Fermi 能階相對於能帶邊緣
$$E_c - E_F = kT\ln\!\left(\frac{N_c}{N_d}\right) \quad\text{（n 型，$N_d \gg n_i$）} \tag{4.79}$$
$N_d$ 越大 → $E_F$ 越靠近 $E_c$。
Si: $N_d=10^{16}$ cm⁻³ → $E_c-E_F\approx0.21$ eV。
📐 例題 4.12 — Fermi 能階位置
Si, $N_d=10^{16}$ cm⁻³, $T=300$ K：
$n_0\approx N_d=10^{16}$ cm⁻³
$E_F-E_{Fi}=kT\ln(10^{16}/1.5\times10^{10})=0.347$ eV
（$E_{Fi}$ 在能隙中央下方 0.013 eV → $E_c-E_F\approx0.21$ eV）
