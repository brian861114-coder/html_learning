2.1.1 能量量子化與光電效應

Planck 量子假說 (1900)
$$E = h\nu = \frac{hc}{\lambda}$$
$E(\text{eV}) = 12400 / \lambda(\text{Å})$
🔍 光電效應推導 (2.1)
1能量守恆
光子能量 = 功函數 + 光電子動能$$h\nu = \phi + T_{\text{max}}$$
2功函數 $\phi = h\nu_0$
$\nu_0$ = 截止頻率。$\nu < \nu_0$ → 無光電子。
3光電子最大動能 (2.1)
$$\boxed{T_{\text{max}} = \frac{1}{2}mv^2 = h\nu - \phi = h(\nu - \nu_0)} \tag{2.1}$$
📐 例題 2.1 — X 射線光子能量
$\lambda = 0.708\ \text{Å}$
$E = hc/\lambda = 2.81 \times 10^{-15}\ \text{J} = 1.75 \times 10^4\ \text{eV} = 17.5\ \text{keV}$
（速算：$E = 12400/0.708 = 17500\ \text{eV}$）

2.1.2 波粒二象性

光子動量 (2.2)
$$p = \frac{h}{\lambda} \tag{2.2}$$
De Broglie 波長 (1924) (2.3)
$$\boxed{\lambda = \frac{h}{p} = \frac{h}{mv}} \tag{2.3}$$
動能形式：$p = \sqrt{2mE_k}$ → $\lambda = h/\sqrt{2mE_k}$
📐 例題 2.2 — 電子 de Broglie 波長
$v = 10^5\ \text{m/s}$，$p = mv = 9.11 \times 10^{-26}\ \text{kg·m/s}$
$\lambda = h/p = 7.27 \times 10^{-9}\ \text{m} = 72.7\ \text{Å}$（紫外線範圍）
⚡ Davisson–Germer 實驗 (1927)
電子 → 鎳單晶 → Bragg 衍射峰值 → 實驗證實 $\lambda = h/p$。

2.1.3 海森堡不確定性原理

位置–動量 (2.4)
$$\Delta p \cdot \Delta x \geq \hbar \tag{2.4}$$
能量–時間 (2.5)
$$\Delta E \cdot \Delta t \geq \hbar \tag{2.5}$$
$\hbar = h/2\pi = 1.054 \times 10^{-34}\ \text{J·s}$
🔍 TYU 2.1 — 不確定性計算
$\Delta x = 8\ \text{Å}$，$\Delta p_{\text{min}} = \hbar/\Delta x = 1.32 \times 10^{-25}\ \text{kg·m/s}$
若 $p = 1.2 \times 10^{-23}$，$\Delta E_k \approx (p/m)\Delta p = 8.5\ \text{eV}$
🔍 TYU 2.2 — 能量–時間不確定性
$\Delta E = 0.8\ \text{eV} = 1.28 \times 10^{-19}\ \text{J}$
$\Delta t_{\text{min}} = \hbar/\Delta E = 8.23 \times 10^{-16}\ \text{s}$（質子與電子相同）
⚡ 核心推論
無法同時精確測定 $x$ 和 $p$ → 只能以機率描述 → Born：$|\psi|^2$ = 機率密度。

2.2 薛丁格波動方程式 (2.6–2.18)

時間相關薛丁格方程 (2.6)
$$-\frac{\hbar^2}{2m}\frac{\partial^2\Psi(x,t)}{\partial x^2} + V(x)\Psi(x,t) = j\hbar\frac{\partial\Psi(x,t)}{\partial t} \tag{2.6}$$
🔍 分離變數法 — (2.6) → (2.13) ⭐ 必考
1假設分離 (2.7)
$$\Psi(x,t) = \psi(x)\,\phi(t) \tag{2.7}$$
2代入 (2.6) 得 (2.8)
$$-\frac{\hbar^2}{2m}\phi\frac{d^2\psi}{dx^2} + V\psi\phi = j\hbar\psi\frac{d\phi}{dt} \tag{2.8}$$
3除以 $\psi\phi$，分離變數 (2.9)
$$-\frac{\hbar^2}{2m}\frac{1}{\psi}\frac{d^2\psi}{dx^2} + V = j\hbar\frac{1}{\phi}\frac{d\phi}{dt} \tag{2.9}$$左僅與 $x$ 有關，右僅與 $t$ 有關 → 等於常數 $E$。
4時間部分 (2.10) → (2.11)
$$j\hbar\frac{1}{\phi}\frac{d\phi}{dt} = E \tag{2.10}$$$$\phi(t) = e^{-j(E/\hbar)t} = e^{-j\omega t} \tag{2.11}$$$\omega = E/\hbar$，$E$ = 粒子總能量。
5時間無關薛丁格方程 (2.12) → (2.13)
$$-\frac{\hbar^2}{2m}\frac{1}{\psi}\frac{d^2\psi}{dx^2} + V = E \tag{2.12}$$$$\boxed{\frac{d^2\psi}{dx^2} + \frac{2m}{\hbar^2}[E - V(x)]\psi = 0} \tag{2.13}$$
Born 機率詮釋 (2.14–2.17)
$$\Psi(x,t) = \psi(x)e^{-j\omega t} \tag{2.14}$$
(2.15) $|\Psi|^2 = \Psi\Psi^*$
(2.16) $\Psi\Psi^* = \psi e^{-j\omega t} \cdot \psi^* e^{+j\omega t} = \psi\psi^*$
(2.17) $\boxed{|\Psi(x,t)|^2 = |\psi(x)|^2}$（與時間無關！）
歸一化條件 (2.18)
$$\int_{-\infty}^{\infty} |\psi(x)|^2\,dx = 1 \tag{2.18}$$
邊界條件
① $\psi$ 有限、單值、連續 ② $\psi'$ 有限、單值、連續（$V$ 有限時）

2.3.1 自由空間中的電子 (2.19–2.27)

⭐ $V=0$，行進波解 (2.19–2.21)
$$\frac{d^2\psi}{dx^2} + \frac{2mE}{\hbar^2}\psi = 0 \tag{2.19}$$
(2.20a) $\psi = A\exp(jx\sqrt{2mE}/\hbar) + B\exp(-jx\sqrt{2mE}/\hbar)$
(2.20b) $\psi = Ae^{jkx} + Be^{-jkx}$
(2.21) $k = \sqrt{2mE}/\hbar$（波數 wave number）
(2.22) $\phi(t) = e^{-j(E/\hbar)t} = e^{-j\omega t}$
⭐ 總波函數 (2.23–2.24)
$$\Psi(x,t) = Ae^{j(kx-\omega t)} + Be^{-j(kx+\omega t)} \tag{2.23}$$
⭐ $k$–$p$–$\lambda$ 關係鏈 (2.25–2.27)
$$k = \frac{\sqrt{2mE}}{\hbar} = \frac{p}{\hbar} = \frac{2\pi}{\lambda}$$
(2.25a) $k = \sqrt{2mE}/\hbar = p/\hbar = 2\pi/\lambda$
(2.25b) $p = \hbar k$
(2.26) $\lambda = h/p = 2\pi\hbar/p$
(2.27a) $\lambda = 2\pi/k$ (2.27b) $k = 2\pi/\lambda$

2.3.3 階梯位能函數 (2.40–2.59)

位能設定（$E < V_0$）
$$V(x) = \begin{cases}0,&x<0\\V_0,&x>0\end{cases}$$
🔍 完整推導 — 20 條公式
1區域 I 波方程 (2.40–2.42)
$$\frac{d^2\psi_1}{dx^2} + \frac{2mE}{\hbar^2}\psi_1 = 0 \tag{2.40}$$$$\psi_1 = A_1e^{jk_1x} + B_1e^{-jk_1x}\;(x<0) \tag{2.41}$$$$k_1 = \frac{\sqrt{2mE}}{\hbar} \tag{2.42}$$
2區域 II 波方程 (2.43–2.46)，$E
$$\frac{d^2\psi_2}{dx^2} - \frac{2m(V_0-E)}{\hbar^2}\psi_2 = 0 \tag{2.43}$$$$\psi_2 = A_2e^{-k_2x} + B_2e^{+k_2x}\;(x>0) \tag{2.44}$$$$k_2 = \frac{\sqrt{2m(V_0-E)}}{\hbar} \tag{2.45}$$$x\to+\infty$ 時 $e^{+k_2x}\to\infty$ → $B_2=0$$$\psi_2 = A_2e^{-k_2x} \tag{2.46}$$
3$x=0$ 邊界匹配 (2.47–2.51)
$$\psi_1(0)=\psi_2(0) \tag{2.47}$$$$A_1 + B_1 = A_2 \tag{2.48}$$$$\psi'_1(0)=\psi'_2(0) \tag{2.49}$$$$jk_1A_1 - jk_1B_1 = -k_2A_2 \tag{2.50}$$解得：$$B_1 = \frac{k_2^2 - k_1^2 + 2jk_1k_2}{k_2^2 + k_1^2}A_1 \tag{2.51a}$$$$A_2 = \frac{2k_1(k_1 - jk_2)}{k_2^2 + k_1^2}A_1 \tag{2.51b}$$
4反射機率密度 (2.52–2.53)
$$B_1B_1^* = \frac{(k_2^2-k_1^2)^2 + 4k_1^2k_2^2}{(k_2^2+k_1^2)^2}|A_1|^2 \tag{2.52}$$
5速度關係 (2.54–2.57)
$$T = \frac{1}{2}mv^2 \tag{2.54}$$$$k_1 = \frac{mv}{\hbar} \tag{2.55}$$$$v_i = \frac{\hbar k_1}{m} \tag{2.56}$$$$v_r = \frac{\hbar k_1}{m} = v_i \tag{2.57}$$
6反射係數 $R=1$ (2.58–2.59)
$$R = \frac{v_r|B_1|^2}{v_i|A_1|^2} = \frac{|B_1|^2}{|A_1|^2} \tag{2.58}$$$$R = \frac{(k_2^2-k_1^2)^2 + 4k_1^2k_2^2}{(k_2^2+k_1^2)^2} = 1.0 \tag{2.59}$$$R=1$ → 全部反射，但 $\psi_2 \neq 0$ → 粒子滲入古典禁區！
