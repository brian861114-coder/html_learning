2.3 內能與熱力學第一定律 (2.1–2.3)

⭐ 第一定律 — 有限形式 (2.1)
$$\boxed{\Delta U = q + w} \tag{2.1}$$
$U$ = 內能（狀態函數），$q$ = 熱（路徑函數），$w$ = 功（路徑函數）。$q > 0$：吸熱；$w > 0$：外界對系統做功。
第一定律 — 微分形式 (2.2)
$$dU = \delta q + \delta w \tag{2.2}$$
$\delta$ 表示非正合微分——$q$ 和 $w$ 不是狀態函數。
⭐ 只做 $P$–$V$ 功時 (2.3)
$$dU = \delta q - P\,dV \tag{2.3}$$
$\delta w = -P\,dV$（可逆過程，$P$ 為系統壓力）。這是第一定律最常用的形式。
🔍 從焦耳實驗到第一定律
1焦耳實驗（1840s）
下降重物帶動攪拌器 → 水溫上升。功 → 熱。定量關係：1 cal = 4.184 J。
2絕熱過程
$q = 0$，$\Delta U = w$。焦耳證明：無論用什麼方式做絕熱功（攪拌、壓縮、電加熱），達到相同狀態變化所需的功都一樣 → $U$ 是狀態函數。
3一般化
允許熱交換 → $\Delta U = q + w$。第一定律本質上是能量守恆的熱力學表述。
⚡ 符號慣例陷阱
本書使用 $\Delta U = q + w$（$w$ 是外界對系統做的功）。有些教科書用 $\Delta U = q - w$（$w$ 是系統對外做的功）。兩種慣例在數值上等價，但計算時必須保持一致。

2.4 定容過程 (2.4)

⭐ 定容熱效應 (2.4)
$$dU = \delta q_V \quad \Rightarrow \quad \Delta U = q_V \tag{2.4}$$
$dV = 0$ → $w = 0$ → $\Delta U = q_V$。彈卡計（bomb calorimeter）的原理。

2.5 定壓過程與焓 (2.5–2.6)

⭐ 焓的定義 (2.5)
$$\boxed{H = U + PV} \tag{2.5}$$
焓是狀態函數（$U$, $P$, $V$ 都是狀態函數）。物理意義：定壓下系統的「總能量內容」。
⭐ 定壓熱效應 (2.6)
$$dH = \delta q_P \quad \Rightarrow \quad \Delta H = q_P \tag{2.6}$$
推導：$dH = dU + P\,dV + V\,dP = (\delta q - P\,dV) + P\,dV + V\,dP$。定壓下 $dP = 0$ → $dH = \delta q_P$。
🔍 $\Delta H$ 與 $\Delta U$ 的關係
1定義展開
$$\Delta H = \Delta U + \Delta(PV)$$
2理想氣體
$PV = nRT$ → $\Delta(PV) = \Delta n_g RT$（僅氣體莫耳數變化貢獻）$$\Delta H = \Delta U + \Delta n_g RT$$
3固體/液體反應
$\Delta V \approx 0$ → $\Delta H \approx \Delta U$。

2.6 熱容量 (2.7–2.9)

熱容量定義
$$C = \frac{\delta q}{dT}$$
熱容量是路徑依賴量——取決於加熱條件（定容或定壓）。
⭐ 定容熱容量 (2.7)
$$\boxed{C_V = \left(\frac{\partial U}{\partial T}\right)_V} \tag{2.7}$$
定容下 $\delta q_V = dU$，因此 $C_V = (\partial U/\partial T)_V$。
⭐ 定壓熱容量 (2.8)
$$\boxed{C_P = \left(\frac{\partial H}{\partial T}\right)_P} \tag{2.8}$$
定壓下 $\delta q_P = dH$，因此 $C_P = (\partial H/\partial T)_P$。
⭐ 理想氣體 $C_P - C_V = R$ (2.9)
$$\boxed{C_P - C_V = R} \tag{2.9}$$
推導：$H = U + RT$（每莫耳）→ $dH/dT = dU/dT + R$。
🔍 氣體 $C_V$ 的能量均分推導
1能量均分定理
每個二次方自由度貢獻 $\frac{1}{2}kT$（每分子）或 $\frac{1}{2}R$（每莫耳）。
2平移自由度 = 3
$x, y, z$ 三個方向 → $\frac{3}{2}R$。
3旋轉自由度
線性分子（如 N₂）：2（無繞軸旋轉）→ 總計 $\frac{5}{2}R$
非線性分子（如 H₂O）：3 → 總計 $3R$
4振動自由度（高溫激活）
每個振動模式貢獻 $R$（動能 $\frac{1}{2}R$ + 位能 $\frac{1}{2}R$）。低溫下凍結。
⚡ $\gamma = C_P/C_V$ 常見值
單原子：$\gamma = 5/3 \approx 1.67$ 雙原子（室溫）：$\gamma = 7/5 = 1.40$ 多原子非線性：$\gamma = 4/3 \approx 1.33$

2.7 可逆絕熱過程 (2.10–2.12)

⭐ 可逆絕熱 $P$–$V$ 關係 (2.10)
$$\boxed{PV^\gamma = \text{constant}} \tag{2.10}$$
$\gamma = C_P/C_V$。推導：$dU = -P\,dV$，$C_V\,dT = -P\,dV = -(RT/V)dV$，積分得 $TV^{\gamma-1} = C$，代入 $T = PV/R$ 得 $PV^\gamma = C$。
等效形式 — $T$–$V$ (2.11)
$$TV^{\gamma-1} = \text{constant} \tag{2.11}$$
等效形式 — $T$–$P$ (2.12)
$$TP^{(1-\gamma)/\gamma} = \text{constant} \tag{2.12}$$
🔍 絕熱過程的完整推導
1第一定律（絕熱：$q = 0$）
$$dU = -P\,dV$$
2理想氣體 $dU = C_V\,dT$
$$C_V\,dT = -\frac{RT}{V}\,dV$$
3分離變數積分
$$\frac{dT}{T} = -\frac{R}{C_V}\frac{dV}{V} = -(\gamma-1)\frac{dV}{V}$$（$R = C_P - C_V$，$R/C_V = \gamma-1$）
4積分結果
$$\ln T = -(\gamma-1)\ln V + C \quad\Rightarrow\quad TV^{\gamma-1} = C$$
📐 數值例題 — 絕熱膨脹冷卻
1 mol 單原子理想氣體（$\gamma = 5/3$）從 $V_1 = 10$ L，$T_1 = 300$ K 可逆絕熱膨脹到 $V_2 = 20$ L。
$T_2 = T_1(V_1/V_2)^{\gamma-1} = 300 \times (0.5)^{2/3} = 189\ \text{K}$（降溫 111°C）
$P_1 = RT_1/V_1$，$P_2 = P_1(V_1/V_2)^\gamma = P_1(0.5)^{5/3} = 0.315P_1$

2.8 可逆等溫過程 (2.13–2.14)

⭐ 可逆等溫功 (2.13)
$$w = -\int_{V_1}^{V_2} P\,dV = -nRT\ln\frac{V_2}{V_1} \tag{2.13}$$
推導：$P = nRT/V$（理想氣體等溫），代入積分。
等效形式 — 壓力比 (2.14)
$$\boxed{w = -nRT\ln\frac{P_1}{P_2}} \tag{2.14}$$
等溫下 $P_1V_1 = P_2V_2$ → $V_2/V_1 = P_1/P_2$。
熱效應
$$q = -w = nRT\ln\frac{V_2}{V_1}$$
等溫下 $dU = 0$ → $q + w = 0$ → $q = -w$。膨脹時系統從環境吸熱。
⚡ 可逆 vs 不可逆等溫過程
可逆等溫過程中，系統對抗的始終是系統壓力 → 對外做功最大（膨脹）或外界對系統做功最小（壓縮）。不可逆等溫過程（如對抗恆定外壓）的功總是小於可逆功。第三章將從熵的角度解釋這一現象。
